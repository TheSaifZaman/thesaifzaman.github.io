Boyer-Moore String Search

Medium Best: O(n/m) Worst: O(n·m) Space: O(k)

How It Works

Boyer-Moore is one of the most efficient string-matching algorithms in practice. Unlike KMP which scans left-to-right, Boyer-Moore compares the pattern from right to left within each alignment. When a mismatch occurs, it uses two heuristics to determine how far to shift the pattern: the bad character rule and the good suffix rule.

The bad character rule looks at the mismatched text character and shifts the pattern so this character aligns with its rightmost occurrence in the pattern (or past the pattern entirely if it doesn't appear). The good suffix rule uses the already-matched suffix to find the next alignment where this suffix occurs in the pattern.

The algorithm takes the maximum of the two shift values, ensuring it makes the longest safe skip. In the best case (e.g., searching for a pattern with characters not in the text), Boyer-Moore achieves O(n/m) time — sublinear! It is the algorithm of choice in many text editors and search tools like grep.

Right-to-Left Scan

Compare pattern from the end; mismatches at the end allow larger skips.

Bad Character Rule

Shift pattern to align the mismatched text character with its last occurrence in pattern.

Good Suffix Rule

Use matched suffix to find next alignment where it reappears in the pattern.

Sublinear Best Case

O(n/m) best case — can skip entire pattern lengths at a time.

Implementation

function boyerMoore(text, pattern) {
    var n = text.length, m = pattern.length;
    if (m === 0) return 0;

    // Bad character table
    var badChar = {};
    for (var i = 0; i < m; i++) {
        badChar[pattern[i]] = i;
    }

    var results = [];
    var s = 0; // shift of pattern with respect to text

    while (s <= n - m) {
        var j = m - 1;

        // Compare from right to left
        while (j >= 0 && pattern[j] === text[s + j]) {
            j--;
        }

        if (j < 0) {
            // Match found
            results.push(s);
            s += (s + m < n) ? m - (badChar[text[s + m]] !== undefined ? badChar[text[s + m]] : -1) : 1;
        } else {
            // Bad character shift
            var bcShift = badChar[text[s + j]] !== undefined ? badChar[text[s + j]] : -1;
            s += Math.max(1, j - bcShift);
        }
    }
    return results;
}
