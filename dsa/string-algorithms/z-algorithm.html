Z-Algorithm

Medium Time: O(n) Space: O(n)

How It Works

The Z-algorithm computes the Z-array for a string S in linear time. For each position i, Z[i] is the length of the longest substring starting at position i that matches a prefix of S. By definition, Z[0] is set to 0 (or the length of the string, depending on convention).

The algorithm maintains a window [L, R] representing the rightmost Z-box found so far — the interval that matches a prefix of S and extends furthest to the right. For each new position i, if i falls within [L, R], we can use previously computed Z-values to get a head start, then extend by comparing characters directly.

For pattern matching, concatenate pattern + "$" + text (where "$" is a character not in either string), then compute the Z-array. Any position i where Z[i] equals the pattern length indicates a match. This gives O(n+m) string matching, similar to KMP but with a different approach.

Z-Array

Z[i] = length of longest substring starting at i matching a prefix of S.

Z-Box Window

Maintains [L,R] to reuse previous work, achieving O(n) amortized.

Concatenation Trick

Pattern + separator + text enables pattern matching via Z-array.

KMP Alternative

Equivalent power to KMP but conceptually simpler for many problems.

Implementation

function zFunction(s) {
    var n = s.length;
    var z = new Array(n).fill(0);
    var l = 0, r = 0;
    for (var i = 1; i < n; i++) {
        if (i < r) {
            z[i] = Math.min(r - i, z[i - l]);
        }
        while (i + z[i] < n && s[z[i]] === s[i + z[i]]) {
            z[i]++;
        }
        if (i + z[i] > r) {
            l = i;
            r = i + z[i];
        }
    }
    return z;
}

function zSearch(text, pattern) {
    var concat = pattern + '$' + text;
    var z = zFunction(concat);
    var m = pattern.length;
    var results = [];
    for (var i = m + 1; i < concat.length; i++) {
        if (z[i] === m) {
            results.push(i - m - 1);
        }
    }
    return results;
}
