Edit Distance (Levenshtein)

Medium Time: O(n·m) Space: O(n·m)

How It Works

Edit distance (Levenshtein distance) measures the minimum number of single-character operations needed to transform one string into another. The three allowed operations are: insert a character, delete a character, or substitute (replace) a character. For example, the edit distance between "kitten" and "sitting" is 3.

The DP approach builds an (n+1) x (m+1) table where dp[i][j] represents the edit distance between the first i characters of word1 and the first j characters of word2. The base cases are dp[i][0] = i (delete all) and dp[0][j] = j (insert all). For each cell, if characters match, dp[i][j] = dp[i-1][j-1] (no edit needed); otherwise, take 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) representing delete, insert, and substitute respectively.

Edit distance is fundamental in spell checking, DNA sequence alignment, natural language processing, and fuzzy string matching. Variations include weighted edit distance (different costs per operation) and Damerau-Levenshtein (which also allows transpositions).

Three Operations

Insert, delete, or substitute a single character. Each costs 1.

DP Table

Fill (n+1)x(m+1) grid bottom-up; answer at dp[n][m].

Spell Checking

Core of autocorrect: find dictionary words within small edit distance.

Space Optimization

Can reduce to O(min(n,m)) space by keeping only two rows at a time.

Implementation

function editDistance(word1, word2) {
    var n = word1.length, m = word2.length;
    var dp = [];
    for (var i = 0; i <= n; i++) {
        dp[i] = new Array(m + 1).fill(0);
        dp[i][0] = i;
    }
    for (var j = 0; j <= m; j++) dp[0][j] = j;

    for (var i = 1; i <= n; i++) {
        for (var j = 1; j <= m; j++) {
            if (word1[i - 1] === word2[j - 1]) {
                dp[i][j] = dp[i - 1][j - 1];
            } else {
                dp[i][j] = 1 + Math.min(
                    dp[i - 1][j],     // delete
                    dp[i][j - 1],     // insert
                    dp[i - 1][j - 1]  // substitute
                );
            }
        }
    }
    return dp[n][m];
}

// Space-optimized version using two rows
function editDistanceOptimized(word1, word2) {
    var n = word1.length, m = word2.length;
    var prev = [], curr = [];
    for (var j = 0; j <= m; j++) prev[j] = j;

    for (var i = 1; i <= n; i++) {
        curr[0] = i;
        for (var j = 1; j <= m; j++) {
            if (word1[i - 1] === word2[j - 1]) {
                curr[j] = prev[j - 1];
            } else {
                curr[j] = 1 + Math.min(prev[j], curr[j - 1], prev[j - 1]);
            }
        }
        prev = curr.slice();
    }
    return prev[m];
}
