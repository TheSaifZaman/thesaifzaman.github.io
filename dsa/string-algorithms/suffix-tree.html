Suffix Tree (Ukkonen's)

Hard Build: O(n) Search: O(m) Space: O(n)

How It Works

A suffix tree is a compressed trie containing all suffixes of a string. Each edge is labeled with a substring, and every suffix of the original string corresponds to a path from the root to a leaf. The tree has at most 2n-1 nodes for a string of length n (n leaves and at most n-1 internal nodes).

Ukkonen's algorithm builds the suffix tree online, left to right, in O(n) time. It processes one character at a time, extending all current suffixes. Three key rules govern extensions: (1) if the current path ends at a leaf, extend the edge label; (2) if the next character already exists in the tree, do nothing (implicit extension); (3) if it doesn't exist, create a new branch. Suffix links speed up traversal between extension points.

Suffix trees enable powerful string operations: O(m) pattern matching, finding the longest repeated substring, longest common substring of two strings, and many more. While they use more space than suffix arrays, they offer faster query times for certain operations.

Compressed Trie

Edge labels are substrings, not single characters, keeping the tree compact.

Suffix Links

Internal nodes link to the node representing the next shorter suffix, enabling O(1) jumps.

Online Construction

Ukkonen's builds incrementally, character by character, in O(n) total time.

O(m) Pattern Search

Match pattern by following edges; if a complete path exists, the pattern occurs.

Implementation

// Simplified suffix tree (naive construction for clarity)
function SuffixTreeNode() {
    this.children = {};
    this.start = -1;
    this.end = -1;
    this.suffixIndex = -1;
}

function buildSuffixTree(text) {
    var root = new SuffixTreeNode();
    for (var i = 0; i < text.length; i++) {
        var current = root;
        for (var j = i; j < text.length; j++) {
            var ch = text[j];
            if (!current.children[ch]) {
                var node = new SuffixTreeNode();
                node.start = j;
                node.end = text.length - 1;
                node.suffixIndex = i;
                current.children[ch] = node;
                break;
            }
            current = current.children[ch];
        }
    }
    return root;
}

function searchInSuffixTree(root, text, pattern) {
    var current = root;
    for (var i = 0; i < pattern.length; i++) {
        if (!current.children[pattern[i]]) return false;
        current = current.children[pattern[i]];
    }
    return true;
}
