String Matching & Manipulation

Easy Naive: O(n·m) Space: O(1)

How It Works

String matching is the problem of finding where a pattern string occurs within a larger text string. The naive (brute-force) approach checks every possible position in the text: for each starting index, it compares the pattern character by character. If all characters match, a match is found; otherwise, it slides forward by one position.

Basic string manipulation includes operations like reversing a string, checking palindromes, finding the longest common prefix, and comparing substrings. These are foundational skills that underpin more advanced string algorithms. Most of these operations work by iterating through characters using one or two pointers.

While the naive approach has O(n*m) worst-case complexity, it performs well in practice for short patterns and random text due to early termination on mismatches. Understanding it is essential before moving to advanced algorithms like KMP, Rabin-Karp, or Boyer-Moore.

Brute Force

Try every alignment of pattern in text; simple but O(n*m) worst case.

Two Pointers

Palindrome checks and reversal use pointers from both ends moving inward.

Common Prefix

Compare characters at the same position across multiple strings.

Building Block

Foundation for KMP, Rabin-Karp, Boyer-Moore, and other advanced algorithms.

// Naive string matching
function naiveSearch(text, pattern) {
    var n = text.length, m = pattern.length;
    var results = [];
    for (var i = 0; i <= n - m; i++) {
        var j = 0;
        while (j < m && text[i + j] === pattern[j]) j++;
        if (j === m) results.push(i);
    }
    return results;
}

// Palindrome check (two pointers)
function isPalindrome(s) {
    var left = 0, right = s.length - 1;
    while (left < right) {
        if (s[left] !== s[right]) return false;
        left++; right--;
    }
    return true;
}

// Longest common prefix
function longestCommonPrefix(strs) {
    if (!strs.length) return "";
    var prefix = strs[0];
    for (var i = 1; i < strs.length; i++) {
        while (strs[i].indexOf(prefix) !== 0) {
            prefix = prefix.slice(0, -1);
            if (!prefix) return "";
        }
    }
    return prefix;
}
