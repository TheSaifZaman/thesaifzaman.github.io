Aho-Corasick Algorithm

Hard Build: O(m·k) Search: O(n+z) Space: O(m·k)

How It Works

The Aho-Corasick algorithm is a multi-pattern string matching algorithm. Given a set of patterns (a "dictionary") and a text string, it finds all occurrences of all patterns in a single pass through the text. It combines a trie (prefix tree) of patterns with failure links (similar to KMP's failure function) and output links.

Construction has three phases: (1) Build a trie from all patterns; (2) Compute failure links using BFS — a failure link from node u points to the longest proper suffix of the string represented by u that is also a prefix of some pattern; (3) Compute output links that chain to the next node along the failure path that represents a complete pattern.

During search, the automaton processes each text character once. On a mismatch, it follows failure links (like KMP) to find the next valid state. At each state, output links report all patterns ending at the current text position. Total search time is O(n + z) where z is the number of pattern occurrences found.

Trie + Failure Links

Combines trie structure with KMP-like failure transitions for multi-pattern matching.

All Patterns at Once

Finds all occurrences of all patterns in a single linear scan of the text.

Finite Automaton

The resulting structure is a deterministic finite automaton that processes text character by character.

Real-World Use

Used in antivirus scanners, intrusion detection, content filtering, and bioinformatics.

Implementation

function AhoCorasick(patterns) {
    this.goto = [{}];        // goto function
    this.fail = [0];         // failure links
    this.output = [[]];      // output at each state
    this.stateCount = 1;

    // Build goto (trie)
    for (var p = 0; p < patterns.length; p++) {
        var state = 0;
        for (var i = 0; i < patterns[p].length; i++) {
            var ch = patterns[p][i];
            if (!this.goto[state][ch]) {
                this.goto[state][ch] = this.stateCount;
                this.goto.push({});
                this.fail.push(0);
                this.output.push([]);
                this.stateCount++;
            }
            state = this.goto[state][ch];
        }
        this.output[state].push(patterns[p]);
    }

    // Build failure links (BFS)
    var queue = [];
    for (var ch in this.goto[0]) {
        var s = this.goto[0][ch];
        this.fail[s] = 0;
        queue.push(s);
    }
    while (queue.length) {
        var r = queue.shift();
        for (var ch in this.goto[r]) {
            var s = this.goto[r][ch];
            queue.push(s);
            var state = this.fail[r];
            while (state && !this.goto[state][ch]) state = this.fail[state];
            this.fail[s] = this.goto[state][ch] || 0;
            if (this.fail[s] === s) this.fail[s] = 0;
            this.output[s] = this.output[s].concat(this.output[this.fail[s]]);
        }
    }
}

AhoCorasick.prototype.search = function (text) {
    var results = [];
    var state = 0;
    for (var i = 0; i < text.length; i++) {
        while (state && !this.goto[state][text[i]]) state = this.fail[state];
        state = this.goto[state][text[i]] || 0;
        for (var j = 0; j < this.output[state].length; j++) {
            results.push({ index: i - this.output[state][j].length + 1, pattern: this.output[state][j] });
        }
    }
    return results;
};
