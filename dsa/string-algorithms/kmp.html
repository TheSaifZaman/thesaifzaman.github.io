KMP Algorithm

Medium Time: O(n+m) Space: O(m)

How It Works

The Knuth-Morris-Pratt (KMP) algorithm is an efficient string-matching algorithm that searches for occurrences of a pattern within a text. Unlike the naive approach that backtracks on every mismatch, KMP preprocesses the pattern to build a failure function (also called the partial match table or prefix function) that tells it how far to skip ahead when a mismatch occurs.

The failure function lps[i] stores the length of the longest proper prefix of pattern[0..i] that is also a suffix. When a mismatch happens at position j in the pattern, instead of restarting from scratch, KMP shifts the pattern so that lps[j-1] characters are already matched, avoiding redundant comparisons.

This preprocessing takes O(m) time where m is the pattern length, and the search itself takes O(n) time where n is the text length. The total time complexity is O(n+m), making KMP significantly faster than the naive O(n*m) approach for large inputs, especially when the pattern contains repeating sub-patterns.

No Backtracking

The text pointer never moves backwards, ensuring O(n) scan time.

Failure Function

LPS array encodes pattern structure to skip redundant comparisons.

Linear Time

O(n+m) guaranteed, unlike naive O(n*m) worst case.

Streaming Friendly

Can process text character by character without needing the full text in memory.

Implementation

function buildLPS(pattern) {
    var m = pattern.length;
    var lps = new Array(m).fill(0);
    var len = 0;
    var i = 1;
    while (i < m) {
        if (pattern[i] === pattern[len]) {
            len++;
            lps[i] = len;
            i++;
        } else {
            if (len !== 0) {
                len = lps[len - 1];
            } else {
                lps[i] = 0;
                i++;
            }
        }
    }
    return lps;
}

function kmpSearch(text, pattern) {
    var n = text.length, m = pattern.length;
    var lps = buildLPS(pattern);
    var results = [];
    var i = 0, j = 0;
    while (i < n) {
        if (text[i] === pattern[j]) {
            i++; j++;
        }
        if (j === m) {
            results.push(i - j);
            j = lps[j - 1];
        } else if (i < n && text[i] !== pattern[j]) {
            if (j !== 0) {
                j = lps[j - 1];
            } else {
                i++;
            }
        }
    }
    return results;
}
