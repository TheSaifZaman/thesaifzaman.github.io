Longest Common Subsequence

Medium Time: O(n·m) Space: O(n·m)

How It Works

The Longest Common Subsequence (LCS) problem finds the longest sequence of characters that appears in the same relative order (but not necessarily contiguously) in both input strings. For example, the LCS of "ABCBDAB" and "BDCAB" is "BCAB" (length 4).

The classic dynamic programming solution builds an (n+1) x (m+1) table where dp[i][j] represents the LCS length of the first i characters of string A and the first j characters of string B. The recurrence is: if A[i-1] == B[j-1], then dp[i][j] = dp[i-1][j-1] + 1; otherwise, dp[i][j] = max(dp[i-1][j], dp[i][j-1]).

The actual LCS string can be recovered by backtracking from dp[n][m]: whenever the characters match, include that character and move diagonally; otherwise, move in the direction of the larger value. LCS has applications in diff tools (comparing files), bioinformatics (DNA sequence alignment), and version control.

DP Table

Build (n+1)x(m+1) table bottom-up; each cell depends on 3 neighbors.

Character Match

When characters match, extend diagonal value by 1; otherwise take the max of left/above.

Backtracking

Trace back through the table from dp[n][m] to reconstruct the actual LCS string.

Diff Tool Foundation

Unix diff and git diff use LCS variants to show minimal changes between files.

Implementation

function lcs(a, b) {
    var n = a.length, m = b.length;
    var dp = [];
    for (var i = 0; i <= n; i++) {
        dp[i] = new Array(m + 1).fill(0);
    }

    for (var i = 1; i <= n; i++) {
        for (var j = 1; j <= m; j++) {
            if (a[i - 1] === b[j - 1]) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }

    // Backtrack to find the LCS string
    var result = '';
    var i = n, j = m;
    while (i > 0 && j > 0) {
        if (a[i - 1] === b[j - 1]) {
            result = a[i - 1] + result;
            i--; j--;
        } else if (dp[i - 1][j] > dp[i][j - 1]) {
            i--;
        } else {
            j--;
        }
    }
    return { length: dp[n][m], subsequence: result };
}
