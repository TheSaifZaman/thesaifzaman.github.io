Manacher's Algorithm

Hard Time: O(n) Space: O(n)

How It Works

Manacher's algorithm finds the longest palindromic substring in linear time. The key challenge with palindromes is handling both odd-length and even-length cases. Manacher's solves this by transforming the string: insert a special character (e.g., '#') between every character and at the boundaries, so "abba" becomes "#a#b#b#a#". This way, every palindrome becomes odd-length.

The algorithm maintains a center C and right boundary R of the rightmost palindrome found so far. For each position i, it uses the mirror position (2*C - i) to get a starting estimate for the palindrome radius. If i is within R, the palindrome at the mirror gives a lower bound. Then it tries to expand further by comparing characters.

This mirror-and-expand strategy ensures each character is visited at most twice, giving O(n) total time. The result is a P-array where P[i] holds the radius of the palindrome centered at i in the transformed string. The longest palindromic substring is found by taking the maximum P[i].

Transform Trick

Insert separators to unify odd and even palindromes into one case.

Mirror Property

Reuse palindrome radii from mirror positions to avoid redundant work.

Center & Right

Track the palindrome extending furthest right; update as new ones are found.

True Linear Time

Each character compared at most twice, achieving genuine O(n).

Implementation

function manacher(s) {
    // Transform: "abc" -> "^#a#b#c#$"
    var t = '^#' + s.split('').join('#') + '#$';
    var n = t.length;
    var p = new Array(n).fill(0);
    var c = 0, r = 0;

    for (var i = 1; i < n - 1; i++) {
        var mirror = 2 * c - i;
        if (i < r) {
            p[i] = Math.min(r - i, p[mirror]);
        }
        // Expand around center i
        while (t[i + p[i] + 1] === t[i - p[i] - 1]) {
            p[i]++;
        }
        // Update center and right boundary
        if (i + p[i] > r) {
            c = i;
            r = i + p[i];
        }
    }

    // Find the maximum palindrome
    var maxLen = 0, centerIdx = 0;
    for (var i = 1; i < n - 1; i++) {
        if (p[i] > maxLen) {
            maxLen = p[i];
            centerIdx = i;
        }
    }
    var start = (centerIdx - maxLen) / 2;
    return s.substring(start, start + maxLen);
}
