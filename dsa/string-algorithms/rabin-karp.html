Rabin-Karp Algorithm

Medium Avg: O(n+m) Worst: O(n·m) Space: O(1)

How It Works

The Rabin-Karp algorithm uses hashing to find pattern matches in text. Instead of comparing characters one by one, it computes a hash value for the pattern and for each window of text of the same length. If the hashes match, it performs a character-by-character verification to confirm (since hash collisions are possible).

The key insight is the rolling hash: when sliding the window one position right, the new hash can be computed in O(1) from the old hash by removing the contribution of the leftmost character and adding the new rightmost character. This avoids recomputing the hash from scratch for each position.

Rabin-Karp excels at multi-pattern matching — searching for any of k patterns simultaneously in O(n*k) average time. It is also used in plagiarism detection, DNA sequence matching, and other applications where multiple pattern searches are needed.

Rolling Hash

Update hash in O(1) as the window slides, avoiding full recomputation.

Multi-Pattern

Easily extended to search for multiple patterns simultaneously.

Probabilistic

Hash collisions require verification; choose a good hash to minimize spurious hits.

Practical Uses

DNA matching, plagiarism detection, longest duplicate substring problems.

Implementation

function rabinKarp(text, pattern) {
    var n = text.length, m = pattern.length;
    var d = 256, q = 101; // base and prime
    var h = 1, p = 0, t = 0;
    var results = [];

    // h = d^(m-1) mod q
    for (var i = 0; i < m - 1; i++)
        h = (h * d) % q;

    // Initial hash values
    for (var i = 0; i < m; i++) {
        p = (d * p + pattern.charCodeAt(i)) % q;
        t = (d * t + text.charCodeAt(i)) % q;
    }

    // Slide pattern over text
    for (var i = 0; i <= n - m; i++) {
        if (p === t) {
            // Verify character by character
            var match = true;
            for (var j = 0; j < m; j++) {
                if (text[i + j] !== pattern[j]) {
                    match = false; break;
                }
            }
            if (match) results.push(i);
        }
        // Compute hash for next window
        if (i < n - m) {
            t = (d * (t - text.charCodeAt(i) * h) +
                 text.charCodeAt(i + m)) % q;
            if (t < 0) t += q;
        }
    }
    return results;
}
