Word2Vec

Medium Time: O(n·d) Space: O(V·d)

How It Works

Word2Vec learns dense vector representations (embeddings) for words such that semantically similar words end up close together in the vector space. It was introduced by Mikolov et al. at Google in 2013 and revolutionized NLP.

Two architectures exist: Skip-gram predicts context words given a center word, and CBOW (Continuous Bag of Words) predicts the center word from context. Training uses a shallow neural network with one hidden layer, where the learned weights become the word embeddings.

Famous property: vector arithmetic captures semantic relationships. For example, king - man + woman ≈ queen.

Dense Embeddings

Each word maps to a compact d-dimensional vector (typically 100-300d).

Semantic Arithmetic

king - man + woman = queen via vector operations.

Context Windows

Learns from surrounding words within a fixed window size.

Efficient Training

Negative sampling makes training scale to huge corpora.

// Skip-gram with negative sampling (simplified)
function trainWord2Vec(corpus, dim, windowSize, negSamples, lr) {
    var vocab = buildVocab(corpus);
    var W = randomMatrix(vocab.length, dim);  // input embeddings
    var C = randomMatrix(vocab.length, dim);  // context embeddings

    corpus.forEach(function(sentence) {
        for (var i = 0; i < sentence.length; i++) {
            var center = vocab.indexOf(sentence[i]);
            // Positive context words
            for (var j = -windowSize; j <= windowSize; j++) {
                if (j === 0 || i + j < 0 || i + j >= sentence.length) continue;
                var context = vocab.indexOf(sentence[i + j]);

                // Positive sample: push center and context together
                var dot = dotProduct(W[center], C[context]);
                var sig = sigmoid(dot);
                updateVectors(W[center], C[context], lr * (1 - sig));

                // Negative samples: push center and random words apart
                for (var n = 0; n < negSamples; n++) {
                    var neg = randomWord(vocab);
                    dot = dotProduct(W[center], C[neg]);
                    sig = sigmoid(dot);
                    updateVectors(W[center], C[neg], lr * (-sig));
                }
            }
        }
    });
    return W; // final word embeddings
}
