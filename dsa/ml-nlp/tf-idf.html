TF-IDF

Easy Time: O(n·V) Space: O(n·V)

How It Works

TF-IDF (Term Frequency - Inverse Document Frequency) is a numerical statistic that measures how important a word is to a document within a collection (corpus). It is the product of two components:

TF (Term Frequency): How often a word appears in a document. TF(t,d) = count(t in d) / |d|

IDF (Inverse Document Frequency): How rare a word is across all documents. IDF(t) = log(N / df(t)), where N is total documents and df(t) is the number of documents containing t.

Common words like "the" get low IDF scores (they appear everywhere), while distinctive words get high scores. TF-IDF is widely used in search engines, document classification, and text summarization.

Term Frequency

Words that appear more in a document are more relevant to it.

Inverse Document Freq

Rare words across the corpus are more distinctive.

Search Ranking

Foundation of information retrieval and search engines.

Stop Word Suppression

Common words naturally get low scores without explicit removal.

Implementation

function computeTFIDF(documents) {
    var N = documents.length;
    var df = {}; // document frequency per term

    // Count document frequency
    documents.forEach(function(doc) {
        var seen = {};
        doc.split(' ').forEach(function(word) {
            if (!seen[word]) { seen[word] = true; df[word] = (df[word] || 0) + 1; }
        });
    });

    // Compute TF-IDF for each document
    return documents.map(function(doc) {
        var words = doc.split(' ');
        var tf = {};
        words.forEach(function(w) { tf[w] = (tf[w] || 0) + 1; });

        var tfidf = {};
        Object.keys(tf).forEach(function(w) {
            var termFreq = tf[w] / words.length;
            var idf = Math.log(N / df[w]);
            tfidf[w] = termFreq * idf;
        });
        return tfidf;
    });
}
