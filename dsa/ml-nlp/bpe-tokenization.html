Byte Pair Encoding (BPE)

Medium Time: O(n·V) Space: O(V)

How It Works

Byte Pair Encoding (BPE) is a subword tokenization algorithm used in modern language models (GPT, BERT, etc.). It starts with a vocabulary of individual characters and iteratively merges the most frequent adjacent pair into a new token.

Process: (1) Split all words into characters. (2) Count all adjacent character pairs. (3) Merge the most frequent pair into a new symbol. (4) Repeat until reaching the desired vocabulary size.

BPE handles rare words by breaking them into known subword units. For example, "unhappiness" might become ["un", "happi", "ness"], allowing the model to understand it even if never seen as a whole word during training.

Subword Units

Breaks rare words into frequently occurring subword pieces.

Iterative Merging

Most frequent character pair merged at each step.

GPT/BERT Standard

BPE or variants used by virtually all modern LLMs.

No OOV Words

Any word can be encoded — no out-of-vocabulary problem.

Implementation

function trainBPE(corpus, numMerges) {
    // Split words into characters
    var vocab = {};
    corpus.forEach(function(word) {
        var chars = word.split('').join(' ') + ' </w>';
        vocab[chars] = (vocab[chars] || 0) + 1;
    });

    var merges = [];
    for (var i = 0; i < numMerges; i++) {
        // Count all adjacent pairs
        var pairs = {};
        Object.keys(vocab).forEach(function(word) {
            var symbols = word.split(' ');
            for (var j = 0; j < symbols.length - 1; j++) {
                var pair = symbols[j] + ',' + symbols[j + 1];
                pairs[pair] = (pairs[pair] || 0) + vocab[word];
            }
        });

        // Find most frequent pair
        var bestPair = null, bestCount = 0;
        Object.keys(pairs).forEach(function(p) {
            if (pairs[p] > bestCount) { bestPair = p; bestCount = pairs[p]; }
        });
        if (!bestPair) break;

        // Merge pair in all words
        var parts = bestPair.split(',');
        merges.push(parts);
        var newVocab = {};
        Object.keys(vocab).forEach(function(word) {
            var newWord = word.split(parts[0] + ' ' + parts[1]).join(parts[0] + parts[1]);
            newVocab[newWord] = vocab[word];
        });
        vocab = newVocab;
    }
    return merges;
}
