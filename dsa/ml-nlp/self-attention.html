Self-Attention

Hard Time: O(n²·d) Space: O(n² + n·d)

How It Works

Self-Attention (Scaled Dot-Product Attention) is the core mechanism of Transformer models. Each token in a sequence attends to every other token, computing how much "attention" to pay to each position when building its representation.

For each token, three vectors are computed: Query (Q), Key (K), and Value (V). Attention scores are computed as: Attention(Q,K,V) = softmax(QKT / √d_k) · V

The resulting attention matrix shows which tokens attend to which. For example, in "The cat sat on the mat", the word "sat" might strongly attend to "cat" (its subject) and "mat" (related by preposition). This enables the model to capture long-range dependencies without recurrence.

All-to-All

Every token attends to every other token in the sequence.

Q, K, V

Query asks, Key answers relevance, Value provides content.

Multi-Head

Multiple attention heads capture different relationship types.

Parallelizable

Unlike RNNs, all positions computed simultaneously.

Implementation

function scaledDotProductAttention(Q, K, V) {
    // Q, K, V: arrays of shape [seqLen, dim]
    var dk = Q[0].length;
    var scale = Math.sqrt(dk);
    var seqLen = Q.length;

    // Compute attention scores: QK^T / sqrt(dk)
    var scores = [];
    for (var i = 0; i < seqLen; i++) {
        scores[i] = [];
        for (var j = 0; j < seqLen; j++) {
            var dot = 0;
            for (var d = 0; d < dk; d++) {
                dot += Q[i][d] * K[j][d];
            }
            scores[i][j] = dot / scale;
        }
    }

    // Apply softmax per row
    var attention = scores.map(function(row) {
        var maxVal = Math.max.apply(null, row);
        var exps = row.map(function(v) { return Math.exp(v - maxVal); });
        var sum = exps.reduce(function(a, b) { return a + b; }, 0);
        return exps.map(function(e) { return e / sum; });
    });

    // Weighted sum of values
    var output = [];
    for (var i = 0; i < seqLen; i++) {
        output[i] = new Array(dk).fill(0);
        for (var j = 0; j < seqLen; j++) {
            for (var d = 0; d < dk; d++) {
                output[i][d] += attention[i][j] * V[j][d];
            }
        }
    }
    return { output: output, weights: attention };
}
