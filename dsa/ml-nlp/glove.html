GloVe

Medium Time: O(V²) Space: O(V² + V·d)

How It Works

GloVe (Global Vectors for Word Representation) combines the strengths of count-based methods (like LSA) and prediction-based methods (like Word2Vec). It first builds a global word-word co-occurrence matrix from the entire corpus, then learns embeddings by factorizing this matrix.

The key insight is that ratios of co-occurrence probabilities encode semantic relationships. For example, if "ice" co-occurs frequently with "solid" but not "gas", and "steam" co-occurs with "gas" but not "solid", the ratio of these probabilities captures the semantic distinction between ice and steam.

GloVe optimizes: w_i · w_j + b_i + b_j = log(X_ij), where X_ij is the co-occurrence count, weighted to reduce the influence of very frequent pairs.

Global Statistics

Uses entire corpus co-occurrence matrix, not just local windows.

Ratio Insight

Co-occurrence probability ratios capture semantic meaning.

Weighted Least Squares

Frequent pairs are downweighted to prevent dominance.

Matrix Factorization

Decomposes co-occurrence into low-rank word vectors.

Implementation

function trainGloVe(cooccurrence, dim, iterations, lr, xmax, alpha) {
    var V = cooccurrence.length;
    var W = randomMatrix(V, dim);    // word vectors
    var Wc = randomMatrix(V, dim);   // context vectors
    var bw = new Array(V).fill(0);   // word biases
    var bc = new Array(V).fill(0);   // context biases

    for (var iter = 0; iter < iterations; iter++) {
        var totalLoss = 0;
        for (var i = 0; i < V; i++) {
            for (var j = 0; j < V; j++) {
                var Xij = cooccurrence[i][j];
                if (Xij === 0) continue;

                // Weighting function
                var weight = (Xij < xmax)
                    ? Math.pow(Xij / xmax, alpha)
                    : 1.0;

                var dot = dotProduct(W[i], Wc[j]) + bw[i] + bc[j];
                var diff = dot - Math.log(Xij);
                var loss = weight * diff * diff;
                totalLoss += loss;

                // Gradient update
                var grad = weight * diff;
                for (var d = 0; d < dim; d++) {
                    W[i][d] -= lr * grad * Wc[j][d];
                    Wc[j][d] -= lr * grad * W[i][d];
                }
                bw[i] -= lr * grad;
                bc[j] -= lr * grad;
            }
        }
    }
    // Final embeddings = W + Wc (or just W)
    return W;
}
