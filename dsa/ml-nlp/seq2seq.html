Sequence-to-Sequence

Hard Time: O(T·d²) Space: O(T·d)

How It Works

Sequence-to-Sequence (Seq2Seq) is an encoder-decoder architecture that transforms one sequence into another. The encoder reads the input sequence and compresses it into a fixed-size context vector. The decoder then generates the output sequence token by token, conditioned on this context.

Originally using RNNs/LSTMs, the encoder processes the input left-to-right, building up hidden state. The final hidden state (context vector) captures the meaning of the entire input. The decoder starts with this context and autoregressively generates output tokens.

Seq2Seq is the foundation for machine translation, text summarization, chatbots, and speech-to-text. The attention mechanism was later added to address the information bottleneck of the fixed context vector.

Encoder

Reads input sequence and produces a context representation.

Decoder

Generates output sequence from context, one token at a time.

Variable Lengths

Input and output can have different lengths.

Teacher Forcing

Uses ground truth as decoder input during training for faster convergence.

Implementation

// Seq2Seq with RNN (simplified pseudocode)
function seq2seq(inputSeq, encoder, decoder, maxOutputLen) {
    // ENCODE: process input sequence
    var hidden = encoder.initHidden();
    for (var i = 0; i < inputSeq.length; i++) {
        var result = encoder.forward(inputSeq[i], hidden);
        hidden = result.hidden;
    }
    // hidden is now the context vector

    // DECODE: generate output sequence
    var output = [];
    var decoderInput = SOS_TOKEN; // start-of-sequence
    var decoderHidden = hidden;   // initialize with encoder's final state

    for (var t = 0; t < maxOutputLen; t++) {
        var result = decoder.forward(decoderInput, decoderHidden);
        var token = argMax(result.logits);
        if (token === EOS_TOKEN) break;
        output.push(token);
        decoderInput = token;          // autoregressive
        decoderHidden = result.hidden;
    }
    return output;
}
