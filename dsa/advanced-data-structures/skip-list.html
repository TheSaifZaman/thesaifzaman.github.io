Skip List

Medium Search: O(log n) avg Insert: O(log n) avg Space: O(n)

How It Works

A Skip List is a probabilistic data structure that allows O(log n) average-case search, insertion, and deletion in a sorted sequence. It extends a sorted linked list with multiple layers of "express lanes" that skip over intermediate elements, similar to how an express train skips stations.

The bottom level is a regular sorted linked list. Each higher level acts as an express lane, containing a random subset of the elements below. When searching, we start at the top level and move right until we would overshoot, then drop down a level. This produces a binary-search-like traversal with expected O(log n) comparisons.

Element promotion to higher levels is decided by a coin flip (typically 50% probability). This randomization means no single input sequence consistently produces worst-case behavior, unlike BSTs. Skip Lists are used in Redis sorted sets, LevelDB, and Apache HBase.

Key Properties

  • Probabilistic Balance: Randomized levels avoid worst-case degeneration.
  • Simple Implementation: Easier to implement than balanced BSTs like AVL or Red-Black trees.
  • Lock-Free Friendly: Concurrent skip lists are simpler than concurrent balanced trees.
  • Expected O(log n): Average height is O(log n) with high probability.

Implementation

class SkipNode {
  constructor(val, level) {
    this.val = val;
    this.next = new Array(level + 1).fill(null);
  }
}

class SkipList {
  constructor(maxLevel) {
    this.maxLevel = maxLevel || 16;
    this.head = new SkipNode(-Infinity, this.maxLevel);
    this.level = 0;
  }

  randomLevel() {
    var lvl = 0;
    while (Math.random() < 0.5 && lvl < this.maxLevel) lvl++;
    return lvl;
  }

  search(target) {
    var curr = this.head;
    for (var i = this.level; i >= 0; i--) {
      while (curr.next[i] && curr.next[i].val < target)
        curr = curr.next[i];
    }
    curr = curr.next[0];
    return curr && curr.val === target;
  }

  insert(val) {
    var update = new Array(this.maxLevel + 1).fill(null);
    var curr = this.head;
    for (var i = this.level; i >= 0; i--) {
      while (curr.next[i] && curr.next[i].val < val)
        curr = curr.next[i];
      update[i] = curr;
    }
    var lvl = this.randomLevel();
    if (lvl > this.level) {
      for (var j = this.level + 1; j <= lvl; j++) update[j] = this.head;
      this.level = lvl;
    }
    var node = new SkipNode(val, lvl);
    for (var k = 0; k <= lvl; k++) {
      node.next[k] = update[k].next[k];
      update[k].next[k] = node;
    }
  }
}
