Splay Tree

Hard Amortized: O(log n) Worst case: O(n) Space: O(n)

How It Works

A Splay Tree is a self-adjusting binary search tree that moves recently accessed elements to the root through a series of rotations called splaying. Every access (search, insert, delete) triggers a splay operation on the accessed node, bringing it to the root. This provides amortized O(log n) time per operation.

Splaying uses three rotation patterns: Zig (single rotation when parent is root), Zig-Zig (two same-direction rotations), and Zig-Zag (two opposite-direction rotations). The zig-zig case is crucial — it restructures the path, preventing the tree from degenerating. Simple rotation-to-root (without zig-zig) does not achieve amortized O(log n).

Splay Trees exploit temporal locality: frequently accessed elements stay near the root. This makes them optimal for non-uniform access patterns like caches, where some elements are accessed much more often than others. They are used in Windows NT's virtual memory manager and GCC's memoization tables.

Key Properties

  • Self-Adjusting: No stored balance information — the tree restructures on every access.
  • Amortized O(log n): Any sequence of m operations on n nodes takes O(m log n) total.
  • Temporal Locality: Recently and frequently accessed items are fast to access again.
  • Working Set Property: Access time adapts to the access pattern, not just input size.

Implementation

class SplayNode {
  constructor(key) {
    this.key = key;
    this.left = null;
    this.right = null;
  }
}

function rightRotate(x) {
  var y = x.left;
  x.left = y.right;
  y.right = x;
  return y;
}

function leftRotate(x) {
  var y = x.right;
  x.right = y.left;
  y.left = x;
  return y;
}

function splay(root, key) {
  if (!root || root.key === key) return root;

  if (key < root.key) {
    if (!root.left) return root;
    if (key < root.left.key) {         // Zig-Zig
      root.left.left = splay(root.left.left, key);
      root = rightRotate(root);
    } else if (key > root.left.key) {  // Zig-Zag
      root.left.right = splay(root.left.right, key);
      if (root.left.right) root.left = leftRotate(root.left);
    }
    return root.left ? rightRotate(root) : root;
  } else {
    if (!root.right) return root;
    if (key > root.right.key) {        // Zig-Zig
      root.right.right = splay(root.right.right, key);
      root = leftRotate(root);
    } else if (key < root.right.key) { // Zig-Zag
      root.right.left = splay(root.right.left, key);
      if (root.right.left) root.right = rightRotate(root.right);
    }
    return root.right ? leftRotate(root) : root;
  }
}
