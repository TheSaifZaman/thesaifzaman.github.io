Segment Trees

Hard Build: O(n) Query: O(log n) Update: O(log n) Space: O(n)

How It Works

A Segment Tree is a binary tree data structure used for storing information about intervals or segments. Each node stores aggregate information (sum, minimum, maximum, GCD, etc.) for a range of the original array. The root represents the entire array, and each leaf represents a single element.

The tree is built bottom-up: leaves hold individual array values, and each internal node stores the result of merging its two children. For a range sum query, an internal node holds the sum of its left and right children. This structure allows both range queries and point updates in O(log n) time.

Segment Trees are extremely versatile. With lazy propagation, they also support range updates in O(log n). They are essential for competitive programming and appear in problems involving range queries with modifications, such as Range Sum Query Mutable, counting inversions, and interval scheduling.

Key Properties

  • Range Queries: Answer aggregate queries over any subarray in O(log n).
  • Point Updates: Modify a single element and update the tree in O(log n).
  • Lazy Propagation: Enables range updates in O(log n) by deferring updates.
  • Flexible Merge: Works with any associative operation (sum, min, max, GCD, XOR).

Implementation

class SegmentTree {
  constructor(arr) {
    this.n = arr.length;
    this.tree = new Array(4 * this.n).fill(0);
    this.build(arr, 1, 0, this.n - 1);
  }

  build(arr, node, start, end) {
    if (start === end) {
      this.tree[node] = arr[start];
    } else {
      var mid = Math.floor((start + end) / 2);
      this.build(arr, 2 * node, start, mid);
      this.build(arr, 2 * node + 1, mid + 1, end);
      this.tree[node] = this.tree[2 * node] + this.tree[2 * node + 1];
    }
  }

  update(idx, val, node, start, end) {
    if (start === end) {
      this.tree[node] = val;
    } else {
      var mid = Math.floor((start + end) / 2);
      if (idx <= mid) this.update(idx, val, 2 * node, start, mid);
      else this.update(idx, val, 2 * node + 1, mid + 1, end);
      this.tree[node] = this.tree[2 * node] + this.tree[2 * node + 1];
    }
  }

  query(l, r, node, start, end) {
    if (r < start || end < l) return 0;
    if (l <= start && end <= r) return this.tree[node];
    var mid = Math.floor((start + end) / 2);
    return this.query(l, r, 2 * node, start, mid) +
           this.query(l, r, 2 * node + 1, mid + 1, end);
  }
}
