Bitwise Trie (Binary Trie)

Medium Insert: O(B) Query: O(B) Space: O(n × B)

How It Works

A Bitwise Trie (Binary Trie) stores integers by their binary representation. Each node has at most two children: one for bit 0 and one for bit 1. Numbers are inserted from the most significant bit (MSB) to the least significant bit (LSB), creating paths of length B (typically 30 or 31 for 32-bit integers).

The killer application is finding the maximum XOR of a given number with any number in the trie. For each bit of the query number, we greedily choose the opposite bit (to maximize XOR). If the opposite child exists, we go there; otherwise, we take the same bit. This finds the maximum XOR pair in O(B) per query.

Bitwise Tries are used in IP routing tables (longest prefix match), XOR-related competitive programming problems, and network address lookups. They can be augmented with subtree counts for problems involving XOR with constraints (e.g., count pairs with XOR greater than K).

Key Properties

  • Binary Branching: Each node has at most 2 children (bit 0 and bit 1).
  • MSB to LSB: Numbers are processed from the most significant bit down.
  • Maximum XOR: Greedily pick opposite bits to maximize XOR in O(B) time.
  • Fixed Depth: All paths have exactly B levels, where B is the bit width.

Implementation

class BitwiseTrie {
  constructor(maxBit) {
    this.maxBit = maxBit || 30; // for values up to ~10^9
    this.root = {};
  }

  insert(num) {
    var node = this.root;
    for (var i = this.maxBit; i >= 0; i--) {
      var bit = (num >> i) & 1;
      if (!node[bit]) node[bit] = {};
      node = node[bit];
    }
  }

  // Find number in trie that maximizes XOR with num
  maxXor(num) {
    var node = this.root;
    var result = 0;
    for (var i = this.maxBit; i >= 0; i--) {
      var bit = (num >> i) & 1;
      var want = 1 - bit; // opposite bit for max XOR
      if (node[want]) {
        result |= (1 << i);
        node = node[want];
      } else {
        node = node[bit];
      }
    }
    return result;
  }
}

// Maximum XOR of Two Numbers in an Array
function findMaximumXOR(nums) {
  var trie = new BitwiseTrie();
  var maxVal = 0;
  trie.insert(nums[0]);
  for (var i = 1; i < nums.length; i++) {
    maxVal = Math.max(maxVal, trie.maxXor(nums[i]));
    trie.insert(nums[i]);
  }
  return maxVal;
}
