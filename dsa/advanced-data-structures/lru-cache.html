LRU Cache

Medium Get: O(1) Put: O(1) Space: O(capacity)

How It Works

An LRU (Least Recently Used) Cache is a data structure that maintains a fixed-size collection of key-value pairs, evicting the least recently accessed entry when capacity is exceeded. It combines a hash map for O(1) key lookup with a doubly linked list for O(1) order maintenance.

On every get or put, the accessed node is moved to the head of the linked list (most recently used). When a new entry is added and the cache is full, the tail node (least recently used) is removed. The hash map stores key-to-node pointers for instant access, while the linked list tracks access order.

LRU Caches are fundamental in systems design: CPU caches, database buffer pools, web browser caches, CDN edge caches, and operating system page replacement all use LRU or LRU-variant policies. LeetCode #146 is one of the most frequently asked interview questions.

Key Properties

  • O(1) Operations: Both get and put run in constant time.
  • Fixed Capacity: Automatically evicts least recently used entries.
  • HashMap + DLL: Combines two structures for optimal performance.
  • Recency Tracking: Most recently accessed items stay at the front.

Implementation

class LRUCache {
  constructor(capacity) {
    this.capacity = capacity;
    this.map = new Map();
    // Sentinel nodes for doubly linked list
    this.head = { key: null, val: null, prev: null, next: null };
    this.tail = { key: null, val: null, prev: null, next: null };
    this.head.next = this.tail;
    this.tail.prev = this.head;
  }

  _remove(node) {
    node.prev.next = node.next;
    node.next.prev = node.prev;
  }

  _addToFront(node) {
    node.next = this.head.next;
    node.prev = this.head;
    this.head.next.prev = node;
    this.head.next = node;
  }

  get(key) {
    if (!this.map.has(key)) return -1;
    var node = this.map.get(key);
    this._remove(node);
    this._addToFront(node);
    return node.val;
  }

  put(key, value) {
    if (this.map.has(key)) {
      this._remove(this.map.get(key));
    }
    var node = { key: key, val: value, prev: null, next: null };
    this._addToFront(node);
    this.map.set(key, node);
    if (this.map.size > this.capacity) {
      var lru = this.tail.prev;
      this._remove(lru);
      this.map.delete(lru.key);
    }
  }
}
