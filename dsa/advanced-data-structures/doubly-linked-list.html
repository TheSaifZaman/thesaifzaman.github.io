Doubly Linked List

Easy Insert/Delete: O(1) at known node Search: O(n) Space: O(n)

How It Works

A Doubly Linked List is a linear data structure where each node contains a value, a pointer to the next node, and a pointer to the previous node. Unlike singly linked lists, doubly linked lists allow traversal in both directions and enable O(1) deletion of a node when you have a reference to it.

Sentinel (dummy) head and tail nodes simplify edge cases by eliminating null checks. Every insertion and deletion becomes a uniform pointer update between two existing nodes, making the code cleaner and less error-prone.

Doubly linked lists are the backbone of LRU caches, browser forward/back navigation, undo/redo systems, and memory allocators (free lists). Combined with a hash map, they enable O(1) access, insertion, and deletion — the classic LRU Cache design pattern.

Key Properties

  • Bidirectional: Traverse forward and backward through the list.
  • O(1) Delete: Remove any node in constant time given a reference to it.
  • Sentinel Nodes: Dummy head/tail eliminate edge case handling.
  • No Index Access: Unlike arrays, O(n) to access element by position.

Implementation

class DLLNode {
  constructor(val) {
    this.val = val;
    this.prev = null;
    this.next = null;
  }
}

class DoublyLinkedList {
  constructor() {
    this.head = new DLLNode(null); // sentinel
    this.tail = new DLLNode(null); // sentinel
    this.head.next = this.tail;
    this.tail.prev = this.head;
    this.size = 0;
  }

  addToFront(val) {
    var node = new DLLNode(val);
    node.next = this.head.next;
    node.prev = this.head;
    this.head.next.prev = node;
    this.head.next = node;
    this.size++;
    return node;
  }

  addToBack(val) {
    var node = new DLLNode(val);
    node.prev = this.tail.prev;
    node.next = this.tail;
    this.tail.prev.next = node;
    this.tail.prev = node;
    this.size++;
    return node;
  }

  remove(node) {
    node.prev.next = node.next;
    node.next.prev = node.prev;
    this.size--;
    return node.val;
  }

  toArray() {
    var arr = [], curr = this.head.next;
    while (curr !== this.tail) { arr.push(curr.val); curr = curr.next; }
    return arr;
  }
}
