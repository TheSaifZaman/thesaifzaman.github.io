Hash Map Internals

Medium Avg: O(1) get/put Worst: O(n) Space: O(n)

How It Works

A Hash Map stores key-value pairs using an array of "buckets." A hash function converts each key into an array index. When two keys hash to the same index (a collision), the collision is resolved using either chaining (linked list at each bucket) or open addressing (probing for the next empty slot).

The load factor (n/m, elements/buckets) determines performance. When it exceeds a threshold (typically 0.75), the array is resized (usually doubled) and all elements are rehashed. This rehashing is O(n) but happens infrequently, giving amortized O(1) insertions.

A good hash function distributes keys uniformly across buckets. Java's HashMap uses chaining with linked lists that convert to red-black trees when a bucket exceeds 8 entries. Python's dict uses open addressing with pseudorandom probing. Understanding these internals helps you choose the right hash function and predict performance in adversarial scenarios.

Key Properties

  • Hash Function: Maps keys to indices; must be deterministic and distribute uniformly.
  • Collision Resolution: Chaining (linked lists) or open addressing (linear/quadratic probing).
  • Load Factor: n/m ratio; rehash when it exceeds threshold for O(1) amortized ops.
  • Rehashing: Resize array and reinsert all elements when load factor is too high.

Implementation

class HashMap {
  constructor(capacity) {
    this.capacity = capacity || 16;
    this.size = 0;
    this.buckets = new Array(this.capacity).fill(null).map(() => []);
    this.loadFactorThreshold = 0.75;
  }

  _hash(key) {
    var hash = 0;
    var str = String(key);
    for (var i = 0; i < str.length; i++) {
      hash = (hash * 31 + str.charCodeAt(i)) & 0x7FFFFFFF;
    }
    return hash % this.capacity;
  }

  put(key, value) {
    var idx = this._hash(key);
    var bucket = this.buckets[idx];
    for (var i = 0; i < bucket.length; i++) {
      if (bucket[i][0] === key) { bucket[i][1] = value; return; }
    }
    bucket.push([key, value]);
    this.size++;
    if (this.size / this.capacity > this.loadFactorThreshold) this._resize();
  }

  get(key) {
    var idx = this._hash(key);
    var bucket = this.buckets[idx];
    for (var i = 0; i < bucket.length; i++) {
      if (bucket[i][0] === key) return bucket[i][1];
    }
    return undefined;
  }

  _resize() {
    var oldBuckets = this.buckets;
    this.capacity *= 2;
    this.buckets = new Array(this.capacity).fill(null).map(() => []);
    this.size = 0;
    for (var i = 0; i < oldBuckets.length; i++) {
      for (var j = 0; j < oldBuckets[i].length; j++) {
        this.put(oldBuckets[i][j][0], oldBuckets[i][j][1]);
      }
    }
  }
}
