Union-Find (Disjoint Set Union)

Medium Find: O(α(n)) Union: O(α(n)) Space: O(n)

How It Works

Union-Find (also called Disjoint Set Union or DSU) is a data structure that tracks a set of elements partitioned into disjoint subsets. It supports two primary operations: Find, which determines which subset a particular element belongs to, and Union, which merges two subsets into one.

Two key optimizations make Union-Find nearly O(1) per operation: Path Compression flattens the tree during Find by making every node point directly to the root. Union by Rank always attaches the shorter tree under the root of the taller tree, keeping the structure balanced.

With both optimizations, the amortized time per operation is O(alpha(n)), where alpha is the inverse Ackermann function — effectively constant for all practical input sizes. Union-Find is widely used in Kruskal's MST algorithm, cycle detection, connected components, and network connectivity problems.

Key Properties

  • Path Compression: Flattens tree structure for near-constant lookups.
  • Union by Rank: Keeps trees balanced to prevent degenerate chains.
  • Connected Components: Efficiently tracks and merges groups of elements.
  • Cycle Detection: If two nodes share the same root, adding an edge creates a cycle.

Implementation

class UnionFind {
  constructor(n) {
    this.parent = Array.from({length: n}, (_, i) => i);
    this.rank = new Array(n).fill(0);
  }

  find(x) {
    if (this.parent[x] !== x) {
      this.parent[x] = this.find(this.parent[x]); // path compression
    }
    return this.parent[x];
  }

  union(x, y) {
    var rootX = this.find(x);
    var rootY = this.find(y);
    if (rootX === rootY) return false;
    // union by rank
    if (this.rank[rootX] < this.rank[rootY]) {
      this.parent[rootX] = rootY;
    } else if (this.rank[rootX] > this.rank[rootY]) {
      this.parent[rootY] = rootX;
    } else {
      this.parent[rootY] = rootX;
      this.rank[rootX]++;
    }
    return true;
  }

  connected(x, y) {
    return this.find(x) === this.find(y);
  }
}
