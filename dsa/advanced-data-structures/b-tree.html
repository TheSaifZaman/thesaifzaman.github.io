B-Tree

Hard Search: O(log n) Insert: O(log n) Delete: O(log n) Space: O(n)

How It Works

A B-Tree is a self-balancing search tree designed for systems that read and write large blocks of data, such as databases and file systems. Unlike binary search trees, each node in a B-Tree can contain multiple keys and have multiple children. A B-Tree of order m allows each node to have up to m children and m-1 keys.

All leaves are at the same depth, ensuring the tree is always perfectly balanced. When a node overflows (exceeds m-1 keys), it splits into two nodes and pushes the median key up to the parent. This splitting propagates upward only when necessary, keeping the tree balanced with minimal restructuring.

B-Trees minimize disk I/O by storing many keys per node, reducing tree height. A B-Tree with branching factor 1000 and height 3 can index over a billion records. This is why databases like MySQL (InnoDB) and PostgreSQL use B-Tree variants (B+ Trees) for their indexes.

Key Properties

  • Multi-way: Each node holds up to m-1 keys and m children.
  • Balanced: All leaves are at the same depth at all times.
  • Disk-Optimized: Wide nodes reduce tree height, minimizing I/O operations.
  • Split on Overflow: Nodes split and push median up when full.

Implementation

class BTreeNode {
  constructor(t, leaf) {
    this.t = t;           // minimum degree
    this.keys = [];
    this.children = [];
    this.leaf = leaf;
  }

  search(k) {
    var i = 0;
    while (i < this.keys.length && k > this.keys[i]) i++;
    if (i < this.keys.length && this.keys[i] === k) return this;
    if (this.leaf) return null;
    return this.children[i].search(k);
  }

  splitChild(i) {
    var t = this.t;
    var y = this.children[i];
    var z = new BTreeNode(t, y.leaf);
    z.keys = y.keys.splice(t);       // right half
    var median = y.keys.pop();        // median key
    if (!y.leaf) z.children = y.children.splice(t);
    this.children.splice(i + 1, 0, z);
    this.keys.splice(i, 0, median);
  }

  insertNonFull(k) {
    var i = this.keys.length - 1;
    if (this.leaf) {
      while (i >= 0 && this.keys[i] > k) i--;
      this.keys.splice(i + 1, 0, k);
    } else {
      while (i >= 0 && this.keys[i] > k) i--;
      i++;
      if (this.children[i].keys.length === 2 * this.t - 1) {
        this.splitChild(i);
        if (k > this.keys[i]) i++;
      }
      this.children[i].insertNonFull(k);
    }
  }
}
