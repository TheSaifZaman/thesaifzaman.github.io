Treap

Hard Search: O(log n) expected Insert: O(log n) expected Delete: O(log n) expected Space: O(n)

How It Works

A Treap is a combination of a binary search tree and a heap. Each node stores a key (maintaining BST property) and a random priority (maintaining heap property). The BST property ensures keys are ordered left-to-right, while the heap property (on random priorities) keeps the tree probabilistically balanced.

Insertion works by first inserting like a standard BST, then assigning a random priority and rotating the node upward until the heap property is restored. The key operation is split (divide tree by key) and merge (combine two treaps), which form the basis of an elegant implicit treap that can handle array operations like insert-at-position, delete-range, and reverse-range.

Treaps provide expected O(log n) operations with simpler code than AVL or Red-Black trees. Implicit treaps (using subtree sizes instead of keys) are powerful competitive programming tools that support all sequence operations — insert, delete, reverse, and aggregate queries — in O(log n).

Key Properties

  • Randomized Balance: Random priorities ensure expected O(log n) height.
  • Split/Merge: Two fundamental operations enable all treap manipulations.
  • Implicit Treap: Uses subtree sizes to support array operations in O(log n).
  • Simpler than AVL/RB: No complex rotation cases — just split and merge.

Implementation

class TreapNode {
  constructor(key) {
    this.key = key;
    this.priority = Math.random();
    this.left = null;
    this.right = null;
  }
}

function split(node, key) {
  if (!node) return [null, null];
  if (key <= node.key) {
    var [left, right] = split(node.left, key);
    node.left = right;
    return [left, node];
  } else {
    var [left, right] = split(node.right, key);
    node.right = left;
    return [node, right];
  }
}

function merge(left, right) {
  if (!left || !right) return left || right;
  if (left.priority > right.priority) {
    left.right = merge(left.right, right);
    return left;
  } else {
    right.left = merge(left, right.left);
    return right;
  }
}

function insert(root, key) {
  var node = new TreapNode(key);
  var [left, right] = split(root, key);
  return merge(merge(left, node), right);
}

function erase(root, key) {
  if (!root) return null;
  if (key === root.key) return merge(root.left, root.right);
  if (key < root.key) root.left = erase(root.left, key);
  else root.right = erase(root.right, key);
  return root;
}
