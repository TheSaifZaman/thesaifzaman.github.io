Sparse Table

Medium Build: O(n log n) Query: O(1) Space: O(n log n)

How It Works

A Sparse Table is a data structure for answering static range queries (no updates) in O(1) time after O(n log n) preprocessing. It works for idempotent operations like minimum, maximum, and GCD, where applying the operation to overlapping ranges gives the correct result.

The table stores precomputed answers for all ranges of length 2^k. Entry sparse[i][k] holds the answer for the range starting at index i with length 2^k. Building the table uses dynamic programming: sparse[i][k] = merge(sparse[i][k-1], sparse[i + 2^(k-1)][k-1]).

For a query over range [l, r], we find the largest k such that 2^k fits within the range, then merge the two overlapping sub-ranges: merge(sparse[l][k], sparse[r - 2^k + 1][k]). Since min/max/GCD are idempotent, the overlap does not affect correctness. This gives O(1) queries without any updates — perfect for static arrays.

Key Properties

  • O(1) Queries: After preprocessing, each query is answered in constant time.
  • Idempotent Only: Works for min, max, GCD — not for sum (overlapping ranges double-count).
  • Static: No support for updates — use Segment Tree or Fenwick Tree if updates needed.
  • LCA Applications: Sparse Tables enable O(1) Lowest Common Ancestor queries via Euler tour.

Implementation

class SparseTable {
  constructor(arr) {
    var n = arr.length;
    var LOG = Math.floor(Math.log2(n)) + 1;
    this.sparse = [];
    this.log = new Array(n + 1).fill(0);

    // Precompute logs
    for (var i = 2; i <= n; i++)
      this.log[i] = this.log[Math.floor(i / 2)] + 1;

    // Initialize level 0
    this.sparse[0] = arr.slice();

    // Build sparse table
    for (var k = 1; k < LOG; k++) {
      this.sparse[k] = [];
      for (var j = 0; j + (1 << k) <= n; j++) {
        this.sparse[k][j] = Math.min(
          this.sparse[k-1][j],
          this.sparse[k-1][j + (1 << (k-1))]
        );
      }
    }
  }

  // O(1) range minimum query
  query(l, r) {
    var k = this.log[r - l + 1];
    return Math.min(
      this.sparse[k][l],
      this.sparse[k][r - (1 << k) + 1]
    );
  }
}
