Fenwick Tree (Binary Indexed Tree)

Medium Update: O(log n) Prefix Query: O(log n) Space: O(n)

How It Works

A Fenwick Tree (Binary Indexed Tree or BIT) is a data structure that efficiently supports prefix sum queries and point updates. It uses a clever bit-manipulation trick: each index i is responsible for a range of elements determined by the lowest set bit of i. For example, index 8 (binary 1000) covers 8 elements, while index 6 (binary 110) covers 2 elements.

To compute a prefix sum from index 1 to i, we add tree[i] and repeatedly strip the lowest set bit (i -= i & (-i)) until i becomes 0. To update index i, we add the delta and repeatedly add the lowest set bit (i += i & (-i)) until i exceeds n. Both operations visit at most O(log n) nodes.

Fenwick Trees are simpler to implement than Segment Trees (only a few lines of code) and have lower constant factors. They are ideal for problems requiring prefix sum queries with updates, counting inversions, and coordinate compression problems. However, they are less flexible than Segment Trees since they naturally support only prefix operations.

Key Properties

  • Compact: Uses only n+1 array elements — no 4n overhead like Segment Trees.
  • Bit Tricks: Lowest set bit (i & -i) determines each node's responsibility range.
  • Prefix Sums: Range sum [l, r] = prefix(r) - prefix(l-1).
  • Simple Code: Both update and query are 3-4 lines each.

Implementation

class FenwickTree {
  constructor(n) {
    this.n = n;
    this.tree = new Array(n + 1).fill(0);
  }

  // Add delta to index i (1-indexed)
  update(i, delta) {
    for (; i <= this.n; i += i & (-i))
      this.tree[i] += delta;
  }

  // Prefix sum from 1 to i
  query(i) {
    var sum = 0;
    for (; i > 0; i -= i & (-i))
      sum += this.tree[i];
    return sum;
  }

  // Range sum from l to r (inclusive)
  rangeQuery(l, r) {
    return this.query(r) - this.query(l - 1);
  }

  // Build from array
  static fromArray(arr) {
    var bit = new FenwickTree(arr.length);
    for (var i = 0; i < arr.length; i++)
      bit.update(i + 1, arr[i]);
    return bit;
  }
}
