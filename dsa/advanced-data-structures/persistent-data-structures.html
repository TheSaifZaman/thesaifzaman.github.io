Persistent Data Structures

Hard Update: O(log n) Query any version: O(log n) Space: O(n log n)

How It Works

A Persistent Data Structure preserves all previous versions of itself when modified. Instead of modifying nodes in-place, each update creates new nodes along the path from root to the modified leaf, sharing all unchanged subtrees with the previous version. This is called path copying.

For a balanced tree of height O(log n), each update creates O(log n) new nodes while sharing O(n) existing nodes. This gives O(log n) time and space per update, and allows querying any historical version in O(log n) time. The total space after m updates is O(n + m log n).

Persistent data structures are fundamental in functional programming (immutable data), version control systems (Git's object model), database MVCC (multi-version concurrency control), and competitive programming (persistent segment trees for offline range queries across time).

Key Properties

  • Version History: Every version is accessible — updates never destroy previous state.
  • Path Copying: Only nodes on the root-to-change path are duplicated.
  • Structural Sharing: Unchanged subtrees are shared across versions.
  • Immutability: Each version is immutable, enabling safe concurrent access.

Implementation

// Persistent Segment Tree
class PersistentNode {
  constructor(val, left, right) {
    this.val = val;
    this.left = left || null;
    this.right = right || null;
  }
}

function build(arr, lo, hi) {
  if (lo === hi) return new PersistentNode(arr[lo]);
  var mid = (lo + hi) >> 1;
  var left = build(arr, lo, mid);
  var right = build(arr, mid + 1, hi);
  return new PersistentNode(left.val + right.val, left, right);
}

// Returns new root (new version)
function update(node, lo, hi, idx, val) {
  if (lo === hi) return new PersistentNode(val);
  var mid = (lo + hi) >> 1;
  if (idx <= mid) {
    var newLeft = update(node.left, lo, mid, idx, val);
    return new PersistentNode(newLeft.val + node.right.val, newLeft, node.right);
  } else {
    var newRight = update(node.right, mid + 1, hi, idx, val);
    return new PersistentNode(node.left.val + newRight.val, node.left, newRight);
  }
}

function query(node, lo, hi, l, r) {
  if (r < lo || hi < l) return 0;
  if (l <= lo && hi <= r) return node.val;
  var mid = (lo + hi) >> 1;
  return query(node.left, lo, mid, l, r) +
         query(node.right, mid + 1, hi, l, r);
}

// Usage
var roots = [];
roots[0] = build([1,2,3,4,5], 0, 4);
roots[1] = update(roots[0], 0, 4, 2, 10); // version 1: arr[2]=10
query(roots[0], 0, 4, 0, 4); // 15 (original)
query(roots[1], 0, 4, 0, 4); // 22 (updated)
