Bloom Filter

Medium Insert: O(k) Lookup: O(k) Space: O(m) bits

How It Works

A Bloom Filter is a space-efficient probabilistic data structure that tests whether an element is a member of a set. It can tell you "definitely not in the set" or "possibly in the set" — it never produces false negatives, but may produce false positives.

It consists of a bit array of m bits (initially all 0) and k independent hash functions. To add an element, compute k hash values and set those bit positions to 1. To query, check if all k bit positions are set — if any is 0, the element is definitely absent. If all are 1, the element is probably present (with a tunable false positive rate).

Bloom Filters are used extensively in practice: web browsers check URLs against phishing databases, databases avoid unnecessary disk reads (LSM-tree levels), distributed systems reduce network round trips, and spell checkers quickly reject non-words. The false positive rate is approximately (1 - e^(-kn/m))^k, where n is the number of inserted elements.

Key Properties

  • No False Negatives: If lookup says absent, the element is guaranteed absent.
  • False Positives: Lookup may report present for absent elements (tunable rate).
  • No Deletion: Standard Bloom Filters cannot remove elements (Counting Bloom Filters can).
  • Space Efficient: Uses far less memory than storing actual elements.

Implementation

class BloomFilter {
  constructor(size, numHashes) {
    this.size = size;
    this.numHashes = numHashes;
    this.bits = new Array(size).fill(0);
  }

  // Simple hash functions using multiplication method
  _hash(value, seed) {
    var hash = 0;
    var str = String(value);
    for (var i = 0; i < str.length; i++) {
      hash = (hash * seed + str.charCodeAt(i)) & 0x7FFFFFFF;
    }
    return hash % this.size;
  }

  add(value) {
    for (var i = 0; i < this.numHashes; i++) {
      var idx = this._hash(value, 31 + i * 37);
      this.bits[idx] = 1;
    }
  }

  mightContain(value) {
    for (var i = 0; i < this.numHashes; i++) {
      var idx = this._hash(value, 31 + i * 37);
      if (this.bits[idx] === 0) return false; // definitely absent
    }
    return true; // possibly present
  }
}

// Usage: ~1% false positive rate with m/n ~ 10, k ~ 7
var bf = new BloomFilter(100, 3);
bf.add("hello");
bf.mightContain("hello"); // true
bf.mightContain("world"); // false (probably)
