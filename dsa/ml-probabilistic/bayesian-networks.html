Bayesian Networks

Hard Inference: O(2k) Space: O(n·2k)

How It Works

A Bayesian Network is a directed acyclic graph (DAG) where each node represents a random variable and edges represent probabilistic dependencies. Each node has a conditional probability table (CPT) encoding P(node | parents). The joint distribution factorizes as the product of all conditional probabilities.

Inference computes the posterior probability of query variables given evidence. Exact inference (variable elimination) is NP-hard in general, but tractable for tree-structured networks. Approximate methods like loopy belief propagation and MCMC are used for complex networks.

Key Properties

  • DAG structure: No cycles; edges encode causal or correlational relationships.
  • Conditional independence: d-separation determines which variables are independent given evidence.
  • Factorization: P(X1,...,Xn) = ∏ P(Xi | Parents(Xi)).
  • Learning: Structure learning (finding DAG) + parameter learning (fitting CPTs).

Implementation

function bayesianInference(network, query, evidence) {
    // Variable elimination (simplified)
    var factors = network.nodes.map(function(n) {
        return n.cpt; // conditional probability table
    });
    // Eliminate hidden variables one by one
    var hidden = network.nodes.filter(function(n) {
        return n.id !== query && !evidence[n.id];
    });
    for (var i = 0; i < hidden.length; i++) {
        factors = sumOut(factors, hidden[i].id);
    }
    // Normalize remaining factor
    var result = multiplyFactors(factors);
    return normalize(result);
}
