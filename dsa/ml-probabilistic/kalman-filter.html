Kalman Filter

Hard Time: O(d³) Space: O(d²)

How It Works

The Kalman Filter is an optimal recursive state estimator for linear-Gaussian systems. It maintains a belief about the system state as a Gaussian distribution (mean and covariance), and updates it in two phases: Predict (propagate state forward using the motion model) and Update (correct using a new measurement).

The filter balances trust between the model prediction and the noisy measurement based on their respective uncertainties. When measurement noise is high, it trusts the prediction more; when process noise is high, it trusts the measurement more.

Key Properties

  • Predict: x̂ = Fx + Bu, P = FPFT + Q (process model).
  • Update: K = PHT(HPHT + R)-1 (Kalman gain).
  • Optimal: Minimizes mean squared error for linear-Gaussian systems.
  • Applications: GPS, robotics, tracking, sensor fusion, finance.

Implementation

function kalmanFilter1D(measurements, processNoise, measureNoise) {
    var x = 0;      // initial state estimate
    var P = 1;      // initial uncertainty
    var Q = processNoise;
    var R = measureNoise;
    var estimates = [];
    for (var i = 0; i < measurements.length; i++) {
        // Predict
        var xPred = x;        // assume constant model
        var PPred = P + Q;
        // Update
        var K = PPred / (PPred + R);  // Kalman gain
        x = xPred + K * (measurements[i] - xPred);
        P = (1 - K) * PPred;
        estimates.push({x: x, P: P, K: K});
    }
    return estimates;
}
