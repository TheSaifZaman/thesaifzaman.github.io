Expectation-Maximization

Hard Time: O(n·k·d) Space: O(n·k)

How It Works

The Expectation-Maximization (EM) algorithm is an iterative method for finding maximum likelihood estimates when the model has latent (hidden) variables. It alternates between two steps: the E-step computes the expected value of the latent variables given current parameters, and the M-step updates parameters to maximize the expected log-likelihood.

EM is guaranteed to increase the log-likelihood at each iteration and converge to a local maximum. It is the standard algorithm for fitting Gaussian Mixture Models, and is also used in HMM training (Baum-Welch), topic models, and missing data imputation.

Key Properties

  • E-step: Compute responsibilities (soft assignments of points to clusters).
  • M-step: Update means, covariances, and mixing weights.
  • Convergence: Monotonically increases likelihood; converges to local maximum.
  • Initialization: Sensitive to starting parameters; often run multiple times.

Implementation

function emGMM(data, k, maxIter) {
    // Initialize means randomly
    var means = randomSample(data, k);
    var weights = new Array(k).fill(1.0 / k);
    var vars = new Array(k).fill(1.0);

    for (var iter = 0; iter < maxIter; iter++) {
        // E-step: compute responsibilities
        var resp = [];
        for (var i = 0; i < data.length; i++) {
            var row = [];
            var total = 0;
            for (var j = 0; j < k; j++) {
                var p = weights[j] * gaussian(data[i], means[j], vars[j]);
                row.push(p);
                total += p;
            }
            resp.push(row.map(function(r) { return r / total; }));
        }
        // M-step: update parameters
        for (var j2 = 0; j2 < k; j2++) {
            var nk = resp.reduce(function(s, r) { return s + r[j2]; }, 0);
            means[j2] = resp.reduce(function(s, r, i) {
                return s + r[j2] * data[i]; }, 0) / nk;
            weights[j2] = nk / data.length;
        }
    }
    return { means: means, weights: weights };
}
