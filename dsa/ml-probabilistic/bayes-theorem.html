Bayes' Theorem

Easy Time: O(1) Space: O(1)

How It Works

Bayes' Theorem describes how to update the probability of a hypothesis when given new evidence. It relates the posterior probability P(H|E) to the prior P(H), the likelihood P(E|H), and the marginal probability of evidence P(E).

The formula is: P(H|E) = P(E|H) · P(H) / P(E). This is the foundation of Bayesian inference, spam filtering, medical diagnosis, and all probabilistic machine learning.

Key Properties

  • Prior P(H): Our belief before seeing evidence.
  • Likelihood P(E|H): Probability of evidence given hypothesis is true.
  • Posterior P(H|E): Updated belief after seeing evidence.
  • Normalization: P(E) = P(E|H)P(H) + P(E|¬H)P(¬H) ensures probabilities sum to 1.

Implementation

function bayesTheorem(priorH, likelihoodEgivenH, likelihoodEgivenNotH) {
    var priorNotH = 1 - priorH;
    var evidenceP = likelihoodEgivenH * priorH +
                    likelihoodEgivenNotH * priorNotH;
    var posterior = (likelihoodEgivenH * priorH) / evidenceP;
    return posterior;
}

// Example: medical test
// P(Disease) = 0.01, P(+|Disease) = 0.95, P(+|Healthy) = 0.05
var result = bayesTheorem(0.01, 0.95, 0.05);
// result = 0.161 — only 16.1% chance of disease given positive test!
