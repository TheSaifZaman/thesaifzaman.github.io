Edge Detection (Sobel/Canny)

Easy Time: O(n·m) Space: O(n·m)

How It Works

Edge detection identifies boundaries in images where pixel intensity changes sharply. The Sobel operator uses two 3×3 kernels to compute horizontal and vertical gradients independently, then combines them to find the edge magnitude and direction at each pixel.

The Canny edge detector extends this with Gaussian smoothing, non-maximum suppression along gradient directions, and hysteresis thresholding (using high and low thresholds) to produce clean, thin edges with minimal noise.

Key Properties

  • Sobel Gx kernel: [[-1,0,1],[-2,0,2],[-1,0,1]] detects vertical edges.
  • Sobel Gy kernel: [[-1,-2,-1],[0,0,0],[1,2,1]] detects horizontal edges.
  • Gradient magnitude: G = √(Gx² + Gy²) gives edge strength.
  • Canny adds: Gaussian blur, NMS, and double-threshold hysteresis.

Implementation

function sobelEdge(image, rows, cols) {
    var Gx = [[-1,0,1],[-2,0,2],[-1,0,1]];
    var Gy = [[-1,-2,-1],[0,0,0],[1,2,1]];
    var result = [];
    for (var r = 1; r < rows - 1; r++) {
        result[r] = [];
        for (var c = 1; c < cols - 1; c++) {
            var sx = 0, sy = 0;
            for (var kr = -1; kr <= 1; kr++) {
                for (var kc = -1; kc <= 1; kc++) {
                    sx += image[r+kr][c+kc] * Gx[kr+1][kc+1];
                    sy += image[r+kr][c+kc] * Gy[kr+1][kc+1];
                }
            }
            result[r][c] = Math.sqrt(sx*sx + sy*sy);
        }
    }
    return result;
}
