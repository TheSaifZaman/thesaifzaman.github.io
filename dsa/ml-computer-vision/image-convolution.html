Image Convolution

Easy Time: O(n·m·k²) Space: O(n·m)

How It Works

Image convolution slides a small kernel (filter) over every pixel in an image. At each position, the kernel values are multiplied element-wise with the overlapping image pixels, and the results are summed to produce one output pixel. This operation is the foundation of all CNN-based vision models.

Different kernels produce different effects: edge detection (Sobel), blurring (Gaussian), sharpening, embossing, and more. The kernel slides left-to-right, top-to-bottom, computing a weighted sum at each position.

Key Properties

  • Kernel: Small matrix (e.g., 3×3) of learnable or fixed weights.
  • Stride: How many pixels the kernel moves each step (default: 1).
  • Padding: Adding zeros around the border to preserve spatial dimensions.
  • Output size: (n − k + 2p) / s + 1 where n=input, k=kernel, p=padding, s=stride.

Implementation

function convolve2D(image, kernel) {
    var rows = image.length, cols = image[0].length;
    var kSize = kernel.length, pad = Math.floor(kSize / 2);
    var output = [];
    for (var r = pad; r < rows - pad; r++) {
        var row = [];
        for (var c = pad; c < cols - pad; c++) {
            var sum = 0;
            for (var kr = 0; kr < kSize; kr++) {
                for (var kc = 0; kc < kSize; kc++) {
                    sum += image[r - pad + kr][c - pad + kc] * kernel[kr][kc];
                }
            }
            row.push(sum);
        }
        output.push(row);
    }
    return output;
}
