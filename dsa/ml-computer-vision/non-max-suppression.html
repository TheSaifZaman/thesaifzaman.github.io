Non-Maximum Suppression

Medium Time: O(n log n) Space: O(n)

How It Works

Non-Maximum Suppression (NMS) is a post-processing step in object detection that eliminates redundant overlapping bounding boxes. After a detector produces many candidate boxes with confidence scores, NMS keeps only the best ones by iteratively selecting the highest-confidence box and removing all boxes that overlap with it above a threshold (typically IoU > 0.5).

The algorithm sorts boxes by confidence, picks the top box, computes IoU with all remaining boxes, discards those exceeding the threshold, and repeats until no boxes remain.

Key Properties

  • Greedy: Always picks the highest-confidence box first.
  • IoU threshold: Typically 0.5; higher keeps more boxes.
  • Soft-NMS: Variant that decays scores instead of hard removal.
  • Per-class: Usually applied independently per object class.

Implementation

function nms(boxes, threshold) {
    // boxes: [{x1,y1,x2,y2,score}]
    boxes.sort(function(a, b) { return b.score - a.score; });
    var keep = [];
    var suppressed = new Array(boxes.length).fill(false);
    for (var i = 0; i < boxes.length; i++) {
        if (suppressed[i]) continue;
        keep.push(boxes[i]);
        for (var j = i + 1; j < boxes.length; j++) {
            if (suppressed[j]) continue;
            if (computeIoU(boxes[i], boxes[j]) > threshold) {
                suppressed[j] = true;
            }
        }
    }
    return keep;
}
