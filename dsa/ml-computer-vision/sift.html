SIFT

Hard Time: O(n·log n) Space: O(n·k)

How It Works

Scale-Invariant Feature Transform (SIFT) detects and describes local features that are invariant to image scale and rotation. It builds a scale-space pyramid using Difference of Gaussians (DoG), finds extrema as candidate keypoints, refines them via sub-pixel localization, assigns orientations, and generates 128-dimensional descriptors.

SIFT keypoints are robust to changes in illumination, noise, and moderate viewpoint changes, making them ideal for image matching, object recognition, and panorama stitching.

Key Properties

  • Scale-space: Gaussian blur at multiple scales, DoG for blob detection.
  • Keypoint localization: Sub-pixel refinement and low-contrast/edge rejection.
  • Orientation: Dominant gradient direction for rotation invariance.
  • Descriptor: 4×4 grid of 8-bin histograms = 128 dimensions.

Implementation

function detectKeypoints(dogPyramid) {
    var keypoints = [];
    for (var s = 1; s < dogPyramid.length - 1; s++) {
        var curr = dogPyramid[s];
        for (var r = 1; r < curr.length - 1; r++) {
            for (var c = 1; c < curr[0].length - 1; c++) {
                var val = curr[r][c];
                var isMax = true, isMin = true;
                // Check 26 neighbors across 3 scales
                for (var ds = -1; ds <= 1; ds++) {
                    for (var dr = -1; dr <= 1; dr++) {
                        for (var dc = -1; dc <= 1; dc++) {
                            if (ds === 0 && dr === 0 && dc === 0) continue;
                            var neighbor = dogPyramid[s+ds][r+dr][c+dc];
                            if (neighbor >= val) isMax = false;
                            if (neighbor <= val) isMin = false;
                        }
                    }
                }
                if (isMax || isMin) {
                    keypoints.push({x: c, y: r, scale: s});
                }
            }
        }
    }
    return keypoints;
}
