Image Augmentation

Easy Time: O(n) Space: O(n)

How It Works

Image augmentation applies random transformations to training images to increase dataset diversity without collecting new data. Common transformations include horizontal/vertical flips, rotations, brightness/contrast changes, cropping, and color jittering.

Augmentation acts as a regularizer, reducing overfitting by ensuring the model sees varied versions of each training sample. Modern pipelines use libraries like albumentations or torchvision.transforms to chain multiple augmentations randomly.

Key Properties

  • Geometric: Flip, rotate, scale, crop, affine transformations.
  • Photometric: Brightness, contrast, saturation, hue shifts.
  • Noise: Gaussian noise, blur, JPEG compression artifacts.
  • Advanced: Mixup, CutMix, CutOut, MosaicAugmentation.

Implementation

function augmentImage(image) {
    var augmented = copyImage(image);
    // Horizontal flip (50% chance)
    if (Math.random() > 0.5) {
        for (var r = 0; r < augmented.length; r++) {
            augmented[r].reverse();
        }
    }
    // Brightness shift
    var shift = Math.floor(Math.random() * 40) - 20;
    for (var r2 = 0; r2 < augmented.length; r2++) {
        for (var c = 0; c < augmented[0].length; c++) {
            augmented[r2][c] = Math.max(0, Math.min(255,
                augmented[r2][c] + shift));
        }
    }
    return augmented;
}
