Intersection over Union (IoU)

Easy Time: O(1) Space: O(1)

How It Works

Intersection over Union (IoU), also called the Jaccard Index, measures the overlap between two bounding boxes. It is the area of intersection divided by the area of union. An IoU of 1.0 means perfect overlap, while 0.0 means no overlap at all.

IoU is the standard metric for evaluating object detection accuracy. A prediction is typically considered correct if IoU ≥ 0.5 with the ground truth box (this threshold can vary by benchmark).

Key Properties

  • Range: [0, 1] where 1 = perfect overlap.
  • Formula: IoU = Area(A ∩ B) / Area(A ∪ B).
  • Threshold: Common thresholds: 0.5 (PASCAL VOC), 0.5:0.95 (COCO).
  • Used in: NMS, mAP calculation, loss functions (GIoU, DIoU, CIoU).

Implementation

function computeIoU(boxA, boxB) {
    var x1 = Math.max(boxA.x1, boxB.x1);
    var y1 = Math.max(boxA.y1, boxB.y1);
    var x2 = Math.min(boxA.x2, boxB.x2);
    var y2 = Math.min(boxA.y2, boxB.y2);
    var intersection = Math.max(0, x2 - x1) * Math.max(0, y2 - y1);
    var areaA = (boxA.x2 - boxA.x1) * (boxA.y2 - boxA.y1);
    var areaB = (boxB.x2 - boxB.x1) * (boxB.y2 - boxB.y1);
    var union = areaA + areaB - intersection;
    return union > 0 ? intersection / union : 0;
}
