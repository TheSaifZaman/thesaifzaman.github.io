Principal Component Analysis

Medium Time: O(d²·n) Space: O(d²)

How It Works

PCA finds the directions (principal components) along which data varies the most. It projects high-dimensional data onto a lower-dimensional subspace while preserving maximum variance. The first principal component captures the most variance, the second captures the next most (orthogonal to the first), and so on.

The algorithm: (1) Center the data by subtracting the mean, (2) Compute the covariance matrix, (3) Find eigenvalues and eigenvectors, (4) Sort by eigenvalue (descending), (5) Project data onto the top k eigenvectors. In 2D, PCA finds the axis of maximum spread and rotates the coordinate system.

Key Properties

  • Linear: Only captures linear relationships.
  • Orthogonal: Principal components are mutually perpendicular.
  • Variance-based: Maximizes retained variance.
  • Deterministic: Same input always gives same result (up to sign).

Implementation

function pca(data, numComponents) {
    var n = data.length, d = data[0].length;

    // 1. Compute mean
    var mean = new Array(d).fill(0);
    data.forEach(function(row) {
        for (var j = 0; j < d; j++) mean[j] += row[j];
    });
    for (var j = 0; j < d; j++) mean[j] /= n;

    // 2. Center data
    var centered = data.map(function(row) {
        return row.map(function(v, j) { return v - mean[j]; });
    });

    // 3. Covariance matrix (d x d)
    var cov = [];
    for (var i = 0; i < d; i++) {
        cov[i] = [];
        for (var j2 = 0; j2 < d; j2++) {
            var s = 0;
            for (var k = 0; k < n; k++) {
                s += centered[k][i] * centered[k][j2];
            }
            cov[i][j2] = s / (n - 1);
        }
    }

    // 4. Eigendecomposition (power iteration for 2D)
    // Returns eigenvectors sorted by eigenvalue
    var eigenvectors = eigenDecompose(cov);

    // 5. Project onto top numComponents
    var projected = centered.map(function(row) {
        var result = [];
        for (var c = 0; c < numComponents; c++) {
            var dot = 0;
            for (var j3 = 0; j3 < d; j3++) {
                dot += row[j3] * eigenvectors[c][j3];
            }
            result.push(dot);
        }
        return result;
    });

    return { projected: projected, components: eigenvectors };
}
