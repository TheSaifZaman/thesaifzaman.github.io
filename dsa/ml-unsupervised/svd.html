Singular Value Decomposition

Medium Time: O(min(mn²,m²n)) Space: O(m·n)

How It Works

SVD decomposes any m×n matrix A into three matrices: A = U·Σ·V^T, where U contains left singular vectors, Σ is a diagonal matrix of singular values, and V^T contains right singular vectors. The singular values represent the importance of each component.

For dimensionality reduction, we keep only the top k singular values and their corresponding vectors (truncated SVD). This gives the best rank-k approximation of the original matrix. SVD is the mathematical foundation behind PCA, LSA (Latent Semantic Analysis), and many recommendation systems.

Key Properties

  • Always exists: Every matrix has an SVD decomposition.
  • Optimal: Truncated SVD gives best low-rank approximation (Eckart-Young).
  • Applications: PCA, LSA, image compression, recommendation.
  • Singular values: Always non-negative, sorted descending.

Implementation

// Conceptual SVD (in practice use LAPACK/numpy)
function svdTruncated(A, k) {
    var m = A.length, n = A[0].length;

    // Compute A^T * A (n x n)
    var AtA = matMul(transpose(A), A);

    // Find eigenvalues/eigenvectors of A^T * A
    var eigen = eigenDecompose(AtA);

    // Singular values = sqrt(eigenvalues)
    var sigma = eigen.values.slice(0, k).map(Math.sqrt);

    // V = eigenvectors of A^T * A
    var V = eigen.vectors.slice(0, k);

    // U = A * V * Sigma^{-1}
    var U = [];
    for (var i = 0; i < k; i++) {
        var col = matVecMul(A, V[i]);
        U.push(col.map(function(v) {
            return v / sigma[i];
        }));
    }

    return { U: U, sigma: sigma, V: V };
}

// Reconstruct: A_approx = U * diag(sigma) * V^T
function reconstruct(U, sigma, V) {
    var m = U[0].length, n = V[0].length;
    var result = zeros(m, n);
    for (var i = 0; i < sigma.length; i++) {
        for (var r = 0; r < m; r++) {
            for (var c = 0; c < n; c++) {
                result[r][c] += sigma[i] *
                    U[i][r] * V[i][c];
            }
        }
    }
    return result;
}
