Isomap

Hard Time: O(n²·log n) Space: O(n²)

How It Works

Isomap (Isometric Mapping) is a nonlinear dimensionality reduction method that preserves geodesic distances rather than Euclidean distances. It extends classical Multidimensional Scaling (MDS) by estimating geodesic distances through a neighborhood graph.

The algorithm: (1) Build a k-nearest neighbor graph, (2) Compute shortest path distances between all pairs (geodesic distances) using Dijkstra's or Floyd-Warshall, (3) Apply classical MDS on the geodesic distance matrix to find the low-dimensional embedding. This allows Isomap to "unfold" curved manifolds like the Swiss Roll.

Key Properties

  • Geodesic: Preserves distances along the manifold, not through it.
  • Global structure: Maintains large-scale geometry.
  • Manifold learning: Can unfold curved surfaces (Swiss Roll).
  • Sensitive to: Choice of k and short-circuit edges.

Implementation

function isomap(points, k, nComponents) {
    var n = points.length;

    // 1. Build k-NN graph
    var graph = buildKNNGraph(points, k);

    // 2. Compute geodesic distances (shortest paths)
    var dist = floydWarshall(graph);

    // 3. Apply classical MDS
    // Center the squared distance matrix
    var D2 = dist.map(function(row) {
        return row.map(function(d) { return d * d; });
    });

    // Double centering: B = -0.5 * H * D2 * H
    var H = centeringMatrix(n);
    var B = matMul(matMul(H, D2), H);
    for (var i = 0; i < n; i++)
        for (var j = 0; j < n; j++)
            B[i][j] *= -0.5;

    // 4. Eigendecompose B, take top nComponents
    var eigen = eigenDecompose(B);
    var embedding = [];
    for (var i2 = 0; i2 < n; i2++) {
        var coord = [];
        for (var c = 0; c < nComponents; c++) {
            coord.push(
                Math.sqrt(eigen.values[c]) *
                eigen.vectors[c][i2]
            );
        }
        embedding.push(coord);
    }

    return embedding;
}
