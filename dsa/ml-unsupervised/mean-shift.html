Mean Shift

Medium Time: O(n²) Space: O(n)

How It Works

Mean Shift is a non-parametric clustering algorithm that does not require specifying the number of clusters. It works by iteratively shifting each data point toward the mean of points within its bandwidth (kernel window), effectively moving points toward density peaks.

Each point is treated as a candidate centroid. At each iteration, the algorithm computes the mean of all points within the bandwidth radius and shifts the candidate toward that mean. Points that converge to the same peak are assigned to the same cluster.

Key Properties

  • No k required: Number of clusters determined automatically.
  • Mode seeking: Finds peaks in the density function.
  • Bandwidth: Single parameter controlling kernel size.
  • Convergence: Guaranteed to converge to local density maxima.

Implementation

function meanShift(points, bandwidth, maxIter) {
    var shifted = points.map(function(p) {
        return { x: p.x, y: p.y };
    });

    for (var iter = 0; iter < maxIter; iter++) {
        var moved = false;
        shifted.forEach(function(s) {
            var sumX = 0, sumY = 0, count = 0;
            points.forEach(function(p) {
                var d = Math.sqrt(
                    Math.pow(s.x - p.x, 2) +
                    Math.pow(s.y - p.y, 2)
                );
                if (d <= bandwidth) {
                    // Gaussian kernel weight
                    var w = Math.exp(-(d * d) /
                        (2 * bandwidth * bandwidth));
                    sumX += w * p.x;
                    sumY += w * p.y;
                    count += w;
                }
            });
            if (count > 0) {
                var newX = sumX / count;
                var newY = sumY / count;
                if (Math.abs(newX-s.x) > 0.01 ||
                    Math.abs(newY-s.y) > 0.01) moved = true;
                s.x = newX;
                s.y = newY;
            }
        });
        if (!moved) break;
    }
    return shifted; // cluster by proximity of converged points
}
