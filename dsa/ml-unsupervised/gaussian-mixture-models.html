Gaussian Mixture Models

Hard Time: O(n·k·d²) Space: O(n·k)

How It Works

Gaussian Mixture Models (GMM) assume data is generated from a mixture of several Gaussian distributions with unknown parameters. Unlike K-Means which makes hard assignments, GMM provides soft (probabilistic) cluster assignments using the Expectation-Maximization (EM) algorithm.

The EM algorithm alternates between two steps: the E-step computes the probability (responsibility) that each Gaussian generated each point, and the M-step updates the Gaussian parameters (mean, covariance, mixing weight) to maximize the likelihood given those responsibilities.

Key Properties

  • Soft assignment: Each point has a probability of belonging to each cluster.
  • Flexible shapes: Elliptical clusters via covariance matrices.
  • Generative model: Can generate new data points.
  • Sensitive to: Initialization and number of components k.

Implementation

function gmmEM(points, k, maxIter) {
    // Initialize means, covariances, weights
    var means = points.slice(0, k).map(function(p) {
        return { x: p.x, y: p.y };
    });
    var weights = new Array(k).fill(1 / k);
    var sigmas = new Array(k).fill(1.0);

    for (var iter = 0; iter < maxIter; iter++) {
        // E-step: compute responsibilities
        var resp = points.map(function(p) {
            var probs = means.map(function(m, j) {
                var dx = p.x - m.x, dy = p.y - m.y;
                var exponent = -(dx*dx + dy*dy) /
                    (2 * sigmas[j] * sigmas[j]);
                return weights[j] *
                    Math.exp(exponent) / sigmas[j];
            });
            var sum = probs.reduce(function(a,b){return a+b;},0);
            return probs.map(function(v){return v/sum;});
        });

        // M-step: update parameters
        for (var j = 0; j < k; j++) {
            var nk = 0, mx = 0, my = 0;
            for (var i = 0; i < points.length; i++) {
                nk += resp[i][j];
                mx += resp[i][j] * points[i].x;
                my += resp[i][j] * points[i].y;
            }
            means[j] = { x: mx/nk, y: my/nk };
            weights[j] = nk / points.length;
            var variance = 0;
            for (var i2 = 0; i2 < points.length; i2++) {
                var dx = points[i2].x - means[j].x;
                var dy = points[i2].y - means[j].y;
                variance += resp[i2][j] * (dx*dx + dy*dy);
            }
            sigmas[j] = Math.sqrt(variance / nk);
        }
    }
    return { means: means, sigmas: sigmas, weights: weights };
}
