Spectral Clustering

Hard Time: O(n³) Space: O(n²)

How It Works

Spectral Clustering uses the eigenvalues of a similarity graph's Laplacian matrix to reduce dimensionality before clustering. This allows it to find clusters with complex, non-convex shapes that K-Means cannot handle.

The algorithm: (1) Build a similarity graph from data, (2) Compute the graph Laplacian, (3) Find the k smallest eigenvectors, (4) Embed points in the eigenspace, (5) Apply K-Means in the reduced space. The key insight is that connected components in the graph correspond to clusters in eigenspace.

Key Properties

  • Complex shapes: Handles non-convex clusters (rings, spirals).
  • Graph-based: Relies on pairwise similarity between points.
  • Eigendecomposition: Computational bottleneck is O(n^3).
  • Sigma parameter: Controls the similarity kernel width.
