OPTICS

Medium Time: O(n log n) Space: O(n)

How It Works

OPTICS (Ordering Points To Identify the Clustering Structure) is an extension of DBSCAN that handles varying density clusters. Instead of producing a flat clustering, it creates an ordering of points augmented with reachability distances that can be used to extract clusters at multiple density levels.

The algorithm maintains a reachability distance for each point, which is the maximum of its core distance and its distance to the point from which it was reached. The resulting reachability plot shows valleys for dense clusters and peaks for cluster boundaries.

Key Properties

  • Varying density: Handles clusters of different densities.
  • Reachability plot: Visual representation of cluster structure.
  • Extends DBSCAN: DBSCAN is a special case of OPTICS.
  • Parameter: minPts and max epsilon (less sensitive than DBSCAN).

Implementation

function optics(points, eps, minPts) {
    var n = points.length;
    var processed = new Array(n).fill(false);
    var reachability = new Array(n).fill(Infinity);
    var ordering = [];

    function dist(a, b) {
        return Math.sqrt(
            Math.pow(a.x-b.x,2) + Math.pow(a.y-b.y,2)
        );
    }

    function getNeighbors(idx) {
        var neighbors = [];
        for (var i = 0; i < n; i++) {
            if (dist(points[idx], points[i]) <= eps) {
                neighbors.push(i);
            }
        }
        return neighbors;
    }

    function coreDist(idx, neighbors) {
        if (neighbors.length < minPts) return Infinity;
        var dists = neighbors.map(function(ni) {
            return dist(points[idx], points[ni]);
        }).sort(function(a,b){return a-b;});
        return dists[minPts - 1];
    }

    for (var i = 0; i < n; i++) {
        if (processed[i]) continue;
        var neighbors = getNeighbors(i);
        processed[i] = true;
        ordering.push(i);
        var cd = coreDist(i, neighbors);

        if (cd !== Infinity) {
            // Update reachability of unprocessed neighbors
            var seeds = [];
            neighbors.forEach(function(ni) {
                if (processed[ni]) return;
                var newReach = Math.max(cd,
                    dist(points[i], points[ni]));
                if (newReach < reachability[ni]) {
                    reachability[ni] = newReach;
                }
                seeds.push(ni);
            });
            // Process seeds by reachability order
            seeds.sort(function(a,b) {
                return reachability[a] - reachability[b];
            });
            // ... expand similarly
        }
    }
    return { ordering: ordering, reachability: reachability };
}
