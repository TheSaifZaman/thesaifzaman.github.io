K-Means Clustering

Easy Time: O(n·k·i) Space: O(n·k)

How It Works

K-Means is the most popular clustering algorithm. It partitions n data points into k clusters by iteratively assigning each point to the nearest centroid, then recomputing centroids as the mean of assigned points. The algorithm converges when assignments no longer change.

The process starts with k randomly placed centroids. In each iteration, every point is assigned to the closest centroid (assignment step), then each centroid moves to the average position of its assigned points (update step). This alternation continues until the centroids stabilize.

Key Properties

  • Convergence: Always converges, but may find local optima.
  • Initialization: K-Means++ improves initial centroid placement.
  • Assumes: Spherical, equally-sized clusters.
  • Sensitive to: Outliers and the choice of k.

Implementation

function kMeans(points, k, maxIter) {
    // Initialize centroids randomly from data points
    var centroids = points.slice(0, k).map(function(p) {
        return { x: p.x, y: p.y };
    });

    for (var iter = 0; iter < maxIter; iter++) {
        // Assignment step
        var clusters = [];
        for (var i = 0; i < k; i++) clusters.push([]);

        points.forEach(function(p) {
            var minDist = Infinity, closest = 0;
            centroids.forEach(function(c, ci) {
                var d = Math.sqrt(
                    Math.pow(p.x - c.x, 2) +
                    Math.pow(p.y - c.y, 2)
                );
                if (d < minDist) { minDist = d; closest = ci; }
            });
            clusters[closest].push(p);
        });

        // Update step
        var moved = false;
        centroids.forEach(function(c, ci) {
            if (clusters[ci].length === 0) return;
            var nx = 0, ny = 0;
            clusters[ci].forEach(function(p) {
                nx += p.x; ny += p.y;
            });
            nx /= clusters[ci].length;
            ny /= clusters[ci].length;
            if (nx !== c.x || ny !== c.y) moved = true;
            c.x = nx; c.y = ny;
        });

        if (!moved) break;
    }
    return { centroids: centroids, clusters: clusters };
}
