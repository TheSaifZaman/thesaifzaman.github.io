t-SNE

Hard Time: O(n²) Space: O(n²)

How It Works

t-SNE (t-distributed Stochastic Neighbor Embedding) is a nonlinear dimensionality reduction technique for visualizing high-dimensional data in 2D or 3D. It converts pairwise distances into probabilities and tries to preserve the local neighborhood structure.

In the high-dimensional space, nearby points get high probability and distant points get low probability (Gaussian distribution). In the low-dimensional embedding, a Student's t-distribution with one degree of freedom is used instead, which has heavier tails. The algorithm minimizes the KL divergence between these two distributions using gradient descent.

Key Properties

  • Nonlinear: Captures complex manifold structure.
  • Local structure: Preserves neighborhoods well.
  • Perplexity: Key parameter controlling neighborhood size.
  • Non-deterministic: Different runs may give different results.

function tSNE(data, dims, perplexity, maxIter) {
    var n = data.length;
    // Compute pairwise distances
    var dist = computeDistances(data);

    // Compute pairwise affinities p_ij (Gaussian)
    var P = computeAffinities(dist, perplexity);

    // Initialize embedding randomly
    var Y = [];
    for (var i = 0; i < n; i++) {
        Y[i] = [Math.random()*0.01, Math.random()*0.01];
    }

    for (var iter = 0; iter < maxIter; iter++) {
        // Compute low-dim affinities q_ij (Student-t)
        var Q = computeStudentT(Y);

        // Compute gradients
        var grad = computeGradient(P, Q, Y);

        // Update embedding
        var lr = 200;
        for (var i2 = 0; i2 < n; i2++) {
            for (var d = 0; d < dims; d++) {
                Y[i2][d] -= lr * grad[i2][d];
            }
        }
    }
    return Y;
}
