DBSCAN

Medium Time: O(n log n) Space: O(n)

How It Works

DBSCAN (Density-Based Spatial Clustering of Applications with Noise) groups points that are closely packed together, marking points in low-density regions as outliers. Unlike K-Means, it does not require specifying the number of clusters and can find arbitrarily shaped clusters.

The algorithm uses two parameters: epsilon (neighborhood radius) and minPts (minimum points to form a dense region). A point is a core point if it has at least minPts within its epsilon neighborhood. Border points are within epsilon of a core point but have fewer than minPts neighbors. All other points are noise.

Key Properties

  • No k required: Automatically determines number of clusters.
  • Handles noise: Identifies outliers as noise points.
  • Arbitrary shapes: Can find non-spherical clusters.
  • Sensitive to: Epsilon and minPts parameters.

Implementation

function dbscan(points, eps, minPts) {
    var labels = new Array(points.length).fill(-1);
    var cluster = 0;

    function regionQuery(idx) {
        var neighbors = [];
        for (var i = 0; i < points.length; i++) {
            var d = Math.sqrt(
                Math.pow(points[i].x - points[idx].x, 2) +
                Math.pow(points[i].y - points[idx].y, 2)
            );
            if (d <= eps) neighbors.push(i);
        }
        return neighbors;
    }

    for (var i = 0; i < points.length; i++) {
        if (labels[i] !== -1) continue;
        var neighbors = regionQuery(i);
        if (neighbors.length < minPts) {
            labels[i] = 0; // noise
            continue;
        }
        cluster++;
        labels[i] = cluster;
        var seed = neighbors.slice();
        for (var j = 0; j < seed.length; j++) {
            var q = seed[j];
            if (labels[q] === 0) labels[q] = cluster;
            if (labels[q] !== -1) continue;
            labels[q] = cluster;
            var qNeighbors = regionQuery(q);
            if (qNeighbors.length >= minPts) {
                seed = seed.concat(qNeighbors);
            }
        }
    }
    return labels;
}
