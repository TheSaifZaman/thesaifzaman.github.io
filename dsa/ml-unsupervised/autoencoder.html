Autoencoder

Medium Time: O(n·d) Space: O(d·h)

How It Works

An autoencoder is a neural network trained to reconstruct its input through a bottleneck layer. The encoder compresses the input into a lower-dimensional latent representation, and the decoder reconstructs the original input from this compressed form. The network learns to capture the most important features in the bottleneck.

The architecture is symmetric: input layer → encoder layers → bottleneck (latent space) → decoder layers → output layer. Training minimizes reconstruction error (MSE between input and output). The bottleneck forces the network to learn a compressed representation, making autoencoders useful for dimensionality reduction, denoising, and anomaly detection.

Key Properties

  • Unsupervised: Learns from unlabeled data (input = target).
  • Nonlinear: Can capture nonlinear relationships (unlike PCA).
  • Bottleneck: Forces compression of essential features.
  • Variants: Denoising, variational (VAE), sparse, contractive.

Implementation

function Autoencoder(inputDim, hiddenDim, latentDim) {
    // Encoder weights
    this.W1 = randomMatrix(inputDim, hiddenDim);
    this.W2 = randomMatrix(hiddenDim, latentDim);
    // Decoder weights
    this.W3 = randomMatrix(latentDim, hiddenDim);
    this.W4 = randomMatrix(hiddenDim, inputDim);
}

Autoencoder.prototype.encode = function(x) {
    var h = relu(matVecMul(this.W1, x));
    return relu(matVecMul(this.W2, h));
};

Autoencoder.prototype.decode = function(z) {
    var h = relu(matVecMul(this.W3, z));
    return sigmoid(matVecMul(this.W4, h));
};

Autoencoder.prototype.forward = function(x) {
    var z = this.encode(x);  // Latent representation
    var xHat = this.decode(z); // Reconstruction
    return { latent: z, output: xHat };
};

Autoencoder.prototype.loss = function(x, xHat) {
    // Mean Squared Error
    var sum = 0;
    for (var i = 0; i < x.length; i++) {
        sum += Math.pow(x[i] - xHat[i], 2);
    }
    return sum / x.length;
};
