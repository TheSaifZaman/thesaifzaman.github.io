UMAP

Hard Time: O(n^1.14) Space: O(n)

How It Works

UMAP (Uniform Manifold Approximation and Projection) is a dimensionality reduction technique based on manifold learning and topological data analysis. It constructs a high-dimensional graph representation of the data, then optimizes a low-dimensional layout to be as structurally similar as possible.

Unlike t-SNE, UMAP preserves more of the global structure and is significantly faster. It uses a fuzzy simplicial set representation of the data manifold: (1) Build a k-nearest-neighbor graph, (2) Convert distances to fuzzy membership strengths, (3) Optimize a low-dimensional embedding using cross-entropy loss.

Key Properties

  • Preserves global structure: Better than t-SNE at maintaining distances between clusters.
  • Fast: Scales much better than t-SNE for large datasets.
  • Topological: Based on Riemannian geometry and algebraic topology.
  • n_neighbors: Key parameter controlling local vs global focus.

function umap(data, nNeighbors, minDist, nComponents) {
    var n = data.length;

    // 1. Build k-NN graph
    var knnGraph = buildKNNGraph(data, nNeighbors);

    // 2. Compute fuzzy simplicial set
    // Convert distances to membership strengths
    var graph = computeFuzzySimplicialSet(
        knnGraph, nNeighbors
    );

    // 3. Initialize low-dim embedding (spectral init)
    var Y = spectralInitialize(graph, nComponents);

    // 4. Optimize layout with SGD
    var nEpochs = 200;
    for (var epoch = 0; epoch < nEpochs; epoch++) {
        var lr = 1.0 * (1.0 - epoch / nEpochs);

        // Attractive forces (connected points)
        graph.edges.forEach(function(edge) {
            var grad = attractiveGrad(
                Y[edge.from], Y[edge.to], minDist
            );
            Y[edge.from] = addVec(Y[edge.from],
                scaleVec(grad, lr));
        });

        // Repulsive forces (negative sampling)
        for (var i = 0; i < n; i++) {
            var j = Math.floor(Math.random() * n);
            if (i === j) continue;
            var repGrad = repulsiveGrad(
                Y[i], Y[j], minDist
            );
            Y[i] = addVec(Y[i], scaleVec(repGrad, lr));
        }
    }

    return Y;
}
