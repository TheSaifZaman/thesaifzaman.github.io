Line Intersection

Medium Two Segments: O(1) Sweep Line: O(n log n) Space: O(n)

How It Works

Determining whether two line segments intersect is a fundamental operation in computational geometry. The standard approach uses the cross product to compute the orientation of point triplets. Given segments P1P2 and P3P4, we compute four orientations: (P1,P2,P3), (P1,P2,P4), (P3,P4,P1), (P3,P4,P2). The segments intersect if the orientations straddle — one pair is clockwise/counterclockwise in opposite directions.

For checking all intersections among n segments, the Bentley-Ottmann sweep line algorithm processes "events" (segment endpoints and intersection points) from left to right. It maintains an ordered set of active segments and only checks adjacent pairs for intersections, reducing the brute-force O(n²) to O((n + k) log n) where k is the number of intersections.

Line intersection tests are essential in computer graphics (ray tracing, polygon clipping), geographic information systems (road network analysis), collision detection (game physics), and VLSI circuit design (wire crossing detection).

Key Properties

  • Orientation Test: Cross product determines CW, CCW, or collinear orientation of 3 points.
  • General Position: Two segments intersect if each segment straddles the line of the other.
  • Collinear Case: Special handling when points are collinear — check overlap of projections.
  • Sweep Line: Efficiently finds all intersections among many segments.

Implementation

function orientation(p, q, r) {
    var val = (q[1] - p[1]) * (r[0] - q[0]) - (q[0] - p[0]) * (r[1] - q[1]);
    if (val === 0) return 0;       // collinear
    return val > 0 ? 1 : 2;        // 1 = CW, 2 = CCW
}

function onSegment(p, q, r) {
    return q[0] <= Math.max(p[0], r[0]) && q[0] >= Math.min(p[0], r[0]) &&
           q[1] <= Math.max(p[1], r[1]) && q[1] >= Math.min(p[1], r[1]);
}

function doIntersect(p1, q1, p2, q2) {
    var d1 = orientation(p1, q1, p2);
    var d2 = orientation(p1, q1, q2);
    var d3 = orientation(p2, q2, p1);
    var d4 = orientation(p2, q2, q1);

    // General case: segments straddle each other
    if (d1 !== d2 && d3 !== d4) return true;

    // Collinear special cases
    if (d1 === 0 && onSegment(p1, p2, q1)) return true;
    if (d2 === 0 && onSegment(p1, q2, q1)) return true;
    if (d3 === 0 && onSegment(p2, p1, q2)) return true;
    if (d4 === 0 && onSegment(p2, q1, q2)) return true;

    return false;
}

// Find intersection point (if segments are not parallel)
function intersectionPoint(p1, q1, p2, q2) {
    var a1 = q1[1] - p1[1], b1 = p1[0] - q1[0];
    var c1 = a1 * p1[0] + b1 * p1[1];
    var a2 = q2[1] - p2[1], b2 = p2[0] - q2[0];
    var c2 = a2 * p2[0] + b2 * p2[1];
    var det = a1 * b2 - a2 * b1;
    if (det === 0) return null; // parallel
    return [(b2*c1 - b1*c2)/det, (a1*c2 - a2*c1)/det];
}
