Convex Hull

Hard Graham Scan: O(n log n) Andrew's: O(n log n) Space: O(n)

How It Works

The Convex Hull of a set of points is the smallest convex polygon that contains all the points. Imagine stretching a rubber band around all the points — the shape it forms is the convex hull. Only the outermost points lie on the hull; all others are inside.

Graham Scan works by: (1) finding the bottom-most point as anchor, (2) sorting all other points by polar angle from the anchor, (3) processing points in order, maintaining a stack of hull vertices. For each new point, we check if adding it makes a left turn (keep) or right turn (pop the previous point). The cross product determines turn direction.

Andrew's Monotone Chain is often preferred: sort points by x-coordinate, then build the upper hull and lower hull separately using the same left-turn check. It is simpler to implement and handles collinear points more cleanly. Both algorithms run in O(n log n) due to sorting.

Key Properties

  • Cross Product: cross(O,A,B) > 0 means left turn, < 0 means right turn, = 0 means collinear.
  • Monotone Chain: Sort by x, build upper and lower hulls separately.
  • Applications: Collision detection, shape analysis, farthest pair (rotating calipers).
  • Output-Sensitive: Algorithms like Jarvis March run in O(nh) where h is hull size.

Implementation

// Andrew's Monotone Chain Algorithm
function convexHull(points) {
    points.sort(function(a, b) { return a[0] - b[0] || a[1] - b[1]; });
    var n = points.length;
    if (n <= 1) return points.slice();

    function cross(O, A, B) {
        return (A[0] - O[0]) * (B[1] - O[1]) - (A[1] - O[1]) * (B[0] - O[0]);
    }

    // Build lower hull
    var lower = [];
    for (var i = 0; i < n; i++) {
        while (lower.length >= 2 && cross(lower[lower.length-2], lower[lower.length-1], points[i]) <= 0)
            lower.pop();
        lower.push(points[i]);
    }

    // Build upper hull
    var upper = [];
    for (var j = n - 1; j >= 0; j--) {
        while (upper.length >= 2 && cross(upper[upper.length-2], upper[upper.length-1], points[j]) <= 0)
            upper.pop();
        upper.push(points[j]);
    }

    // Remove last point of each half (duplicate of first point of other half)
    lower.pop();
    upper.pop();
    return lower.concat(upper);
}
