Point in Polygon

Medium Ray Casting: O(n) Winding Number: O(n) Space: O(1)

How It Works

The Point-in-Polygon problem determines whether a given point lies inside, outside, or on the boundary of a polygon. The most popular solution is the Ray Casting algorithm (also called the even-odd rule): cast a ray from the point in any direction (typically rightward) and count how many polygon edges it crosses. An odd count means inside; even means outside.

The Winding Number algorithm is more robust, especially for complex (self-intersecting) polygons. It counts how many times the polygon winds around the point. A non-zero winding number means the point is inside. This correctly handles polygons with holes and self-intersections.

Point-in-polygon tests are fundamental in GIS (geofencing), computer graphics (hit testing, rendering), games (collision detection with complex shapes), and computational geometry (polygon operations like union, intersection, and containment).

Key Properties

  • Ray Casting: Count edge crossings — odd = inside, even = outside.
  • Edge Cases: Ray through vertex or along edge requires careful handling.
  • Winding Number: More robust for self-intersecting polygons.
  • Preprocessing: For many queries, triangulation or trapezoidal decomposition enables O(log n) per query.

Implementation

// Ray Casting Algorithm
function pointInPolygon(point, polygon) {
    var x = point[0], y = point[1];
    var n = polygon.length;
    var inside = false;

    for (var i = 0, j = n - 1; i < n; j = i++) {
        var xi = polygon[i][0], yi = polygon[i][1];
        var xj = polygon[j][0], yj = polygon[j][1];

        // Check if ray from (x,y) rightward crosses edge (i,j)
        var intersects = ((yi > y) !== (yj > y)) &&
            (x < (xj - xi) * (y - yi) / (yj - yi) + xi);

        if (intersects) inside = !inside;
    }
    return inside;
}

// Winding Number Algorithm
function windingNumber(point, polygon) {
    var wn = 0;
    var n = polygon.length;

    for (var i = 0; i < n; i++) {
        var j = (i + 1) % n;
        if (polygon[i][1] <= point[1]) {
            if (polygon[j][1] > point[1]) {
                if (isLeft(polygon[i], polygon[j], point) > 0) wn++;
            }
        } else {
            if (polygon[j][1] <= point[1]) {
                if (isLeft(polygon[i], polygon[j], point) < 0) wn--;
            }
        }
    }
    return wn; // 0 = outside, nonzero = inside
}

function isLeft(p0, p1, p2) {
    return (p1[0]-p0[0]) * (p2[1]-p0[1]) - (p2[0]-p0[0]) * (p1[1]-p0[1]);
}
