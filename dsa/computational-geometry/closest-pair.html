Closest Pair of Points

Hard Divide & Conquer: O(n log n) Brute Force: O(n²) Space: O(n)

How It Works

The Closest Pair problem asks: given n points in 2D, find the two points with the smallest Euclidean distance. The brute-force approach checks all O(n²) pairs, but a divide-and-conquer algorithm solves it in O(n log n).

The algorithm: (1) Sort points by x-coordinate. (2) Split into left and right halves. (3) Recursively find the closest pair in each half (distance d). (4) Let delta = min(d_left, d_right). (5) Check points within a vertical strip of width 2*delta around the dividing line. The key insight is that within this strip, each point needs to be compared with at most 6-7 nearby points (sorted by y-coordinate), making the merge step O(n).

This gives recurrence T(n) = 2T(n/2) + O(n), which solves to O(n log n). The algorithm is a classic example of divide-and-conquer with a non-trivial merge step. It is used in clustering algorithms, collision detection, and nearest-neighbor approximation.

Key Properties

  • Divide & Conquer: Split points by x-coordinate, solve halves, merge with strip check.
  • Strip Optimization: Only 6-7 comparisons per point in the strip (proven bound).
  • Sorted by Y: Strip points sorted by y-coordinate enable the O(n) merge step.
  • Applications: Clustering, collision detection, wireless network optimization.

Implementation

function dist(p1, p2) {
    return Math.sqrt((p1[0]-p2[0])**2 + (p1[1]-p2[1])**2);
}

function closestPair(points) {
    points.sort(function(a,b) { return a[0]-b[0]; });
    return closestRec(points, 0, points.length - 1);
}

function closestRec(pts, lo, hi) {
    if (hi - lo < 3) {
        // Brute force for small sets
        var best = Infinity;
        for (var i = lo; i <= hi; i++)
            for (var j = i+1; j <= hi; j++)
                best = Math.min(best, dist(pts[i], pts[j]));
        return best;
    }

    var mid = (lo + hi) >> 1;
    var midX = pts[mid][0];
    var dL = closestRec(pts, lo, mid);
    var dR = closestRec(pts, mid+1, hi);
    var delta = Math.min(dL, dR);

    // Points in the strip
    var strip = [];
    for (var k = lo; k <= hi; k++) {
        if (Math.abs(pts[k][0] - midX) < delta)
            strip.push(pts[k]);
    }
    strip.sort(function(a,b) { return a[1]-b[1]; });

    // Check strip pairs (at most 7 comparisons each)
    for (var a = 0; a < strip.length; a++) {
        for (var b = a+1; b < strip.length && strip[b][1]-strip[a][1] < delta; b++) {
            delta = Math.min(delta, dist(strip[a], strip[b]));
        }
    }
    return delta;
}
