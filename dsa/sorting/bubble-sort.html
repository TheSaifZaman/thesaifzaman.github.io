Bubble Sort

Easy Best: O(n) Avg: O(n²) Worst: O(n²) Space: O(1)

How It Works

Bubble Sort is the simplest comparison-based sorting algorithm. It works by repeatedly stepping through the list, comparing each pair of adjacent elements, and swapping them if they are in the wrong order. This process is repeated until no more swaps are needed, meaning the array is sorted.

The algorithm gets its name because smaller elements "bubble" to the top (beginning) of the list with each pass. After the first pass, the largest element is guaranteed to be in its correct position at the end of the array. After the second pass, the second-largest is in place, and so on.

While Bubble Sort is easy to understand and implement, it is inefficient for large datasets due to its O(n²) average and worst-case time complexity. However, it has the advantage of being able to detect an already-sorted list in O(n) time with the optimized version that tracks whether any swaps were made.

Key Properties

  • Stable: Yes — equal elements maintain their relative order.
  • In-place: Yes — only O(1) extra space needed.
  • Adaptive: Yes — O(n) when the input is already sorted (with early termination).
  • Online: No — needs the entire list to begin sorting.

Implementation

function bubbleSort(arr) {
    var n = arr.length;
    for (var i = 0; i < n - 1; i++) {
        var swapped = false;
        for (var j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                var temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }
        if (!swapped) break;
    }
    return arr;
}
