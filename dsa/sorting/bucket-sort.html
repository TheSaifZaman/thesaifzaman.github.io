Bucket Sort

Medium Best: O(n+k) Avg: O(n+k) Worst: O(n²) Space: O(n+k)

How It Works

Bucket Sort distributes elements into a number of "buckets" based on their value range. Each bucket is then sorted individually, either using a different sorting algorithm (like Insertion Sort) or by recursively applying Bucket Sort. Finally, the sorted buckets are concatenated to produce the sorted array.

The algorithm works best when input is uniformly distributed across a range. It divides the range into equal-sized intervals (buckets), places each element into the appropriate bucket, sorts each bucket, and then gathers them back together. With uniform distribution, each bucket contains roughly n/k elements.

Bucket Sort achieves average-case O(n+k) time when the input is uniformly distributed. In the worst case (all elements in one bucket), it degrades to whatever the bucket-level sorting algorithm's complexity is. It is commonly used for sorting floating-point numbers uniformly distributed over [0, 1).

Key Properties

  • Stable: Yes — if the underlying bucket sort is stable.
  • In-place: No — requires O(n+k) extra space for buckets.
  • Distribution-based: Best with uniformly distributed data.
  • Generalizable: Works for any range by adjusting bucket boundaries.

Implementation

function bucketSort(arr) {
    var n = arr.length;
    if (n <= 1) return arr;

    var min = Math.min.apply(null, arr);
    var max = Math.max.apply(null, arr);
    var bucketCount = Math.max(Math.floor(n / 2), 1);
    var range = (max - min) / bucketCount + 1;

    var buckets = [];
    for (var i = 0; i < bucketCount; i++) buckets.push([]);

    // Distribute into buckets
    for (var i = 0; i < n; i++) {
        var idx = Math.floor((arr[i] - min) / range);
        buckets[idx].push(arr[i]);
    }

    // Sort each bucket (insertion sort)
    var result = [];
    for (var i = 0; i < buckets.length; i++) {
        buckets[i].sort(function(a, b) { return a - b; });
        for (var j = 0; j < buckets[i].length; j++) {
            result.push(buckets[i][j]);
        }
    }
    return result;
}
