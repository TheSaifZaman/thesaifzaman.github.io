Tim Sort

Hard Best: O(n) Avg: O(n log n) Worst: O(n log n) Space: O(n)

How It Works

Tim Sort is a hybrid sorting algorithm derived from Merge Sort and Insertion Sort. Invented by Tim Peters in 2002, it is the default sort in Python and Java (for objects). It is designed to perform well on real-world data that often contains already-ordered subsequences ("runs").

The algorithm first divides the array into small chunks called "runs" (typically 32-64 elements). Each run is sorted using Insertion Sort, which is efficient for small arrays. Then, these sorted runs are merged together using a modified Merge Sort, with optimizations like galloping mode to handle cases where one run is consistently smaller than another.

Tim Sort's key advantage is its adaptivity: it detects existing order in the input and exploits it. For already-sorted data, it achieves O(n) time. Its stability guarantee and practical efficiency make it the algorithm of choice for most standard library sort implementations.

Key Properties

  • Stable: Yes — preserves relative order of equal elements.
  • In-place: No — requires O(n) auxiliary space for merging.
  • Adaptive: Yes — O(n) for nearly sorted data, exploits natural runs.
  • Hybrid: Combines Insertion Sort (for small runs) with Merge Sort.

Implementation

var RUN = 4; // Small for visualization; typically 32

function insertionSortRange(arr, left, right) {
    for (var i = left + 1; i <= right; i++) {
        var key = arr[i];
        var j = i - 1;
        while (j >= left && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

function mergeRuns(arr, l, m, r) {
    var left = arr.slice(l, m + 1);
    var right = arr.slice(m + 1, r + 1);
    var i = 0, j = 0, k = l;
    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) arr[k++] = left[i++];
        else arr[k++] = right[j++];
    }
    while (i < left.length) arr[k++] = left[i++];
    while (j < right.length) arr[k++] = right[j++];
}

function timSort(arr) {
    var n = arr.length;
    for (var i = 0; i < n; i += RUN) {
        insertionSortRange(arr, i, Math.min(i + RUN - 1, n - 1));
    }
    for (var size = RUN; size < n; size *= 2) {
        for (var left = 0; left < n; left += 2 * size) {
            var mid = Math.min(left + size - 1, n - 1);
            var right = Math.min(left + 2 * size - 1, n - 1);
            if (mid < right) mergeRuns(arr, left, mid, right);
        }
    }
    return arr;
}
