Difference Array

Easy Time: O(n + q) Space: O(n)

How It Works

A difference array is the inverse of a prefix sum. Given an array, diff[i] = arr[i] - arr[i-1]. The key advantage: a range update add val to arr[l..r] becomes just two operations: diff[l] += val and diff[r+1] -= val. This turns O(n) range updates into O(1).

After all updates, reconstruct the final array by computing the prefix sum of the difference array. This makes the technique ideal when you have many range updates but only need the final result — O(1) per update instead of O(n).

Common applications include: applying multiple range increments (corporate flight bookings, car pooling), computing the effect of overlapping intervals, and any scenario where many “add to range” operations are batched before reading.

O(1) Range Update
Add val to [l, r]: diff[l] += val, diff[r+1] -= val. No need to iterate the range.
Prefix Sum to Reconstruct
After all updates, compute prefix sum of diff array to get the final values.
Batch Updates
Ideal when many range updates are applied before any queries on individual elements.
Inverse of Prefix Sum
Difference array is the inverse operation: prefixSum(differenceArray(arr)) = arr.

Implementation

// Difference Array — range update technique
function DifferenceArray(n) {
    this.diff = new Array(n + 1).fill(0);
}
DifferenceArray.prototype.update = function(l, r, val) {
    this.diff[l] += val;
    if (r + 1 < this.diff.length) this.diff[r + 1] -= val;
};
DifferenceArray.prototype.build = function() {
    var result = [];
    var sum = 0;
    for (var i = 0; i < this.diff.length - 1; i++) {
        sum += this.diff[i];
        result.push(sum);
    }
    return result;
};

// Corporate Flight Bookings (LeetCode #1109)
function corpFlightBookings(bookings, n) {
    var diff = new Array(n + 1).fill(0);
    for (var i = 0; i < bookings.length; i++) {
        diff[bookings[i][0] - 1] += bookings[i][2];
        if (bookings[i][1] < n) diff[bookings[i][1]] -= bookings[i][2];
    }
    var result = [];
    var sum = 0;
    for (var i = 0; i < n; i++) {
        sum += diff[i];
        result.push(sum);
    }
    return result;
}
