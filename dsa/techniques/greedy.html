Greedy

Medium Time: O(n log n) Space: O(1)

How It Works

A greedy algorithm makes the locally optimal choice at each step, hoping this leads to a globally optimal solution. Unlike dynamic programming which considers all possibilities, greedy commits to the best immediate option and never backtracks.

Greedy works when a problem has the greedy choice property (a locally optimal choice leads to a global optimum) and optimal substructure. Not all problems have these properties — proving correctness via exchange argument or contradiction is essential.

Classic greedy strategies include sorting by deadline/profit, always picking the minimum cost edge, or jumping as far as possible. The elegance of greedy lies in its simplicity: no recursion or table needed, just a single pass with the right criterion.

Greedy Choice Property
Making the locally best choice at each step leads to a globally optimal solution.
Sort First
Many greedy problems require sorting the input by some criterion before making greedy choices.
Prove Correctness
Greedy solutions need proof (exchange argument, contradiction) — intuition alone is not enough.
No Backtracking
Unlike DP or backtracking, greedy never revisits a decision once made.

// Jump Game (LeetCode #55)
function canJump(nums) {
    var farthest = 0;
    for (var i = 0; i < nums.length; i++) {
        if (i > farthest) return false;
        farthest = Math.max(farthest, i + nums[i]);
    }
    return true;
}

// Activity Selection — select maximum non-overlapping activities
function activitySelection(activities) {
    activities.sort(function(a, b) { return a[1] - b[1]; });
    var selected = [activities[0]];
    var lastEnd = activities[0][1];
    for (var i = 1; i < activities.length; i++) {
        if (activities[i][0] >= lastEnd) {
            selected.push(activities[i]);
            lastEnd = activities[i][1];
        }
    }
    return selected;
}

// Task Scheduler (LeetCode #621)
function leastInterval(tasks, n) {
    var freq = new Array(26).fill(0);
    for (var i = 0; i < tasks.length; i++) {
        freq[tasks[i].charCodeAt(0) - 65]++;
    }
    freq.sort(function(a, b) { return b - a; });
    var maxFreq = freq[0];
    var idleSlots = (maxFreq - 1) * n;
    for (var i = 1; i < freq.length && freq[i] > 0; i++) {
        idleSlots -= Math.min(freq[i], maxFreq - 1);
    }
    return Math.max(tasks.length, tasks.length + idleSlots);
}
