Divide and Conquer

Medium Time: O(n log n) Space: O(log n)

How It Works

Divide and Conquer breaks a problem into smaller sub-problems of the same type, solves each sub-problem recursively, and then combines the results. This strategy is the backbone of efficient algorithms like Merge Sort, Quick Sort, and Binary Search.

The three steps are: Divide the problem into sub-problems, Conquer by solving sub-problems recursively (base case for small inputs), and Combine the sub-problem solutions into the final answer. The recursion tree shows how the problem splits at each level.

The Master Theorem provides a framework for analyzing the time complexity of divide-and-conquer recurrences: T(n) = aT(n/b) + O(n^d). By comparing log_b(a) with d, we can determine whether the algorithm is dominated by the divide step, the combine step, or both equally.

Divide
Split the problem into two or more smaller instances of the same problem.
Conquer
Recursively solve each sub-problem. When small enough, solve directly (base case).
Combine
Merge sub-problem solutions. This step often determines the overall complexity.
Master Theorem
T(n) = aT(n/b) + O(n^d): compare log_b(a) with d to determine time complexity.

// Merge Sort — classic divide and conquer
function mergeSort(arr) {
    if (arr.length <= 1) return arr;
    var mid = Math.floor(arr.length / 2);
    var left = mergeSort(arr.slice(0, mid));
    var right = mergeSort(arr.slice(mid));
    return merge(left, right);
}

function merge(left, right) {
    var result = [], i = 0, j = 0;
    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) result.push(left[i++]);
        else result.push(right[j++]);
    }
    return result.concat(left.slice(i)).concat(right.slice(j));
}

// Maximum Subarray (Divide and Conquer approach)
function maxSubArray(nums, lo, hi) {
    if (lo === hi) return nums[lo];
    var mid = Math.floor((lo + hi) / 2);
    var leftMax = maxSubArray(nums, lo, mid);
    var rightMax = maxSubArray(nums, mid + 1, hi);
    var crossMax = maxCrossing(nums, lo, mid, hi);
    return Math.max(leftMax, rightMax, crossMax);
}

function maxCrossing(nums, lo, mid, hi) {
    var leftSum = -Infinity, sum = 0;
    for (var i = mid; i >= lo; i--) {
        sum += nums[i];
        leftSum = Math.max(leftSum, sum);
    }
    var rightSum = -Infinity; sum = 0;
    for (var i = mid + 1; i <= hi; i++) {
        sum += nums[i];
        rightSum = Math.max(rightSum, sum);
    }
    return leftSum + rightSum;
}
