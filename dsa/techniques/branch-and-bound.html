Branch and Bound

Hard Time: Exponential (pruned) Space: O(n)

How It Works

Branch and Bound is an optimization technique for combinatorial problems. Like backtracking, it explores a search tree, but it also computes a bound (optimistic estimate) for each partial solution. If the bound is worse than the best known solution, the entire subtree is pruned.

The “branch” part generates child nodes (partial solutions). The “bound” part computes an upper bound (for maximization) or lower bound (for minimization). If the bound cannot beat the current best, skip the subtree. This is more aggressive than backtracking’s feasibility pruning.

Common applications: TSP (Traveling Salesman Problem), 0/1 Knapsack, job scheduling, integer linear programming. The bounding function quality determines effectiveness — tighter bounds prune more aggressively.

Branch
Generate child nodes by making one decision (include/exclude an item, visit a city, etc.).
Bound
Compute an optimistic estimate for each node. If bound <= current best, prune the subtree.
Best-First Search
Use a priority queue to explore the most promising nodes first, finding good solutions early.
Pruning Power
Tighter bounds = more pruning = faster. The bounding function is the key design choice.

Implementation

// Branch and Bound — 0/1 Knapsack
function knapsackBnB(weights, values, capacity) {
    var n = weights.length;
    var bestValue = 0;

    // Upper bound: greedy fractional knapsack from current state
    function bound(level, weight, value) {
        if (weight > capacity) return 0;
        var b = value;
        var w = weight;
        for (var i = level; i < n && w + weights[i] <= capacity; i++) {
            w += weights[i];
            b += values[i];
        }
        if (i < n) {
            b += (capacity - w) * (values[i] / weights[i]);
        }
        return b;
    }

    // Sort by value/weight ratio descending
    var items = [];
    for (var i = 0; i < n; i++) items.push([weights[i], values[i], i]);
    items.sort(function(a, b) { return (b[1]/b[0]) - (a[1]/a[0]); });
    weights = items.map(function(x) { return x[0]; });
    values = items.map(function(x) { return x[1]; });

    function solve(level, weight, value) {
        if (level === n) {
            bestValue = Math.max(bestValue, value);
            return;
        }
        // Include item
        if (weight + weights[level] <= capacity) {
            solve(level + 1, weight + weights[level], value + values[level]);
        }
        // Exclude item (check bound first)
        if (bound(level + 1, weight, value) > bestValue) {
            solve(level + 1, weight, value);
        }
    }

    solve(0, 0, 0);
    return bestValue;
}
