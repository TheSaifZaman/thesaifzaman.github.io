Kadane's Algorithm

Easy Time: O(n) Space: O(1)

How It Works

Kadane's Algorithm finds the maximum sum contiguous subarray in O(n) time using a simple observation: at each position, the maximum subarray ending here is either the current element alone, or the current element plus the maximum subarray ending at the previous position.

We maintain currentMax (best sum ending at the current position) and globalMax (best sum found so far). At each element: currentMax = max(arr[i], currentMax + arr[i]). If currentMax drops below arr[i], we start a new subarray from arr[i]. Then update globalMax.

This is fundamentally a DP algorithm with space-optimized to O(1). The key insight is: if the running sum becomes negative, it is always better to start fresh from the next element rather than carry a negative prefix.

Current Max
max(arr[i], currentMax + arr[i]). Decides whether to extend the subarray or start fresh.
Global Max
Tracks the best result across all positions. Updated after each currentMax computation.
Negative Prefix
A negative running sum is discarded — starting fresh from the next element is always better.
DP with O(1) Space
This is dp[i] = max(arr[i], dp[i-1] + arr[i]) with only the previous value needed.

// Maximum Subarray (LeetCode #53)
function maxSubArray(nums) {
    var currentMax = nums[0], globalMax = nums[0];
    for (var i = 1; i < nums.length; i++) {
        currentMax = Math.max(nums[i], currentMax + nums[i]);
        globalMax = Math.max(globalMax, currentMax);
    }
    return globalMax;
}

// Maximum Subarray with indices
function maxSubArrayWithIndices(nums) {
    var currentMax = nums[0], globalMax = nums[0];
    var start = 0, end = 0, tempStart = 0;
    for (var i = 1; i < nums.length; i++) {
        if (nums[i] > currentMax + nums[i]) {
            currentMax = nums[i];
            tempStart = i;
        } else {
            currentMax += nums[i];
        }
        if (currentMax > globalMax) {
            globalMax = currentMax;
            start = tempStart;
            end = i;
        }
    }
    return { sum: globalMax, start: start, end: end };
}

// Maximum Product Subarray (LeetCode #152)
function maxProduct(nums) {
    var maxSoFar = nums[0], minSoFar = nums[0], result = nums[0];
    for (var i = 1; i < nums.length; i++) {
        var candidates = [nums[i], maxSoFar * nums[i], minSoFar * nums[i]];
        maxSoFar = Math.max.apply(null, candidates);
        minSoFar = Math.min.apply(null, candidates);
        result = Math.max(result, maxSoFar);
    }
    return result;
}
