Prefix Sum

Easy Time: O(n) build, O(1) query Space: O(n)

How It Works

A prefix sum array stores cumulative sums: prefix[i] = arr[0] + arr[1] + ... + arr[i]. Once built in O(n), any range sum query sum(l, r) can be answered in O(1) as prefix[r] - prefix[l-1].

This is one of the most versatile techniques in competitive programming. Beyond range sums, prefix sums enable: finding subarrays with a given sum (using hash maps), computing running averages, and solving problems where the “contribution” of each element to many ranges must be computed efficiently.

The technique extends to 2D (prefix sum of a matrix for O(1) rectangle sum queries) and can be combined with hash maps for problems like “subarray sum equals k” where we look for prefix[j] - prefix[i] = k.

Build in O(n)
prefix[0] = arr[0]; prefix[i] = prefix[i-1] + arr[i]. Single pass construction.
Query in O(1)
sum(l, r) = prefix[r] - prefix[l-1]. No need to iterate over the range.
Hash Map Combo
Store prefix sums in a hash map to find subarrays with a target sum in O(n).
2D Extension
prefix[i][j] = sum of all elements in rectangle (0,0) to (i,j). Enables O(1) rectangle queries.

// Range Sum Query — Immutable (LeetCode #303)
function NumArray(nums) {
    this.prefix = [0];
    for (var i = 0; i < nums.length; i++) {
        this.prefix.push(this.prefix[i] + nums[i]);
    }
}
NumArray.prototype.sumRange = function(left, right) {
    return this.prefix[right + 1] - this.prefix[left];
};

// Subarray Sum Equals K (LeetCode #560)
function subarraySum(nums, k) {
    var count = 0, sum = 0;
    var map = {0: 1};
    for (var i = 0; i < nums.length; i++) {
        sum += nums[i];
        if (map[sum - k] !== undefined) count += map[sum - k];
        map[sum] = (map[sum] || 0) + 1;
    }
    return count;
}

// Product of Array Except Self (LeetCode #238)
function productExceptSelf(nums) {
    var n = nums.length;
    var result = new Array(n).fill(1);
    var prefix = 1;
    for (var i = 0; i < n; i++) {
        result[i] = prefix;
        prefix *= nums[i];
    }
    var suffix = 1;
    for (var i = n - 1; i >= 0; i--) {
        result[i] *= suffix;
        suffix *= nums[i];
    }
    return result;
}
