Cycle Detection (Arrays)

Medium Time: O(n) Space: O(1)

How It Works

Floyd's cycle detection (tortoise and hare) uses two pointers moving at different speeds to detect cycles. The slow pointer moves one step at a time; the fast pointer moves two steps. If there is a cycle, they will meet inside it. If not, the fast pointer reaches the end.

Once a meeting point is found, to locate the cycle start: reset one pointer to the beginning and keep the other at the meeting point. Move both one step at a time — they meet at the cycle entrance. This mathematical property follows from the relationship between the distances traveled.

In array problems, the “next” function is f(x) = arr[x]. If array values are in [1, n] for an array of size n+1, the pigeonhole principle guarantees a duplicate (cycle). This is how LeetCode #287 “Find the Duplicate Number” is solved in O(1) space.

Two Speeds
Slow moves 1 step, fast moves 2 steps. If they meet, a cycle exists.
Finding Cycle Start
After meeting: reset one pointer to start, advance both at speed 1. They meet at the cycle entrance.
Array as Graph
Treat arr[i] as a “next pointer” from index i. Duplicates create cycles in this implicit graph.
O(1) Space
No hash set or marking needed. Only two pointer variables.

// Find the Duplicate Number (LeetCode #287)
function findDuplicate(nums) {
    var slow = nums[0], fast = nums[0];
    // Phase 1: Detect cycle
    do {
        slow = nums[slow];
        fast = nums[nums[fast]];
    } while (slow !== fast);

    // Phase 2: Find cycle entrance
    slow = nums[0];
    while (slow !== fast) {
        slow = nums[slow];
        fast = nums[fast];
    }
    return slow;
}

// Linked List Cycle Detection
function hasCycle(head) {
    var slow = head, fast = head;
    while (fast && fast.next) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow === fast) return true;
    }
    return false;
}

// Find Cycle Start in Linked List
function detectCycle(head) {
    var slow = head, fast = head;
    while (fast && fast.next) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow === fast) {
            slow = head;
            while (slow !== fast) {
                slow = slow.next;
                fast = fast.next;
            }
            return slow;
        }
    }
    return null;
}
