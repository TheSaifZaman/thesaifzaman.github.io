Fisher-Yates Shuffle

Easy Time: O(n) Space: O(1)

How It Works

The Fisher-Yates (Knuth) shuffle generates a uniformly random permutation of an array in O(n) time and O(1) extra space. It iterates from the end of the array to the beginning, swapping each element with a randomly chosen element from the remaining unshuffled portion.

For each position i from n-1 down to 1, pick a random index j in [0, i] and swap arr[i] with arr[j]. After the swap, arr[i] is in its final position. This guarantees each of the n! permutations is equally likely.

A common mistake is the “naive shuffle” where j is picked from [0, n-1] for every i — this does NOT produce uniform distributions. The correct algorithm restricts the random range to shrink with each step.

Uniform Distribution
Every permutation has exactly 1/n! probability of being produced.
In-Place
Modifies the array in place with O(1) extra space.
Shrinking Range
Random index j is chosen from [0, i], not [0, n-1]. Critical for uniformity.
Linear Time
Exactly n-1 swaps for an array of size n.

// Fisher-Yates Shuffle (Knuth version)
function shuffle(arr) {
    for (var i = arr.length - 1; i > 0; i--) {
        var j = Math.floor(Math.random() * (i + 1));
        var temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
    return arr;
}

// Shuffle an Array (LeetCode #384)
function Solution(nums) {
    this.original = nums.slice();
    this.array = nums.slice();
}
Solution.prototype.reset = function() {
    this.array = this.original.slice();
    return this.array;
};
Solution.prototype.shuffle = function() {
    for (var i = this.array.length - 1; i > 0; i--) {
        var j = Math.floor(Math.random() * (i + 1));
        var temp = this.array[i];
        this.array[i] = this.array[j];
        this.array[j] = temp;
    }
    return this.array;
};
