Boyer-Moore Voting

Easy Time: O(n) Space: O(1)

How It Works

The Boyer-Moore Voting Algorithm finds the majority element (appearing more than n/2 times) in O(n) time and O(1) space. It works by maintaining a candidate and a count. As you scan the array, equal elements increment the count, different elements decrement it. When count hits zero, switch the candidate.

The intuition is that the majority element “survives” all cancellations. If an element appears more than n/2 times, it will outlast all other elements combined. After the first pass identifies the candidate, a second pass (optional if majority is guaranteed) verifies it.

This extends to finding elements appearing more than n/3 times (at most 2 such elements) by maintaining two candidates. The generalized version finds up to k-1 candidates for elements appearing more than n/k times.

Candidate + Count
Track one candidate and its running count. Increment for matches, decrement for mismatches.
Cancellation Intuition
Different elements cancel each other. The majority element has enough copies to survive all cancellations.
Verification Pass
If majority is not guaranteed, do a second pass to confirm the candidate appears > n/2 times.
O(1) Space
Only a candidate variable and a counter needed — no hash map or sorting required.

// Majority Element (LeetCode #169)
function majorityElement(nums) {
    var candidate = null, count = 0;
    for (var i = 0; i < nums.length; i++) {
        if (count === 0) candidate = nums[i];
        count += (nums[i] === candidate) ? 1 : -1;
    }
    return candidate;
}

// Majority Element II — elements > n/3 (LeetCode #229)
function majorityElementII(nums) {
    var c1 = null, c2 = null, cnt1 = 0, cnt2 = 0;
    for (var i = 0; i < nums.length; i++) {
        if (nums[i] === c1) cnt1++;
        else if (nums[i] === c2) cnt2++;
        else if (cnt1 === 0) { c1 = nums[i]; cnt1 = 1; }
        else if (cnt2 === 0) { c2 = nums[i]; cnt2 = 1; }
        else { cnt1--; cnt2--; }
    }
    // Verify
    cnt1 = 0; cnt2 = 0;
    for (var i = 0; i < nums.length; i++) {
        if (nums[i] === c1) cnt1++;
        else if (nums[i] === c2) cnt2++;
    }
    var result = [];
    if (cnt1 > nums.length / 3) result.push(c1);
    if (cnt2 > nums.length / 3) result.push(c2);
    return result;
}
