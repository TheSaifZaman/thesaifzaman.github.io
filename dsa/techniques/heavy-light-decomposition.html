Heavy-Light Decomposition

Hard Time: O(n) build, O(log² n) query Space: O(n)

How It Works

Heavy-Light Decomposition (HLD) decomposes a tree into chains of heavy edges, allowing path queries and updates to be answered in O(log² n) time using a segment tree. A heavy child of a node is the child with the largest subtree; all other children are light.

Following heavy edges from any node creates a “heavy chain.” The key insight: any path from a node to the root crosses at most O(log n) light edges, meaning it traverses at most O(log n) chains. By flattening chains into contiguous array segments, each chain query becomes a segment tree query.

HLD enables: path sum/max queries, path updates, and subtree queries on trees, all in O(log² n) per operation. Combined with a lazy segment tree, it handles range updates on paths efficiently.

Heavy vs Light Edges
Heavy edge connects to the child with the largest subtree. Light edges go to smaller subtrees.
O(log n) Chain Switches
Any root-to-leaf path crosses at most O(log n) light edges (chain boundaries).
Flatten to Array
DFS order maps each chain to contiguous positions, enabling segment tree queries.
Segment Tree on Chains
Path queries decompose into O(log n) chain queries, each answered in O(log n) by segment tree.

Implementation

// Heavy-Light Decomposition (outline)
var parent, depth, heavy, head, pos, curPos;

function dfs(v, adj, par) {
    parent[v] = par;
    var maxSize = 0, heavyChild = -1, size = 1;
    for (var i = 0; i < adj[v].length; i++) {
        var u = adj[v][i];
        if (u === par) continue;
        depth[u] = depth[v] + 1;
        var childSize = dfs(u, adj, v);
        size += childSize;
        if (childSize > maxSize) { maxSize = childSize; heavyChild = u; }
    }
    heavy[v] = heavyChild;
    return size;
}

function decompose(v, h, adj) {
    head[v] = h;
    pos[v] = curPos++;
    if (heavy[v] !== -1) decompose(heavy[v], h, adj); // continue chain
    for (var i = 0; i < adj[v].length; i++) {
        var u = adj[v][i];
        if (u !== parent[v] && u !== heavy[v]) {
            decompose(u, u, adj); // start new chain
        }
    }
}

function pathQuery(u, v, segTree) {
    var result = 0; // or -Infinity for max
    while (head[u] !== head[v]) {
        if (depth[head[u]] < depth[head[v]]) { var t = u; u = v; v = t; }
        result += segTree.query(pos[head[u]], pos[u]);
        u = parent[head[u]];
    }
    if (depth[u] > depth[v]) { var t = u; u = v; v = t; }
    result += segTree.query(pos[u], pos[v]);
    return result;
}
