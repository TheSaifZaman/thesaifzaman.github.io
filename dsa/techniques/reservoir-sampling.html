Reservoir Sampling

Medium Time: O(n) Space: O(k)

How It Works

Reservoir Sampling is a family of randomized algorithms for choosing k items uniformly at random from a stream of unknown or very large size n, using only O(k) memory. You do not need to know n in advance.

The basic algorithm (k=1): keep the first item. For the i-th item (1-indexed), replace the current choice with probability 1/i. After processing all n items, each item has exactly 1/n probability of being selected. For k items, fill a reservoir of size k, then for each subsequent item i, replace a random reservoir element with probability k/i.

This is essential for streaming scenarios: sampling from a huge file, random selection from a database without knowing the total count, or any situation where you cannot store all elements.

Streaming Input
Process elements one at a time without storing them all. Only O(k) space needed.
Uniform Probability
Each element has exactly k/n probability of being in the final sample.
Replace Strategy
For item i > k, include it with probability k/i, replacing a random reservoir element.
No Advance Knowledge
You do not need to know the total number of elements in advance.

// Reservoir Sampling — select 1 random element from stream
function reservoirSample1(stream) {
    var result = stream[0];
    for (var i = 1; i < stream.length; i++) {
        var j = Math.floor(Math.random() * (i + 1));
        if (j === 0) result = stream[i];
    }
    return result;
}

// Reservoir Sampling — select k random elements
function reservoirSampleK(stream, k) {
    var reservoir = stream.slice(0, k);
    for (var i = k; i < stream.length; i++) {
        var j = Math.floor(Math.random() * (i + 1));
        if (j < k) reservoir[j] = stream[i];
    }
    return reservoir;
}

// Linked List Random Node (LeetCode #382)
function RandomNode(head) { this.head = head; }
RandomNode.prototype.getRandom = function() {
    var result = this.head.val, node = this.head.next, i = 2;
    while (node) {
        if (Math.floor(Math.random() * i) === 0) result = node.val;
        node = node.next; i++;
    }
    return result;
};
