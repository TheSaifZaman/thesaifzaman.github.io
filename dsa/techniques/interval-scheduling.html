Interval Scheduling

Medium Time: O(n log n) Space: O(n)

How It Works

Interval scheduling problems deal with selecting, merging, or partitioning intervals. The most common variant is selecting the maximum number of non-overlapping intervals, solved greedily by sorting by end time and always picking the earliest-ending interval that does not overlap with the last selected one.

For merging intervals, sort by start time and merge overlapping ones. For minimum meeting rooms, use a min-heap to track the earliest ending meeting, or use the event-point technique (count +1 at each start, -1 at each end, find the maximum overlap).

The key insight across all interval problems is sorting: either by start time (for merging) or by end time (for selection). Once sorted, a single pass through the intervals with greedy logic yields the optimal solution.

Sort by End Time
For maximum non-overlapping selection, always pick the interval that ends earliest.
Sort by Start Time
For merging intervals, sort by start then merge consecutive overlapping ones.
Sweep Line
+1 at start, -1 at end. The running maximum gives the peak overlap count.
Greedy Proof
The earliest-deadline-first strategy provably maximizes the number of non-overlapping intervals.

// Merge Intervals (LeetCode #56)
function merge(intervals) {
    intervals.sort(function(a, b) { return a[0] - b[0]; });
    var merged = [intervals[0]];
    for (var i = 1; i < intervals.length; i++) {
        var last = merged[merged.length - 1];
        if (intervals[i][0] <= last[1]) {
            last[1] = Math.max(last[1], intervals[i][1]);
        } else {
            merged.push(intervals[i]);
        }
    }
    return merged;
}

// Non-overlapping Intervals (LeetCode #435)
function eraseOverlapIntervals(intervals) {
    intervals.sort(function(a, b) { return a[1] - b[1]; });
    var count = 0, lastEnd = -Infinity;
    for (var i = 0; i < intervals.length; i++) {
        if (intervals[i][0] >= lastEnd) {
            lastEnd = intervals[i][1];
        } else {
            count++; // remove this interval
        }
    }
    return count;
}

// Minimum Number of Arrows to Burst Balloons (LeetCode #452)
function findMinArrowShots(points) {
    points.sort(function(a, b) { return a[1] - b[1]; });
    var arrows = 1, lastEnd = points[0][1];
    for (var i = 1; i < points.length; i++) {
        if (points[i][0] > lastEnd) {
            arrows++;
            lastEnd = points[i][1];
        }
    }
    return arrows;
}
