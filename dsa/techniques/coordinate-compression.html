Coordinate Compression

Medium Time: O(n log n) Space: O(n)

How It Works

Coordinate compression maps a set of values to a smaller range while preserving their relative order. If you have values like [100, 5000, 200, 10000], they become [0, 2, 1, 3]. This enables using arrays instead of hash maps, and makes segment trees and BITs feasible when the value range is huge.

The algorithm: (1) collect all unique values, (2) sort them, (3) assign each a rank (its sorted index). Now any O(V) algorithm (where V is the value range) becomes O(n) where n is the number of distinct values. This is essential when values can be up to 10^9 but there are only 10^5 of them.

Common applications: counting inversions with BIT, rectangle union area, sweep line with coordinates, and any problem where you need an array indexed by value but the values are too large for direct indexing.

Rank Mapping
Sort unique values, assign each its index. Maps large range to [0, n-1].
Preserves Order
Relative ordering of values is maintained. Only absolute differences change.
Enables Array Indexing
Replace hash maps with arrays. Enables BIT/segment tree on value-indexed data.
O(n log n) Overhead
Sorting dominates. The mapping itself is O(n) with a hash map or binary search.

Implementation

// Coordinate Compression
function compress(arr) {
    var sorted = Array.from(new Set(arr)).sort(function(a, b) { return a - b; });
    var rank = {};
    for (var i = 0; i < sorted.length; i++) {
        rank[sorted[i]] = i;
    }
    return arr.map(function(v) { return rank[v]; });
}

// Count of Smaller Numbers After Self (uses compression + BIT)
function countSmaller(nums) {
    var sorted = Array.from(new Set(nums)).sort(function(a, b) { return a - b; });
    var rank = {};
    for (var i = 0; i < sorted.length; i++) rank[sorted[i]] = i + 1;

    var n = sorted.length;
    var bit = new Array(n + 1).fill(0);
    function update(i) { for (; i <= n; i += i & (-i)) bit[i]++; }
    function query(i) { var s = 0; for (; i > 0; i -= i & (-i)) s += bit[i]; return s; }

    var result = new Array(nums.length);
    for (var i = nums.length - 1; i >= 0; i--) {
        result[i] = query(rank[nums[i]] - 1);
        update(rank[nums[i]]);
    }
    return result;
}
