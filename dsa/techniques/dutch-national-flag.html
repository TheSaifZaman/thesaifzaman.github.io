Dutch National Flag

Easy Time: O(n) Space: O(1)

How It Works

The Dutch National Flag algorithm, designed by Dijkstra, partitions an array of three distinct values into three contiguous groups in a single pass. It uses three pointers: low (boundary for 0s), mid (current element), and high (boundary for 2s).

The invariant maintained is: elements before low are all 0 (red), elements between low and mid are all 1 (white), elements after high are all 2 (blue), and elements between mid and high are unprocessed. The algorithm processes mid and moves pointers accordingly.

When arr[mid]=0, swap with arr[low] and advance both; when arr[mid]=1, just advance mid; when arr[mid]=2, swap with arr[high] and decrement high (do not advance mid since the swapped value is unprocessed).

Three Pointers
low tracks the boundary of 0s, mid scans the array, high tracks the boundary of 2s.
Single Pass
Process each element once. mid advances right, partitioning into three groups in O(n).
In-Place
Only pointer variables needed — O(1) extra space.
Three-Way Partition
Generalizes to any three-way partition problem around a pivot value.

Implementation

// Sort Colors (LeetCode #75) — Dutch National Flag
function sortColors(nums) {
    var low = 0, mid = 0, high = nums.length - 1;
    while (mid <= high) {
        if (nums[mid] === 0) {
            // Swap with low, advance both
            var temp = nums[low];
            nums[low] = nums[mid];
            nums[mid] = temp;
            low++; mid++;
        } else if (nums[mid] === 1) {
            mid++;  // already in correct region
        } else {
            // Swap with high, decrement high
            var temp = nums[mid];
            nums[mid] = nums[high];
            nums[high] = temp;
            high--;
            // Don't advance mid — swapped value is unprocessed
        }
    }
}

// Three-way partition around a pivot
function threeWayPartition(arr, pivot) {
    var low = 0, mid = 0, high = arr.length - 1;
    while (mid <= high) {
        if (arr[mid] < pivot) {
            var t = arr[low]; arr[low] = arr[mid]; arr[mid] = t;
            low++; mid++;
        } else if (arr[mid] === pivot) {
            mid++;
        } else {
            var t = arr[mid]; arr[mid] = arr[high]; arr[high] = t;
            high--;
        }
    }
}
