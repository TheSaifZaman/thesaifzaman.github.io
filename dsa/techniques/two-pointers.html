Two Pointers

Easy Time: O(n) Space: O(1)

How It Works

The two-pointer technique uses two indices that move through the data structure (usually a sorted array) from different directions or at different speeds. By leveraging the sorted order or structural properties, it eliminates the need for nested loops, reducing O(n²) brute force to O(n).

The most common variant places one pointer at the start and one at the end. If the current pair's sum is too small, advance the left pointer; if too large, retreat the right pointer. This works because the array is sorted — moving left increases the sum, moving right decreases it.

Other variants include the “slow and fast” pattern (for cycle detection or partitioning) and the “same direction” pattern (for merging or removing duplicates). The key insight is always: each pointer moves in only one direction, guaranteeing linear time.

Sorted Input
Most two-pointer problems require sorted input. If not sorted, sort first in O(n log n).
Converging Pointers
Left starts at 0, right at n-1. They move toward each other until they meet.
Same-Direction Pointers
Both pointers start at 0 and move right — used for removing duplicates or partitioning.
No Extra Space
The technique works in-place with O(1) extra space, making it highly efficient.

// Two Sum II — Input Array Is Sorted (LeetCode #167)
function twoSum(numbers, target) {
    var left = 0, right = numbers.length - 1;
    while (left < right) {
        var sum = numbers[left] + numbers[right];
        if (sum === target) return [left + 1, right + 1];
        if (sum < target) left++;
        else right--;
    }
    return [-1, -1];
}

// Container With Most Water (LeetCode #11)
function maxArea(height) {
    var left = 0, right = height.length - 1;
    var best = 0;
    while (left < right) {
        var w = right - left;
        var h = Math.min(height[left], height[right]);
        best = Math.max(best, w * h);
        if (height[left] < height[right]) left++;
        else right--;
    }
    return best;
}

// Remove Duplicates from Sorted Array
function removeDuplicates(nums) {
    if (nums.length === 0) return 0;
    var slow = 0;
    for (var fast = 1; fast < nums.length; fast++) {
        if (nums[fast] !== nums[slow]) {
            slow++;
            nums[slow] = nums[fast];
        }
    }
    return slow + 1;
}
