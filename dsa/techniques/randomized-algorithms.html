Randomized Algorithms

Medium Time: Varies (expected) Space: O(1)–O(n)

How It Works

Randomized algorithms use random choices during execution to achieve good expected performance. They come in two flavors: Las Vegas (always correct, random running time — e.g., randomized QuickSort) and Monte Carlo (fast, but may be incorrect with small probability — e.g., randomized primality testing).

Randomized QuickSort picks a random pivot, making worst-case O(n²) input impossible to construct. The expected time is O(n log n) regardless of input. Similarly, Randomized Quick Select finds the k-th smallest element in O(n) expected time.

Randomization is also used in hashing (random hash functions reduce collisions), skip lists (randomized balanced search structure), and randomized geometric algorithms. The key advantage: simple implementation with provable expected-case guarantees, avoiding adversarial worst cases.

Las Vegas
Always correct, but running time is random. Expected time is usually O(n log n) or O(n).
Monte Carlo
Fast (deterministic time), but answer may be wrong with probability < epsilon.
Avoid Adversarial Input
Random choices prevent adversaries from crafting worst-case inputs.
Expected vs Worst Case
Expected O(n log n) is often better in practice than deterministic O(n log n) with large constants.

// Randomized QuickSort
function quickSort(arr, lo, hi) {
    if (lo >= hi) return;
    // Random pivot selection
    var pivotIdx = lo + Math.floor(Math.random() * (hi - lo + 1));
    var temp = arr[pivotIdx]; arr[pivotIdx] = arr[hi]; arr[hi] = temp;

    var pivot = arr[hi], p = lo;
    for (var i = lo; i < hi; i++) {
        if (arr[i] < pivot) {
            var t = arr[i]; arr[i] = arr[p]; arr[p] = t;
            p++;
        }
    }
    var t = arr[p]; arr[p] = arr[hi]; arr[hi] = t;

    quickSort(arr, lo, p - 1);
    quickSort(arr, p + 1, hi);
}

// Randomized Quick Select — O(n) expected
function quickSelect(arr, lo, hi, k) {
    if (lo === hi) return arr[lo];
    var pivotIdx = lo + Math.floor(Math.random() * (hi - lo + 1));
    var t = arr[pivotIdx]; arr[pivotIdx] = arr[hi]; arr[hi] = t;

    var pivot = arr[hi], p = lo;
    for (var i = lo; i < hi; i++) {
        if (arr[i] < pivot) {
            var t = arr[i]; arr[i] = arr[p]; arr[p] = t;
            p++;
        }
    }
    t = arr[p]; arr[p] = arr[hi]; arr[hi] = t;

    if (p === k) return arr[p];
    if (p < k) return quickSelect(arr, p + 1, hi, k);
    return quickSelect(arr, lo, p - 1, k);
}
