Binary Lifting

Hard Time: O(n log n) build, O(log n) query Space: O(n log n)

How It Works

Binary lifting precomputes up[v][k] = the 2^k-th ancestor of node v in a tree. By expressing any distance as a sum of powers of two, we can jump to any ancestor in O(log n) time. This is the standard approach for answering Lowest Common Ancestor (LCA) queries efficiently.

The precomputation: up[v][0] = parent of v. For k > 0: up[v][k] = up[up[v][k-1]][k-1] (the 2^k-th ancestor is the 2^(k-1)-th ancestor of the 2^(k-1)-th ancestor). This builds the table in O(n log n) time and space.

For LCA: first bring both nodes to the same depth using binary lifting, then lift both nodes simultaneously until they meet. Each step doubles the jump distance, ensuring O(log n) total steps.

Sparse Table of Ancestors
up[v][k] stores the 2^k-th ancestor. Any ancestor query decomposes into O(log n) jumps.
Power-of-Two Jumps
Jump 2^k, 2^(k-1), ..., 2^0 steps. Like binary representation of the target distance.
LCA in O(log n)
Equalize depths, then lift both nodes until they meet. The standard offline LCA technique.
O(n log n) Preprocessing
Build the table with DFS + dynamic programming on the ancestor table.

Implementation

// Binary Lifting for LCA
var LOG = 20;
var up = [];  // up[v][k] = 2^k-th ancestor of v
var depth = [];

function buildBinaryLifting(adj, root, n) {
    up = new Array(n);
    depth = new Array(n).fill(0);
    for (var i = 0; i < n; i++) up[i] = new Array(LOG).fill(-1);

    // BFS to set parent and depth
    var queue = [root];
    var visited = new Array(n).fill(false);
    visited[root] = true;
    while (queue.length) {
        var v = queue.shift();
        for (var i = 0; i < adj[v].length; i++) {
            var u = adj[v][i];
            if (!visited[u]) {
                visited[u] = true;
                up[u][0] = v;
                depth[u] = depth[v] + 1;
                queue.push(u);
            }
        }
    }
    // Fill sparse table
    for (var k = 1; k < LOG; k++) {
        for (var v = 0; v < n; v++) {
            if (up[v][k-1] !== -1)
                up[v][k] = up[up[v][k-1]][k-1];
        }
    }
}

function lca(u, v) {
    if (depth[u] < depth[v]) { var t = u; u = v; v = t; }
    var diff = depth[u] - depth[v];
    for (var k = 0; k < LOG; k++) {
        if ((diff >> k) & 1) u = up[u][k];
    }
    if (u === v) return u;
    for (var k = LOG - 1; k >= 0; k--) {
        if (up[u][k] !== up[v][k]) {
            u = up[u][k]; v = up[v][k];
        }
    }
    return up[u][0];
}
