Top-K Elements

Medium Time: O(n log k) Space: O(k)

How It Works

The Top-K pattern finds the k largest (or smallest) elements efficiently. The optimal approach uses a min-heap of size k: iterate through all n elements, maintaining only the k largest in the heap. Each insertion/removal is O(log k), giving O(n log k) total — better than O(n log n) sorting.

Alternatively, Quickselect (partition-based) achieves O(n) average time: partition around a random pivot, recurse into the half containing the k-th element. This is the same idea behind finding the median in O(n) average time.

The pattern applies to: k largest/smallest elements, k most frequent elements (use frequency map + heap), k closest points, and streaming top-k where elements arrive one at a time and the heap maintains a running answer.

Min-Heap of Size K
For top-k largest: maintain a min-heap of size k. The heap root is the k-th largest.
Quickselect
O(n) average: partition around pivot, recurse into the side containing k-th element.
Frequency + Heap
For k most frequent: build frequency map O(n), then use heap to extract top-k O(m log k).
Streaming
Process elements one at a time. Heap always holds the current top-k answer.

// Kth Largest Element — Quickselect (LeetCode #215)
function findKthLargest(nums, k) {
    k = nums.length - k; // convert to k-th smallest
    function quickselect(lo, hi) {
        var pivot = nums[hi], p = lo;
        for (var i = lo; i < hi; i++) {
            if (nums[i] <= pivot) {
                var t = nums[i]; nums[i] = nums[p]; nums[p] = t;
                p++;
            }
        }
        var t = nums[p]; nums[p] = nums[hi]; nums[hi] = t;
        if (p === k) return nums[p];
        if (p < k) return quickselect(p + 1, hi);
        return quickselect(lo, p - 1);
    }
    return quickselect(0, nums.length - 1);
}

// Top K Frequent Elements (LeetCode #347)
function topKFrequent(nums, k) {
    var freq = {};
    for (var i = 0; i < nums.length; i++) {
        freq[nums[i]] = (freq[nums[i]] || 0) + 1;
    }
    // Bucket sort by frequency
    var buckets = new Array(nums.length + 1);
    for (var num in freq) {
        var f = freq[num];
        if (!buckets[f]) buckets[f] = [];
        buckets[f].push(Number(num));
    }
    var result = [];
    for (var i = buckets.length - 1; i >= 0 && result.length < k; i--) {
        if (buckets[i]) result = result.concat(buckets[i]);
    }
    return result.slice(0, k);
}
