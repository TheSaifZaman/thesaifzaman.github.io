String Hashing

Medium Time: O(n) build, O(1) query Space: O(n)

How It Works

String hashing converts a string into a numeric fingerprint, enabling O(1) substring comparison after O(n) preprocessing. The polynomial rolling hash is: hash(s) = s[0]*p^(n-1) + s[1]*p^(n-2) + ... + s[n-1]*p^0 (mod m), where p is a prime base and m is a large prime modulus.

By precomputing prefix hashes and power arrays, the hash of any substring s[l..r] can be computed in O(1): hash(l,r) = (prefix[r+1] - prefix[l] * pow[r-l+1]) mod m. This enables O(1) string comparison (with small collision probability).

Applications include: Rabin-Karp pattern matching (O(n+m)), finding the longest palindromic substring, counting distinct substrings, and detecting duplicate substrings. Use double hashing (two different (p, m) pairs) to minimize collision probability.

Polynomial Hash
hash = sum of s[i] * p^(n-1-i) mod m. O(n) to compute, O(1) for substring hash.
Rolling Hash
Update hash in O(1) when sliding a window: remove leftmost char, add rightmost char.
Collision Risk
Hash match ≠ guaranteed string match. Use double hashing for high-stakes applications.
Rabin-Karp
Pattern matching in O(n+m) average: compute pattern hash, slide window over text comparing hashes.

// String Hashing — polynomial rolling hash
function StringHash(s) {
    var P = 31, MOD = 1e9 + 9;
    var n = s.length;
    this.prefix = new Array(n + 1);
    this.pow = new Array(n + 1);
    this.prefix[0] = 0;
    this.pow[0] = 1;
    for (var i = 0; i < n; i++) {
        this.prefix[i + 1] = (this.prefix[i] + (s.charCodeAt(i) - 96) * this.pow[0]) % MOD;
        // Actually: prefix[i+1] = (prefix[i] * P + char) % MOD
        this.pow[i + 1] = (this.pow[i] * P) % MOD;
    }
    // Correct implementation:
    this.prefix[0] = 0;
    for (var i = 0; i < n; i++) {
        this.prefix[i + 1] = (this.prefix[i] * P + (s.charCodeAt(i) - 96)) % MOD;
    }
    this.MOD = MOD;
}

StringHash.prototype.getHash = function(l, r) {
    // hash of s[l..r]
    var h = (this.prefix[r + 1] - this.prefix[l] * this.pow[r - l + 1]) % this.MOD;
    return (h + this.MOD) % this.MOD;
};

// Rabin-Karp Pattern Matching
function rabinKarp(text, pattern) {
    var P = 31, MOD = 1e9 + 9;
    var n = text.length, m = pattern.length;
    if (m > n) return [];

    var patHash = 0, pow = 1;
    for (var i = 0; i < m; i++) {
        patHash = (patHash * P + (pattern.charCodeAt(i) - 96)) % MOD;
        if (i > 0) pow = (pow * P) % MOD;
    }

    var textHash = 0, results = [];
    for (var i = 0; i < n; i++) {
        textHash = (textHash * P + (text.charCodeAt(i) - 96)) % MOD;
        if (i >= m) {
            textHash = (textHash - (text.charCodeAt(i - m) - 96) * pow % MOD * P % MOD + MOD * 2) % MOD;
        }
        if (i >= m - 1 && textHash === patHash) {
            // Verify to handle collisions
            if (text.substring(i - m + 1, i + 1) === pattern) results.push(i - m + 1);
        }
    }
    return results;
}
