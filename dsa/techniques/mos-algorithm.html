Mo's Algorithm

Hard Time: O((n + q) * √n) Space: O(n)

How It Works

Mo's Algorithm answers offline range queries in O((n + q) * sqrt(n)) time by sorting queries in a specific order that minimizes the total pointer movement. It maintains a “current range” [curL, curR] and incrementally adds/removes elements as it moves to the next query.

The key insight is the sorting order: divide the array into blocks of size sqrt(n). Sort queries by (block of L, R). Within the same block of L, queries are sorted by R. This ensures the right pointer moves O(n * sqrt(n)) total, and the left pointer moves O(n * sqrt(n)) total across all queries.

Mo's algorithm works for any problem where you can efficiently add/remove a single element to/from the current answer. Examples: count distinct elements in range, range mode query, range frequency query. It requires offline processing (all queries known in advance).

Block Decomposition
Divide indices into blocks of size sqrt(n). Sort queries by (block of L, then R).
Incremental Updates
Move [curL, curR] to each query by adding/removing one element at a time.
O((n+q) * sqrt(n))
Total pointer movement is bounded. Each pointer moves O(n * sqrt(n)) across all queries.
Offline Only
All queries must be known in advance. Not suitable for online/interactive problems.

Implementation

// Mo's Algorithm — count distinct elements in range queries
function mosAlgorithm(arr, queries) {
    var n = arr.length;
    var block = Math.floor(Math.sqrt(n)) || 1;

    // Sort queries by (block of L, R)
    var indexed = queries.map(function(q, i) { return [q[0], q[1], i]; });
    indexed.sort(function(a, b) {
        var blockA = Math.floor(a[0] / block);
        var blockB = Math.floor(b[0] / block);
        if (blockA !== blockB) return blockA - blockB;
        return (blockA % 2 === 0) ? a[1] - b[1] : b[1] - a[1];
    });

    var freq = {}, distinct = 0;
    var curL = 0, curR = -1;
    var result = new Array(queries.length);

    function add(idx) {
        freq[arr[idx]] = (freq[arr[idx]] || 0) + 1;
        if (freq[arr[idx]] === 1) distinct++;
    }
    function remove(idx) {
        freq[arr[idx]]--;
        if (freq[arr[idx]] === 0) distinct--;
    }

    for (var i = 0; i < indexed.length; i++) {
        var qL = indexed[i][0], qR = indexed[i][1], qIdx = indexed[i][2];
        while (curR < qR) add(++curR);
        while (curL > qL) add(--curL);
        while (curR > qR) remove(curR--);
        while (curL < qL) remove(curL++);
        result[qIdx] = distinct;
    }
    return result;
}
