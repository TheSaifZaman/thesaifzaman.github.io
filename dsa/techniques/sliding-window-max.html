Sliding Window Maximum

Medium Time: O(n) Space: O(k)

How It Works

The Sliding Window Maximum problem finds the maximum element in every contiguous window of size k. A naive O(nk) approach checks every element in each window. The optimal O(n) solution uses a monotonic deque (double-ended queue) that maintains indices in decreasing order of their values.

As the window slides, we: (1) remove indices from the front that are outside the window, (2) remove indices from the back whose values are smaller than the new element (they can never be the maximum while the new element exists in the window), (3) add the new index to the back. The front always holds the maximum.

The monotonic deque ensures each element is added and removed at most once, giving amortized O(1) per element. This pattern extends to sliding window minimum (use increasing deque) and is used in DP optimization.

Monotonic Deque
Maintains indices in decreasing order of values. Front = index of current maximum.
Remove Expired
Pop from front if the index is outside the current window [i-k+1, i].
Maintain Monotonicity
Pop from back while back value <= new value. Smaller values cannot be future maxima.
Amortized O(1)
Each element enters and leaves the deque exactly once. Total: O(n).

Implementation

// Sliding Window Maximum (LeetCode #239)
function maxSlidingWindow(nums, k) {
    var deque = []; // stores indices
    var result = [];

    for (var i = 0; i < nums.length; i++) {
        // Remove indices outside window
        while (deque.length && deque[0] < i - k + 1) {
            deque.shift();
        }
        // Remove smaller elements from back
        while (deque.length && nums[deque[deque.length - 1]] <= nums[i]) {
            deque.pop();
        }
        deque.push(i);
        // Window is complete when i >= k - 1
        if (i >= k - 1) {
            result.push(nums[deque[0]]);
        }
    }
    return result;
}

// Sliding Window Minimum (same pattern, flip comparison)
function minSlidingWindow(nums, k) {
    var deque = [], result = [];
    for (var i = 0; i < nums.length; i++) {
        while (deque.length && deque[0] < i - k + 1) deque.shift();
        while (deque.length && nums[deque[deque.length - 1]] >= nums[i]) deque.pop();
        deque.push(i);
        if (i >= k - 1) result.push(nums[deque[0]]);
    }
    return result;
}
