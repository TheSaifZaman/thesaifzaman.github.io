Line Sweep

Hard Time: O(n log n) Space: O(n)

How It Works

Line sweep (or sweep line) processes events along a sorted axis. Imagine a vertical line sweeping left-to-right across a plane. At each event point (start or end of an interval/rectangle), we update an active set. This converts 2D problems into a sequence of 1D operations.

Events are created for each interval: a “start” event (+1) and an “end” event (-1). After sorting all events by position, sweep through them maintaining a counter or balanced BST of active intervals. The running state at each event point captures the answer.

Applications include: counting maximum overlapping intervals, computing union of rectangles, the skyline problem, and computational geometry problems like finding intersecting line segments. The technique is powerful because it reduces complexity from O(n²) to O(n log n).

Event Points
Create +1 events at interval starts and -1 events at interval ends. Sort all events.
Active Set
Maintain the set of currently active intervals as the sweep line moves through events.
Sort by Position
Events must be processed in order. Ties broken by type (end before start, or vice versa).
2D to 1D Reduction
Converts spatial problems into event-sequence problems, enabling efficient solutions.

// The Skyline Problem (LeetCode #218)
function getSkyline(buildings) {
    var events = [];
    for (var i = 0; i < buildings.length; i++) {
        events.push([buildings[i][0], -buildings[i][2]]); // start: negative height
        events.push([buildings[i][1], buildings[i][2]]);   // end: positive height
    }
    events.sort(function(a, b) { return a[0] - b[0] || a[1] - b[1]; });

    var result = [], heights = [0], prevMax = 0;
    // Simplified — in practice use a max-heap for heights
    for (var i = 0; i < events.length; i++) {
        if (events[i][1] < 0) heights.push(-events[i][1]);
        else heights.splice(heights.indexOf(events[i][1]), 1);
        var curMax = Math.max.apply(null, heights);
        if (curMax !== prevMax) {
            result.push([events[i][0], curMax]);
            prevMax = curMax;
        }
    }
    return result;
}

// Maximum overlap count
function maxOverlap(intervals) {
    var events = [];
    for (var i = 0; i < intervals.length; i++) {
        events.push([intervals[i][0], 1]);
        events.push([intervals[i][1], -1]);
    }
    events.sort(function(a, b) { return a[0] - b[0] || a[1] - b[1]; });
    var count = 0, maxCount = 0;
    for (var i = 0; i < events.length; i++) {
        count += events[i][1];
        maxCount = Math.max(maxCount, count);
    }
    return maxCount;
}
