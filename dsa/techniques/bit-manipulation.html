Bit Manipulation

Medium Time: O(1)–O(n) Space: O(1)

How It Works

Bit manipulation operates directly on the binary representation of integers using bitwise operators: AND (&), OR (|), XOR (^), NOT (~), and shifts (<<, >>). These operations execute in a single CPU cycle, making them extremely fast.

Key tricks include: n & (n-1) clears the lowest set bit (used to count bits), n & (-n) isolates the lowest set bit, and a ^ b finds bits that differ. XOR is particularly powerful: a ^ a = 0 and a ^ 0 = a, enabling finding the single unique number in a list of pairs.

Bit manipulation is essential for power-of-two checks, parity, subsets (each subset maps to a bitmask), and space-efficient flags. Many interview problems have elegant O(1) space solutions using bit tricks.

XOR Properties
a ^ a = 0, a ^ 0 = a. XOR cancels pairs, leaving the unique element.
Clear Lowest Bit
n & (n-1) removes the rightmost set bit. Useful for counting set bits.
Check Power of Two
n > 0 && (n & (n-1)) === 0. Powers of two have exactly one set bit.
Bitmask Subsets
For n items, iterate 0 to 2^n - 1. Each integer represents a subset via its bits.

Implementation

// Single Number (LeetCode #136) — XOR all elements
function singleNumber(nums) {
    var result = 0;
    for (var i = 0; i < nums.length; i++) result ^= nums[i];
    return result;
}

// Number of 1 Bits (LeetCode #191)
function hammingWeight(n) {
    var count = 0;
    while (n) { n &= (n - 1); count++; }
    return count;
}

// Power of Two
function isPowerOfTwo(n) {
    return n > 0 && (n & (n - 1)) === 0;
}

// Counting Bits (LeetCode #338)
function countBits(n) {
    var dp = new Array(n + 1).fill(0);
    for (var i = 1; i <= n; i++) dp[i] = dp[i >> 1] + (i & 1);
    return dp;
}

// Missing Number using XOR
function missingNumber(nums) {
    var xor = nums.length;
    for (var i = 0; i < nums.length; i++) xor ^= i ^ nums[i];
    return xor;
}
