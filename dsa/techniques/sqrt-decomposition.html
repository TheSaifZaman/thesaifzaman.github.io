Sqrt Decomposition

Medium Time: O(√n) per query Space: O(n)

How It Works

Sqrt decomposition divides an array of n elements into blocks of size sqrt(n). Each block precomputes an aggregate (sum, min, max, etc.). Queries span at most 2 partial blocks plus O(sqrt(n)) complete blocks, giving O(sqrt(n)) per query.

For a range query [l, r]: handle the partial left block (l to block end), all complete middle blocks (precomputed answers), and the partial right block (block start to r). Updates: modify the element and recompute its block in O(sqrt(n)).

While segment trees provide O(log n) queries, sqrt decomposition is simpler to implement, has smaller constants, and handles certain operations (like Mo's algorithm) more naturally. It is the workhorse technique for competitive programming when full generality of segment trees is not needed.

Block Size √n
Divide array into blocks of size √n. Total blocks: √n.
O(√n) Query
At most 2 partial blocks + O(√n) full blocks = O(√n) total work per query.
O(√n) Update

Implementation

// Sqrt Decomposition — range sum query with point update
function SqrtDecomp(arr) {
    this.arr = arr.slice();
    this.n = arr.length;
    this.block = Math.floor(Math.sqrt(this.n)) || 1;
    this.blocks = new Array(Math.ceil(this.n / this.block)).fill(0);
    for (var i = 0; i < this.n; i++) {
        this.blocks[Math.floor(i / this.block)] += arr[i];
    }
}

SqrtDecomp.prototype.update = function(idx, val) {
    var b = Math.floor(idx / this.block);
    this.blocks[b] += val - this.arr[idx];
    this.arr[idx] = val;
};

SqrtDecomp.prototype.query = function(l, r) {
    var sum = 0;
    var bl = Math.floor(l / this.block);
    var br = Math.floor(r / this.block);

    if (bl === br) {
        for (var i = l; i <= r; i++) sum += this.arr[i];
        return sum;
    }
    // Partial left block
    for (var i = l; i < (bl + 1) * this.block; i++) sum += this.arr[i];
    // Full middle blocks
    for (var b = bl + 1; b < br; b++) sum += this.blocks[b];
    // Partial right block
    for (var i = br * this.block; i <= r; i++) sum += this.arr[i];
    return sum;
};
