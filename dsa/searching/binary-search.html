Binary Search

Halves search space each step in a sorted array — O(log n)

Concept

Binary Search works on a sorted array by repeatedly dividing the search interval in half. Compare the target with the middle element: if equal, found; if target is smaller, search the left half; otherwise search the right half.

  • Time Complexity: O(log n)
  • Space Complexity: O(1) iterative, O(log n) recursive
  • Precondition: Array must be sorted
  • Key Idea: Eliminate half the remaining elements at each step

Implementation

function binarySearch(arr, target) {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);

    if (arr[mid] === target) {
      return mid;           // Found
    } else if (arr[mid] < target) {
      left = mid + 1;       // Search right half
    } else {
      right = mid - 1;      // Search left half
    }
  }

  return -1;                 // Not found
}
