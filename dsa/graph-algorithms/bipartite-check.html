Bipartite Graph Check

Medium Time: O(V+E) Space: O(V)

How It Works

A graph is bipartite if its vertices can be divided into two disjoint sets such that every edge connects a vertex in one set to a vertex in the other set. Equivalently, the graph is 2-colorable — no two adjacent vertices share the same color.

The check uses BFS or DFS: start by coloring any uncolored vertex with color 0, then color all its neighbors with color 1. Continue this process. If we ever find a neighbor with the same color as the current vertex, the graph is not bipartite (it contains an odd-length cycle).

A graph is bipartite if and only if it contains no odd-length cycles. All trees, paths, and even-length cycles are bipartite. Odd-length cycles make bipartiteness impossible.

2-Coloring
Try to assign two colors such that no adjacent vertices share a color.
BFS/DFS
Start from any vertex, alternate colors layer by layer. Conflict = not bipartite.
Odd Cycles
A graph is bipartite iff it has no odd-length cycles.
Applications
Matching problems, scheduling, conflict detection, graph theory foundations.

Implementation

function isBipartite(n, adj) {
    var color = new Array(n).fill(-1);

    for (var start = 0; start < n; start++) {
        if (color[start] !== -1) continue;
        color[start] = 0;
        var queue = [start];

        while (queue.length > 0) {
            var u = queue.shift();
            for (var i = 0; i < adj[u].length; i++) {
                var v = adj[u][i];
                if (color[v] === -1) {
                    color[v] = 1 - color[u]; // opposite color
                    queue.push(v);
                } else if (color[v] === color[u]) {
                    return false; // same color = conflict
                }
            }
        }
    }
    return true;
}

// DFS version
function isBipartiteDFS(n, adj) {
    var color = new Array(n).fill(-1);

    function dfs(u, c) {
        color[u] = c;
        for (var i = 0; i < adj[u].length; i++) {
            var v = adj[u][i];
            if (color[v] === -1) {
                if (!dfs(v, 1 - c)) return false;
            } else if (color[v] === c) {
                return false;
            }
        }
        return true;
    }

    for (var i = 0; i < n; i++) {
        if (color[i] === -1 && !dfs(i, 0)) return false;
    }
    return true;
}
