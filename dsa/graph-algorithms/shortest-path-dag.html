Shortest Path in DAG

Medium Time: O(V+E) Space: O(V)

How It Works

In a Directed Acyclic Graph (DAG), the shortest path from a source can be found in O(V+E) time — faster than Dijkstra's or Bellman-Ford. The key insight is that topological order guarantees we process each vertex only after all vertices that can reach it.

Step 1: Compute topological order of the DAG. Step 2: Initialize distances to infinity (source = 0). Step 3: Process vertices in topological order. For each vertex u, relax all its outgoing edges: if dist[u] + w(u,v) < dist[v], update dist[v].

This works because in topological order, when we process vertex u, we've already processed all vertices that can reach u, so dist[u] is already finalized. This also handles negative edge weights correctly (unlike Dijkstra), since there are no cycles in a DAG.

O(V+E) Time
Faster than Dijkstra's O((V+E) log V) because we exploit the DAG structure.
Negative Weights OK
No cycles means negative weights are safe — no negative cycle issue.
Topological Order
Process vertices in order so all predecessors are already finalized.
Applications
Critical path method, task scheduling, longest path in DAG (negate weights).

Implementation

function shortestPathDAG(n, adj, src) {
    // Step 1: Topological sort (Kahn's)
    var indegree = new Array(n).fill(0);
    for (var u = 0; u < n; u++) {
        for (var i = 0; i < adj[u].length; i++) {
            indegree[adj[u][i][0]]++;
        }
    }
    var queue = [];
    for (var i = 0; i < n; i++) {
        if (indegree[i] === 0) queue.push(i);
    }
    var topoOrder = [];
    while (queue.length > 0) {
        var u = queue.shift();
        topoOrder.push(u);
        for (var j = 0; j < adj[u].length; j++) {
            var v = adj[u][j][0];
            if (--indegree[v] === 0) queue.push(v);
        }
    }

    // Step 2: Initialize distances
    var dist = new Array(n).fill(Infinity);
    dist[src] = 0;

    // Step 3: Relax in topological order
    for (var t = 0; t < topoOrder.length; t++) {
        var u = topoOrder[t];
        if (dist[u] === Infinity) continue;
        for (var k = 0; k < adj[u].length; k++) {
            var v = adj[u][k][0], w = adj[u][k][1];
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
            }
        }
    }

    return dist;
}
