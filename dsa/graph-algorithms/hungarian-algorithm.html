Hungarian Algorithm

Hard Time: O(n³) Space: O(n²)

How It Works

The Hungarian Algorithm solves the assignment problem: given n workers and n jobs with a cost matrix, find the optimal one-to-one assignment that minimizes total cost. It works on bipartite graphs where one set represents workers and the other represents jobs.

The algorithm works by manipulating a cost matrix through row and column reductions. Step 1: Subtract the row minimum from each row. Step 2: Subtract the column minimum from each column. Step 3: Cover all zeros with minimum number of lines. If lines equals n, an optimal assignment exists among the zeros. Otherwise, adjust the matrix and repeat.

The key insight is that subtracting a constant from a row or column doesn't change the optimal assignment, only the total cost. The algorithm creates enough zeros to find a perfect matching.

Row Reduction
Subtract minimum value from each row, creating at least one zero per row.
Column Reduction
Subtract minimum value from each column after row reduction.
Line Covering
Cover all zeros with minimum lines. If count = n, assignment exists.
Applications
Job scheduling, resource allocation, matching problems, transportation.

Implementation

// Hungarian Algorithm (simplified Kuhn-Munkres)
function hungarian(cost) {
    var n = cost.length;
    var u = new Array(n + 1).fill(0);
    var v = new Array(n + 1).fill(0);
    var p = new Array(n + 1).fill(0);
    var way = new Array(n + 1).fill(0);

    for (var i = 1; i <= n; i++) {
        p[0] = i;
        var j0 = 0;
        var minv = new Array(n + 1).fill(Infinity);
        var used = new Array(n + 1).fill(false);

        do {
            used[j0] = true;
            var i0 = p[j0], delta = Infinity, j1;

            for (var j = 1; j <= n; j++) {
                if (!used[j]) {
                    var cur = cost[i0-1][j-1] - u[i0] - v[j];
                    if (cur < minv[j]) {
                        minv[j] = cur;
                        way[j] = j0;
                    }
                    if (minv[j] < delta) {
                        delta = minv[j];
                        j1 = j;
                    }
                }
            }

            for (var j = 0; j <= n; j++) {
                if (used[j]) { u[p[j]] += delta; v[j] -= delta; }
                else { minv[j] -= delta; }
            }
            j0 = j1;
        } while (p[j0] !== 0);

        do {
            var j1 = way[j0];
            p[j0] = p[j1];
            j0 = j1;
        } while (j0);
    }

    // Extract assignment
    var result = new Array(n);
    for (var j = 1; j <= n; j++) {
        result[p[j] - 1] = j - 1;
    }
    return result;
}
