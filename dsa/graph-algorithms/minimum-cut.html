Minimum Cut (Karger's)

Hard Time: O(V² log V) Space: O(V+E)

How It Works

A minimum cut is the smallest set of edges whose removal disconnects the graph into two or more components. Karger's algorithm finds it using random edge contraction.

The algorithm repeatedly picks a random edge and contracts it — merging its two endpoints into a single super-node. Self-loops are removed. This continues until only two super-nodes remain. The edges between them form a cut. By repeating this process O(V² log V) times and taking the minimum, the algorithm finds the minimum cut with high probability.

The probability that a single run finds the minimum cut is at least 2/(V(V-1)). Running O(V² log V) times gives high success probability. The algorithm is simple to implement and is one of the most elegant randomized algorithms.

Edge Contraction
Merge two endpoints of a random edge into one super-node. Remove self-loops.
Randomized
Single run has ≥ 2/(n(n-1)) success probability. Repeat for high confidence.
Min-Cut Max-Flow
Min cut value equals max flow (by max-flow min-cut theorem).
Applications
Network reliability, image segmentation, clustering, graph partitioning.

Implementation

// Karger's Minimum Cut Algorithm
function kargerMinCut(n, edges) {
    var bestCut = Infinity;
    var trials = n * n * Math.ceil(Math.log(n));

    for (var t = 0; t < trials; t++) {
        // Initialize Union-Find
        var parent = [];
        for (var i = 0; i < n; i++) parent.push(i);

        function find(x) {
            if (parent[x] !== x) parent[x] = find(parent[x]);
            return parent[x];
        }

        var remaining = n;
        var edgeCopy = edges.slice();

        // Shuffle edges for randomness
        for (var i = edgeCopy.length - 1; i > 0; i--) {
            var j = Math.floor(Math.random() * (i + 1));
            var tmp = edgeCopy[i];
            edgeCopy[i] = edgeCopy[j];
            edgeCopy[j] = tmp;
        }

        var ei = 0;
        while (remaining > 2 && ei < edgeCopy.length) {
            var u = find(edgeCopy[ei][0]);
            var v = find(edgeCopy[ei][1]);
            if (u !== v) {
                parent[u] = v;
                remaining--;
            }
            ei++;
        }

        // Count crossing edges
        var cut = 0;
        for (var e = 0; e < edges.length; e++) {
            if (find(edges[e][0]) !== find(edges[e][1])) cut++;
        }

        bestCut = Math.min(bestCut, cut);
    }

    return bestCut;
}
