Network Flow (Ford-Fulkerson)

Hard Time: O(V·E²) Space: O(V+E)

How It Works

The Ford-Fulkerson method computes the maximum flow from a source to a sink in a flow network. Each edge has a capacity, and flow along an edge cannot exceed its capacity. The algorithm repeatedly finds augmenting paths from source to sink and pushes flow along them.

The algorithm uses a residual graph that shows remaining capacity on each edge. For each augmenting path found (via BFS in Edmonds-Karp variant), the bottleneck (minimum residual capacity along the path) is identified, and flow is pushed. Reverse edges in the residual graph allow flow to be rerouted.

The process continues until no augmenting path exists in the residual graph. By the Max-Flow Min-Cut Theorem, the maximum flow equals the minimum cut capacity — the minimum total capacity of edges that, if removed, would disconnect source from sink.

Augmenting Paths
Find paths from source to sink in residual graph. Push flow equal to the bottleneck.
Residual Graph
Tracks remaining capacity. Includes reverse edges for flow rerouting.
Edmonds-Karp
Uses BFS for augmenting paths → O(VE²) guaranteed. Ford-Fulkerson with DFS can be slower.
Max-Flow = Min-Cut
Maximum flow equals the minimum capacity of edges separating source from sink.

Implementation

// Edmonds-Karp (BFS-based Ford-Fulkerson)
function maxFlow(n, capacity, source, sink) {
    var flow = 0;

    while (true) {
        // BFS to find augmenting path
        var parent = new Array(n).fill(-1);
        parent[source] = source;
        var queue = [[source, Infinity]];

        while (queue.length > 0) {
            var curr = queue.shift();
            var u = curr[0], pushed = curr[1];

            for (var v = 0; v < n; v++) {
                if (parent[v] === -1 && capacity[u][v] > 0) {
                    parent[v] = u;
                    var newFlow = Math.min(pushed, capacity[u][v]);
                    if (v === sink) {
                        // Augment flow along path
                        var node = sink;
                        while (node !== source) {
                            var prev = parent[node];
                            capacity[prev][node] -= newFlow;
                            capacity[node][prev] += newFlow;
                            node = prev;
                        }
                        flow += newFlow;
                        break;
                    }
                    queue.push([v, newFlow]);
                }
            }
            if (parent[sink] !== -1) break;
        }

        if (parent[sink] === -1) break; // no augmenting path
    }
    return flow;
}
