Tarjan's SCC

Hard Time: O(V+E) Space: O(V)

How It Works

Tarjan's algorithm finds all strongly connected components (SCCs) in a directed graph using a single DFS pass. An SCC is a maximal set of vertices where every vertex is reachable from every other vertex in the set.

Each vertex is assigned a discovery index (disc) and a low-link value (low). The low-link value represents the smallest discovery index reachable from the vertex through the DFS subtree. Vertices are pushed onto a stack during DFS. When a vertex u has low[u] === disc[u], it is the root of an SCC, and all vertices above it on the stack form that SCC.

The key insight is that if low[u] equals disc[u], then u cannot reach any vertex discovered before it (in the current DFS path), making it the earliest vertex in its SCC. All vertices on the stack above u must be in the same SCC.

Single Pass DFS
Finds all SCCs in one DFS traversal — more efficient than Kosaraju's two passes.
Low-Link Values
low[u] = min discovery index reachable from u's subtree via back/cross edges to stack vertices.
SCC Root
A vertex u is an SCC root when disc[u] === low[u]. Pop stack to get the SCC.
Applications
2-SAT, finding cycles, condensation graphs, compiler optimization.

Implementation

function tarjanSCC(n, adj) {
    var disc = new Array(n).fill(-1);
    var low = new Array(n).fill(-1);
    var onStack = new Array(n).fill(false);
    var stack = [];
    var timer = 0;
    var sccs = [];

    function dfs(u) {
        disc[u] = low[u] = timer++;
        stack.push(u);
        onStack[u] = true;

        for (var i = 0; i < adj[u].length; i++) {
            var v = adj[u][i];
            if (disc[v] === -1) {
                dfs(v);
                low[u] = Math.min(low[u], low[v]);
            } else if (onStack[v]) {
                low[u] = Math.min(low[u], disc[v]);
            }
        }

        // If u is root of an SCC
        if (low[u] === disc[u]) {
            var scc = [];
            while (true) {
                var v = stack.pop();
                onStack[v] = false;
                scc.push(v);
                if (v === u) break;
            }
            sccs.push(scc);
        }
    }

    for (var i = 0; i < n; i++) {
        if (disc[i] === -1) dfs(i);
    }
    return sccs;
}
