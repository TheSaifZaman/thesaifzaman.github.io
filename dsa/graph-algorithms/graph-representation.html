Graph Representation

Easy Time: — Space: O(V+E) list / O(V²) matrix

How It Works

Graphs can be represented in multiple ways depending on the problem requirements. The two most common representations are the adjacency matrix and the adjacency list. Each has trade-offs in terms of space, edge lookup time, and iteration efficiency.

An adjacency matrix is a 2D array of size V×V where entry (i,j) is 1 (or the edge weight) if there is an edge from vertex i to vertex j, and 0 otherwise. It provides O(1) edge lookup but uses O(V²) space, making it ideal for dense graphs.

An adjacency list stores, for each vertex, a list of its neighbors. It uses O(V+E) space and is preferred for sparse graphs. Iterating over neighbors is efficient, but checking if a specific edge exists takes O(degree) time.

Adjacency Matrix
O(1) edge lookup, O(V²) space. Best for dense graphs or when frequent edge existence checks are needed.
Adjacency List
O(V+E) space, efficient neighbor iteration. Best for sparse graphs and most graph algorithms.
Edge List
Simple list of (u, v, weight) tuples. O(E) space, used in algorithms like Kruskal's MST.
Directed vs Undirected
Undirected graphs store each edge twice (both directions). Directed graphs store edges only in the specified direction.

Implementation

// Adjacency List representation
function buildAdjList(n, edges) {
    var adj = [];
    for (var i = 0; i < n; i++) adj.push([]);
    for (var e = 0; e < edges.length; e++) {
        var u = edges[e][0], v = edges[e][1];
        adj[u].push(v);
        adj[v].push(u); // remove for directed
    }
    return adj;
}

// Adjacency Matrix representation
function buildAdjMatrix(n, edges) {
    var mat = [];
    for (var i = 0; i < n; i++) {
        mat.push(new Array(n).fill(0));
    }
    for (var e = 0; e < edges.length; e++) {
        var u = edges[e][0], v = edges[e][1];
        var w = edges[e][2] || 1;
        mat[u][v] = w;
        mat[v][u] = w; // remove for directed
    }
    return mat;
}

// Check if edge exists — Matrix: O(1)
function hasEdgeMatrix(mat, u, v) {
    return mat[u][v] !== 0;
}

// Check if edge exists — List: O(degree)
function hasEdgeList(adj, u, v) {
    return adj[u].indexOf(v) !== -1;
}

// Get all neighbors
function neighbors(adj, u) {
    return adj[u];
}
