Floyd-Warshall Algorithm

Medium Time: O(V³) Space: O(V²)

How It Works

Floyd-Warshall computes the shortest paths between all pairs of vertices simultaneously. It uses dynamic programming, considering each vertex as a potential intermediate node in paths between every pair.

The algorithm maintains a distance matrix where dist[i][j] represents the shortest known distance from i to j. For each intermediate vertex k (from 0 to V-1), it checks whether the path i→k→j is shorter than the current i→j path: dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]).

The key insight is that after considering vertices 0 through k as intermediates, dist[i][j] holds the shortest path from i to j using only vertices 0..k as intermediates. After all V iterations, all shortest paths are found.

All-Pairs
Computes shortest distances between every pair of vertices in one run.
DP Recurrence
dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]) for each intermediate k.
Negative Cycles
Detected when any diagonal entry dist[i][i] becomes negative.
Use Cases
Dense graphs, transitive closure, finding graph diameter, detecting negative cycles.

Implementation

function floydWarshall(n, edges) {
    var INF = Infinity;
    // Initialize distance matrix
    var dist = [];
    for (var i = 0; i < n; i++) {
        dist.push([]);
        for (var j = 0; j < n; j++) {
            dist[i].push(i === j ? 0 : INF);
        }
    }

    // Fill in direct edges
    for (var e = 0; e < edges.length; e++) {
        var u = edges[e][0], v = edges[e][1], w = edges[e][2];
        dist[u][v] = w;
    }

    // DP: consider each vertex as intermediate
    for (var k = 0; k < n; k++) {
        for (var i = 0; i < n; i++) {
            for (var j = 0; j < n; j++) {
                if (dist[i][k] + dist[k][j] < dist[i][j]) {
                    dist[i][j] = dist[i][k] + dist[k][j];
                }
            }
        }
    }

    // Check negative cycles
    for (var i = 0; i < n; i++) {
        if (dist[i][i] < 0) return null; // negative cycle
    }

    return dist;
}
