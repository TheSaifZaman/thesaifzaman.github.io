Euler Path & Circuit

Medium Time: O(E) Space: O(V+E)

How It Works

An Euler path visits every edge exactly once. An Euler circuit is an Euler path that starts and ends at the same vertex. These exist under specific degree conditions.

For an undirected graph: an Euler circuit exists if every vertex has even degree. An Euler path exists if exactly two vertices have odd degree (these are the start and end). Hierholzer's algorithm finds the circuit/path in O(E) time by following edges, backtracking when stuck, and splicing together sub-circuits.

Hierholzer's works by starting at a valid vertex, following unused edges until returning to the start (forming a circuit). If unvisited edges remain, it picks a vertex on the current circuit with unused edges, finds another sub-circuit, and merges them. This process repeats until all edges are used.

Euler Circuit
All vertices even degree. Starts and ends at same vertex.
Euler Path
Exactly 2 vertices with odd degree. Start at one, end at the other.
Hierholzer's
Greedily follow edges, backtrack to splice sub-circuits. O(E) time.
Connectivity
Graph must be connected (ignoring isolated vertices) for Euler path/circuit to exist.

// Hierholzer's Algorithm for Euler Circuit/Path
function findEulerPath(n, adj) {
    // Find start vertex (odd degree vertex or 0)
    var start = 0;
    for (var i = 0; i < n; i++) {
        if (adj[i].length % 2 === 1) { start = i; break; }
    }

    // Deep copy adjacency lists (we remove edges as we use them)
    var graph = [];
    for (var i = 0; i < n; i++) graph.push(adj[i].slice());

    var stack = [start];
    var path = [];

    while (stack.length > 0) {
        var v = stack[stack.length - 1];
        if (graph[v].length > 0) {
            var u = graph[v].pop();
            // Remove reverse edge for undirected graph
            var idx = graph[u].indexOf(v);
            if (idx !== -1) graph[u].splice(idx, 1);
            stack.push(u);
        } else {
            path.push(stack.pop());
        }
    }

    return path.reverse();
}

// Check if Euler path/circuit exists
function hasEulerPath(n, adj) {
    var oddCount = 0;
    for (var i = 0; i < n; i++) {
        if (adj[i].length % 2 === 1) oddCount++;
    }
    if (oddCount === 0) return "circuit";
    if (oddCount === 2) return "path";
    return "none";
}
