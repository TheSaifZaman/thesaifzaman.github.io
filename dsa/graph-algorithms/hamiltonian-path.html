Hamiltonian Path & Cycle

Hard Time: O(n!) Space: O(n)

How It Works

A Hamiltonian path visits every vertex exactly once. A Hamiltonian cycle is a Hamiltonian path that returns to the starting vertex. Unlike Euler paths (which visit every edge), Hamiltonian paths visit every vertex.

Finding a Hamiltonian path is NP-complete — no known polynomial-time algorithm exists. The brute-force approach uses backtracking: try to extend the current path by visiting an unvisited adjacent vertex. If stuck (all neighbors visited), backtrack and try a different neighbor.

For small n, a DP with bitmask approach can solve it in O(n² · 2^n), which is faster than O(n!). The state dp[mask][i] represents whether we can reach vertex i having visited exactly the vertices in the bitmask.

NP-Complete
No known polynomial solution. Exponential time is unavoidable in worst case.
Backtracking
Try extending path with each unvisited neighbor. Backtrack on dead ends. O(n!).
DP + Bitmask
dp[mask][i] tracks reachability. O(n² · 2^n) — better than backtracking for n ≤ 20.
vs Euler Path
Euler = visit every edge once. Hamiltonian = visit every vertex once. Very different problems.

Implementation

// Hamiltonian Path using backtracking
function hamiltonianPath(n, adj) {
    var path = [];
    var visited = new Array(n).fill(false);

    function backtrack(u) {
        path.push(u);
        visited[u] = true;

        if (path.length === n) return true; // found!

        for (var i = 0; i < adj[u].length; i++) {
            var v = adj[u][i];
            if (!visited[v]) {
                if (backtrack(v)) return true;
            }
        }

        // Backtrack
        path.pop();
        visited[u] = false;
        return false;
    }

    // Try each vertex as start
    for (var start = 0; start < n; start++) {
        if (backtrack(start)) return path;
    }
    return null; // no Hamiltonian path
}

// DP + Bitmask approach: O(n² · 2^n)
function hamiltonianDP(n, adj) {
    var ALL = (1 << n) - 1;
    var dp = [];
    for (var m = 0; m <= ALL; m++) dp.push(new Array(n).fill(false));

    // Base: single vertices
    for (var i = 0; i < n; i++) dp[1 << i][i] = true;

    for (var mask = 1; mask <= ALL; mask++) {
        for (var u = 0; u < n; u++) {
            if (!dp[mask][u]) continue;
            for (var j = 0; j < adj[u].length; j++) {
                var v = adj[u][j];
                if (!(mask & (1 << v))) {
                    dp[mask | (1 << v)][v] = true;
                }
            }
        }
    }

    for (var i = 0; i < n; i++) {
        if (dp[ALL][i]) return true;
    }
    return false;
}
