Articulation Points & Bridges

Hard Time: O(V+E) Space: O(V)

How It Works

An articulation point (cut vertex) is a vertex whose removal disconnects the graph. A bridge (cut edge) is an edge whose removal disconnects the graph. Both are critical for understanding graph connectivity and vulnerability.

The algorithm uses a modified DFS with discovery times (disc) and low-link values (low). For vertex u with child v in DFS tree: u is an articulation point if low[v] ≥ disc[u] (v cannot reach above u), OR u is the root with 2+ children. An edge u-v is a bridge if low[v] > disc[u] (v cannot reach u or above without this edge).

The low-link value low[u] is the minimum discovery time reachable from u's subtree through back edges. If a child's subtree has no back edge reaching above u, removing u would disconnect that subtree.

Articulation Point
Vertex whose removal increases the number of connected components.
Bridge
Edge whose removal increases connected components. low[v] > disc[u].
DFS Tree
Back edges create alternative paths. If no back edge bypasses u, u is critical.
Applications
Network reliability, finding vulnerable links, 2-edge-connected components.

Implementation

function findArticulationPointsAndBridges(n, adj) {
    var disc = new Array(n).fill(-1);
    var low = new Array(n).fill(-1);
    var parent = new Array(n).fill(-1);
    var ap = new Array(n).fill(false);
    var bridges = [];
    var timer = 0;

    function dfs(u) {
        disc[u] = low[u] = timer++;
        var children = 0;

        for (var i = 0; i < adj[u].length; i++) {
            var v = adj[u][i];
            if (disc[v] === -1) {
                children++;
                parent[v] = u;
                dfs(v);
                low[u] = Math.min(low[u], low[v]);

                // Articulation point checks
                if (parent[u] === -1 && children > 1) ap[u] = true;
                if (parent[u] !== -1 && low[v] >= disc[u]) ap[u] = true;

                // Bridge check
                if (low[v] > disc[u]) bridges.push([u, v]);
            } else if (v !== parent[u]) {
                low[u] = Math.min(low[u], disc[v]);
            }
        }
    }

    for (var i = 0; i < n; i++) {
        if (disc[i] === -1) dfs(i);
    }

    var articulationPoints = [];
    for (var i = 0; i < n; i++) {
        if (ap[i]) articulationPoints.push(i);
    }

    return { articulationPoints: articulationPoints, bridges: bridges };
}
