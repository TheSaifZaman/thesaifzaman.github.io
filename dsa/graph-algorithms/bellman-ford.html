Bellman-Ford Algorithm

Medium Time: O(V·E) Space: O(V)

How It Works

The Bellman-Ford algorithm finds shortest paths from a single source vertex to all other vertices, even when the graph contains negative edge weights. Unlike Dijkstra's, it can also detect negative weight cycles.

The algorithm performs V-1 iterations over all edges. In each iteration, it attempts to relax every edge: if dist[u] + w(u,v) < dist[v], then dist[v] is updated. After V-1 iterations, all shortest paths with at most V-1 edges are found. A final V-th iteration checks for negative cycles — if any distance can still be reduced, a negative cycle exists.

The intuition is that the shortest path between any two vertices contains at most V-1 edges. Each iteration guarantees that paths using one more edge are correctly computed. So after V-1 iterations, all shortest paths are found.

V-1 Iterations
Relax all edges V-1 times. Each iteration extends shortest paths by one more edge.
Negative Weights
Handles negative edge weights correctly, unlike Dijkstra's algorithm.
Cycle Detection
An extra iteration detects negative cycles if any distance can still be reduced.
Trade-off
Slower than Dijkstra O(V·E) vs O((V+E)log V), but more versatile.

Implementation

function bellmanFord(n, edges, src) {
    var dist = new Array(n).fill(Infinity);
    var prev = new Array(n).fill(-1);
    dist[src] = 0;

    // Relax all edges V-1 times
    for (var i = 0; i < n - 1; i++) {
        for (var j = 0; j < edges.length; j++) {
            var u = edges[j][0], v = edges[j][1], w = edges[j][2];
            if (dist[u] !== Infinity && dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                prev[v] = u;
            }
        }
    }

    // Check for negative cycles
    for (var k = 0; k < edges.length; k++) {
        var u = edges[k][0], v = edges[k][1], w = edges[k][2];
        if (dist[u] !== Infinity && dist[u] + w < dist[v]) {
            return { dist: null, hasNegativeCycle: true };
        }
    }

    return { dist: dist, prev: prev, hasNegativeCycle: false };
}
