Dijkstra's Algorithm

Medium Time: O((V+E) log V) Space: O(V)

How It Works

Dijkstra's algorithm finds the shortest path from a single source vertex to all other vertices in a weighted graph with non-negative edge weights. It uses a greedy approach, always processing the unvisited vertex with the smallest known distance.

The algorithm maintains a distance array initialized to infinity (except the source which is 0) and a priority queue (min-heap). At each step, it extracts the vertex with minimum distance, then relaxes all its outgoing edges: if going through this vertex offers a shorter path to a neighbor, the distance is updated.

The key insight is that once a vertex is extracted from the priority queue, its shortest distance is finalized. This is because all remaining unprocessed vertices have equal or greater distance, and edge weights are non-negative, so no future path can improve the distance.

Greedy Choice
Always process the closest unvisited vertex. This guarantees optimality with non-negative weights.
Relaxation
For each neighbor v of current node u: if dist[u] + w(u,v) < dist[v], update dist[v].
Priority Queue
A min-heap enables efficient extraction of the minimum distance vertex in O(log V).
Limitation
Does not work with negative edge weights. Use Bellman-Ford for graphs with negative weights.

// Dijkstra's Algorithm using a min-heap (priority queue)
function dijkstra(graph, src, n) {
    var dist = new Array(n).fill(Infinity);
    var visited = new Array(n).fill(false);
    var prev = new Array(n).fill(-1);
    dist[src] = 0;

    // Simple priority queue using array
    var pq = [[0, src]]; // [distance, vertex]

    while (pq.length > 0) {
        // Extract minimum
        pq.sort(function(a, b) { return a[0] - b[0]; });
        var curr = pq.shift();
        var d = curr[0], u = curr[1];

        if (visited[u]) continue;
        visited[u] = true;

        // Relax neighbors
        for (var i = 0; i < graph[u].length; i++) {
            var v = graph[u][i][0];
            var w = graph[u][i][1];
            if (!visited[v] && dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                prev[v] = u;
                pq.push([dist[v], v]);
            }
        }
    }

    return { dist: dist, prev: prev };
}

// Reconstruct path from source to target
function getPath(prev, target) {
    var path = [];
    for (var v = target; v !== -1; v = prev[v]) {
        path.unshift(v);
    }
    return path;
}
