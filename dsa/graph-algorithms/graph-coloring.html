Graph Coloring

Hard Time: O(n!) Space: O(V)

How It Works

Graph coloring assigns colors to vertices such that no two adjacent vertices share the same color. The minimum number of colors needed is called the chromatic number. Finding the chromatic number is NP-hard.

The greedy coloring algorithm processes vertices in order, assigning each vertex the smallest color not used by its neighbors. While not guaranteed to use the minimum colors, it provides an upper bound and works well in practice. The result depends on vertex ordering.

For exact solutions, backtracking tries each color for each vertex and prunes when a conflict is found. The Four Color Theorem states that any planar graph can be colored with at most 4 colors.

Chromatic Number
Minimum colors needed. Finding it exactly is NP-hard.
Greedy Coloring
Assign smallest available color to each vertex. At most Δ+1 colors (Δ = max degree).
Four Color Theorem
Any planar graph can be colored with at most 4 colors.
Applications
Scheduling, register allocation, map coloring, frequency assignment.

// Greedy Graph Coloring
function greedyColoring(n, adj) {
    var color = new Array(n).fill(-1);

    for (var u = 0; u < n; u++) {
        // Find colors used by neighbors
        var usedColors = {};
        for (var i = 0; i < adj[u].length; i++) {
            var v = adj[u][i];
            if (color[v] !== -1) usedColors[color[v]] = true;
        }

        // Assign smallest available color
        var c = 0;
        while (usedColors[c]) c++;
        color[u] = c;
    }

    return color;
}

// Backtracking: can graph be colored with m colors?
function canColor(n, adj, m) {
    var color = new Array(n).fill(-1);

    function isSafe(u, c) {
        for (var i = 0; i < adj[u].length; i++) {
            if (color[adj[u][i]] === c) return false;
        }
        return true;
    }

    function backtrack(u) {
        if (u === n) return true;
        for (var c = 0; c < m; c++) {
            if (isSafe(u, c)) {
                color[u] = c;
                if (backtrack(u + 1)) return true;
                color[u] = -1;
            }
        }
        return false;
    }

    return backtrack(0) ? color : null;
}
