Topological Sort

Medium Time: O(V+E) Space: O(V)

How It Works

Topological sort produces a linear ordering of vertices in a Directed Acyclic Graph (DAG) such that for every directed edge u→v, vertex u comes before v in the ordering. This is essential for scheduling tasks with dependencies.

Kahn's Algorithm (BFS): Compute in-degrees for all vertices. Add all vertices with in-degree 0 to a queue. Repeatedly remove a vertex from the queue, add it to the result, and decrement in-degrees of its neighbors. If a neighbor's in-degree becomes 0, add it to the queue. If all vertices are processed, we have a valid topological order; otherwise, a cycle exists.

DFS approach: Run DFS, and when a vertex finishes (all descendants explored), push it onto a stack. The stack gives the topological order when popped.

DAG Only
Topological sort is only possible for directed acyclic graphs. Cycles make it impossible.
Kahn's (BFS)
Uses in-degree counting and a queue. Also detects cycles if not all vertices are processed.
DFS Approach
Post-order DFS traversal reversed gives topological order.
Applications
Build systems, course scheduling, task ordering, dependency resolution.

// Kahn's Algorithm (BFS-based Topological Sort)
function topologicalSort(n, edges) {
    var adj = [], indegree = new Array(n).fill(0);
    for (var i = 0; i < n; i++) adj.push([]);

    for (var e = 0; e < edges.length; e++) {
        adj[edges[e][0]].push(edges[e][1]);
        indegree[edges[e][1]]++;
    }

    var queue = [];
    for (var i = 0; i < n; i++) {
        if (indegree[i] === 0) queue.push(i);
    }

    var order = [];
    while (queue.length > 0) {
        var u = queue.shift();
        order.push(u);
        for (var j = 0; j < adj[u].length; j++) {
            var v = adj[u][j];
            indegree[v]--;
            if (indegree[v] === 0) queue.push(v);
        }
    }

    if (order.length !== n) return null; // cycle detected
    return order;
}

// DFS-based Topological Sort
function topSortDFS(n, adj) {
    var visited = new Array(n).fill(false);
    var stack = [];

    function dfs(u) {
        visited[u] = true;
        for (var i = 0; i < adj[u].length; i++) {
            if (!visited[adj[u][i]]) dfs(adj[u][i]);
        }
        stack.push(u);
    }

    for (var i = 0; i < n; i++) {
        if (!visited[i]) dfs(i);
    }
    return stack.reverse();
}
