Binary Search Tree

Medium Time: O(log n) avg Space: O(n)

How It Works

A Binary Search Tree (BST) is a binary tree where for every node, all values in its left subtree are less than the node's value, and all values in its right subtree are greater than the node's value. This ordering property enables efficient search, insertion, and deletion in O(log n) average time.

Insertion starts at the root and compares the new value with the current node: go left if smaller, right if larger, until reaching a null position where the new node is placed. Search follows the same path. Deletion has three cases: leaf node (simply remove), one child (replace with child), or two children (replace with inorder successor or predecessor).

The efficiency of a BST depends on its balance. In the worst case (inserting sorted data), the tree degenerates into a linked list with O(n) operations. This motivates self-balancing trees like AVL and Red-Black trees.

BST Property
Left subtree values < node value < right subtree values. This invariant must hold for every node.
Search: O(log n) avg
Each comparison eliminates half the remaining tree, similar to binary search on arrays.
Insertion
Follow the search path until a null link is found, then attach the new node there.
Inorder = Sorted
An inorder traversal of a BST always produces values in ascending order.

Implementation

// BST Node
function TreeNode(val) {
    this.val = val;
    this.left = null;
    this.right = null;
}

// Insert into BST
function insertIntoBST(root, val) {
    if (!root) return new TreeNode(val);
    if (val < root.val) {
        root.left = insertIntoBST(root.left, val);
    } else {
        root.right = insertIntoBST(root.right, val);
    }
    return root;
}

// Search in BST
function searchBST(root, val) {
    if (!root || root.val === val) return root;
    if (val < root.val) return searchBST(root.left, val);
    return searchBST(root.right, val);
}

// Delete from BST
function deleteNode(root, key) {
    if (!root) return null;
    if (key < root.val) {
        root.left = deleteNode(root.left, key);
    } else if (key > root.val) {
        root.right = deleteNode(root.right, key);
    } else {
        // Found node to delete
        if (!root.left) return root.right;
        if (!root.right) return root.left;
        // Two children: find inorder successor
        var successor = root.right;
        while (successor.left) successor = successor.left;
        root.val = successor.val;
        root.right = deleteNode(root.right, successor.val);
    }
    return root;
}

// Validate BST
function isValidBST(root, min, max) {
    if (min === undefined) min = -Infinity;
    if (max === undefined) max = Infinity;
    if (!root) return true;
    if (root.val <= min || root.val >= max) return false;
    return isValidBST(root.left, min, root.val) &&
           isValidBST(root.right, root.val, max);
}

// Kth Smallest Element
function kthSmallest(root, k) {
    var count = 0;
    var result = null;
    function inorder(node) {
        if (!node || result !== null) return;
        inorder(node.left);
        count++;
        if (count === k) { result = node.val; return; }
        inorder(node.right);
    }
    inorder(root);
    return result;
}
