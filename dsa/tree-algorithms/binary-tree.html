Binary Tree

Medium Time: O(n) traversal Space: O(h) recursive

How It Works

A binary tree is a hierarchical data structure in which each node has at most two children, commonly referred to as the left and right child. Binary trees form the basis for many advanced structures like BSTs, heaps, and segment trees.

The three fundamental traversal orders are:

  • Inorder (Left, Root, Right) — visits nodes in sorted order for BSTs.
  • Preorder (Root, Left, Right) — useful for copying/serializing a tree.
  • Postorder (Left, Right, Root) — useful for deletion and expression evaluation.
  • Level Order (BFS) — visits nodes level by level using a queue.

The depth of a node is the number of edges from the root to that node. The height of the tree is the maximum depth among all nodes. A balanced tree has height O(log n), while a skewed tree degrades to O(n).

Recursive Structure
Every subtree of a binary tree is itself a binary tree, enabling elegant recursive algorithms.
Traversal Orders
Inorder, preorder, and postorder each process nodes in a different sequence, suited to different tasks.
Level Order (BFS)
Uses a queue to visit all nodes at depth d before any node at depth d+1.
Height & Depth
Height determines the worst-case time for recursive operations: O(h) per path from root to leaf.

Implementation

// Binary Tree Node
function TreeNode(val, left, right) {
    this.val = val;
    this.left = left || null;
    this.right = right || null;
}

// Inorder Traversal (Left, Root, Right)
function inorderTraversal(root) {
    var result = [];
    function dfs(node) {
        if (!node) return;
        dfs(node.left);
        result.push(node.val);
        dfs(node.right);
    }
    dfs(root);
    return result;
}

// Preorder Traversal (Root, Left, Right)
function preorderTraversal(root) {
    var result = [];
    function dfs(node) {
        if (!node) return;
        result.push(node.val);
        dfs(node.left);
        dfs(node.right);
    }
    dfs(root);
    return result;
}

// Level Order Traversal (BFS)
function levelOrder(root) {
    if (!root) return [];
    var result = [];
    var queue = [root];
    while (queue.length) {
        var size = queue.length;
        var level = [];
        for (var i = 0; i < size; i++) {
            var node = queue.shift();
            level.push(node.val);
            if (node.left) queue.push(node.left);
            if (node.right) queue.push(node.right);
        }
        result.push(level);
    }
    return result;
}

// Maximum Depth
function maxDepth(root) {
    if (!root) return 0;
    return 1 + Math.max(maxDepth(root.left), maxDepth(root.right));
}

// Invert Binary Tree
function invertTree(root) {
    if (!root) return null;
    var temp = root.left;
    root.left = invertTree(root.right);
    root.right = invertTree(temp);
    return root;
}
