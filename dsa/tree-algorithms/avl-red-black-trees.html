AVL & Red-Black Trees

Hard Time: O(log n) Space: O(n)

How It Works

AVL trees and Red-Black trees are self-balancing binary search trees that guarantee O(log n) height. They achieve this by performing rotations after insertions and deletions to restore balance invariants.

An AVL tree (Adelson-Velsky and Landis) maintains the invariant that for every node, the heights of its left and right subtrees differ by at most 1. When an insertion or deletion violates this, one of four rotation types restores balance:

  • Left Rotation (LL) — right-heavy subtree; rotate left around the unbalanced node.
  • Right Rotation (RR) — left-heavy subtree; rotate right around the unbalanced node.
  • Left-Right (LR) — left child is right-heavy; left rotate child, then right rotate node.
  • Right-Left (RL) — right child is left-heavy; right rotate child, then left rotate node.

A Red-Black tree uses node coloring (red or black) with five properties to ensure the longest path is at most twice the shortest. Red-Black trees are used in many standard library implementations (e.g., Java TreeMap, C++ std::map) because they require fewer rotations on average than AVL trees.

AVL Balance Factor
balance(node) = height(left) - height(right). Must be in {-1, 0, 1} for every node.
Rotation Types
Single rotations (LL, RR) and double rotations (LR, RL) restore the balance invariant in O(1).
Red-Black Properties
Root is black, red nodes have black children, every path has equal black-height.
Trade-offs
AVL is more strictly balanced (faster lookups), Red-Black has fewer rotations (faster inserts/deletes).

Implementation

// AVL Tree Node
function AVLNode(val) {
    this.val = val;
    this.left = null;
    this.right = null;
    this.height = 1;
}

function height(node) {
    return node ? node.height : 0;
}

function updateHeight(node) {
    node.height = 1 + Math.max(height(node.left), height(node.right));
}

function getBalance(node) {
    return node ? height(node.left) - height(node.right) : 0;
}

// Right Rotation
function rightRotate(y) {
    var x = y.left;
    var T2 = x.right;
    x.right = y;
    y.left = T2;
    updateHeight(y);
    updateHeight(x);
    return x; // new root
}

// Left Rotation
function leftRotate(x) {
    var y = x.right;
    var T2 = y.left;
    y.left = x;
    x.right = T2;
    updateHeight(x);
    updateHeight(y);
    return y; // new root
}

// AVL Insert
function avlInsert(node, val) {
    // Standard BST insert
    if (!node) return new AVLNode(val);
    if (val < node.val) {
        node.left = avlInsert(node.left, val);
    } else if (val > node.val) {
        node.right = avlInsert(node.right, val);
    } else {
        return node; // no duplicates
    }

    updateHeight(node);
    var balance = getBalance(node);

    // Left Left Case
    if (balance > 1 && val < node.left.val)
        return rightRotate(node);
    // Right Right Case
    if (balance < -1 && val > node.right.val)
        return leftRotate(node);
    // Left Right Case
    if (balance > 1 && val > node.left.val) {
        node.left = leftRotate(node.left);
        return rightRotate(node);
    }
    // Right Left Case
    if (balance < -1 && val < node.right.val) {
        node.right = rightRotate(node.right);
        return leftRotate(node);
    }

    return node;
}
