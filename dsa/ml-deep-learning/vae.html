Variational Autoencoder (VAE)

Hard Loss: Recon + KL Latent: continuous

How It Works

A Variational Autoencoder (VAE) is a generative model that learns a latent representation of data. Unlike a regular autoencoder, the VAE encodes inputs into a probability distribution (mean and variance) in latent space, then samples from this distribution to decode.

The architecture has three parts: an Encoder that maps input x to latent distribution parameters (mu, sigma), a Sampling step using the reparameterization trick (z = mu + sigma * epsilon, where epsilon ~ N(0,1)), and a Decoder that reconstructs x from z.

The loss function has two terms: reconstruction loss (how well the decoder rebuilds the input) and KL divergence (how close the latent distribution is to a standard normal). This encourages a smooth, continuous latent space where similar inputs are nearby.

Key Properties

  • Probabilistic Encoding: Maps to distributions, not points — enables generation.
  • Reparameterization Trick: Makes sampling differentiable for backpropagation.
  • KL Divergence: Regularizes latent space to be smooth and continuous.
  • Generation: Sample from N(0,1) in latent space and decode to generate new data.

Implementation

function VAE(inputDim, latentDim) {
    var encoder = buildEncoder(inputDim, latentDim);
    var decoder = buildDecoder(latentDim, inputDim);

    this.encode = function(x) {
        var params = encoder.forward(x);
        var mu = params.slice(0, latentDim);
        var logVar = params.slice(latentDim);
        return { mu: mu, logVar: logVar };
    };

    this.reparameterize = function(mu, logVar) {
        var std = logVar.map(function(v) { return Math.exp(0.5 * v); });
        var eps = randomNormal(latentDim);
        return mu.map(function(m, i) { return m + std[i] * eps[i]; });
    };

    this.forward = function(x) {
        var enc = this.encode(x);
        var z = this.reparameterize(enc.mu, enc.logVar);
        var recon = decoder.forward(z);
        return { recon: recon, mu: enc.mu, logVar: enc.logVar };
    };

    // Loss = Reconstruction + KL divergence
    this.loss = function(x, recon, mu, logVar) {
        var reconLoss = mse(x, recon);
        var kl = -0.5 * sum(logVar.map(function(lv, i) {
            return 1 + lv - mu[i]*mu[i] - Math.exp(lv);
        }));
        return reconLoss + kl;
    };
}
