Recurrent Neural Network (RNN)

Hard Time: O(T·h²) Space: O(T·h)

How It Works

A Recurrent Neural Network (RNN) processes sequential data by maintaining a hidden state that acts as memory. At each time step, the network takes an input and the previous hidden state, computes a new hidden state, and optionally produces an output.

The key equation is: h_t = tanh(W_hh * h_{t-1} + W_xh * x_t + b). The same weights are shared across all time steps (parameter sharing), making RNNs suitable for variable-length sequences.

Vanilla RNNs suffer from the vanishing gradient problem: gradients diminish exponentially during backpropagation through time (BPTT), making it hard to learn long-range dependencies. This motivated LSTM and GRU architectures.

Key Properties

  • Sequential Processing: Processes one element at a time, maintaining hidden state.
  • Parameter Sharing: Same weights used at every time step.
  • Vanishing Gradients: Difficult to learn long-range dependencies.
  • Applications: Text generation, time series, speech recognition.

Implementation

function RNN(inputSize, hiddenSize, outputSize) {
    var Wxh = randomMatrix(hiddenSize, inputSize);
    var Whh = randomMatrix(hiddenSize, hiddenSize);
    var Why = randomMatrix(outputSize, hiddenSize);

    this.forward = function(inputs) {
        var h = zeroVector(hiddenSize);
        var outputs = [];
        for (var t = 0; t < inputs.length; t++) {
            // h_t = tanh(Wxh * x_t + Whh * h_{t-1})
            h = tanh(add(matVec(Wxh, inputs[t]), matVec(Whh, h)));
            // y_t = Why * h_t
            outputs.push(matVec(Why, h));
        }
        return outputs;
    };
}
