Dropout

Easy Overhead: minimal Rate: typically 0.2-0.5

How It Works

Dropout is a regularization technique where, during training, each neuron is randomly "dropped out" (set to zero) with a probability p. This forces the network to not rely on any single neuron and to learn redundant representations.

During training, each neuron's output is multiplied by a Bernoulli random variable (0 with probability p, 1 with probability 1-p). At test time, all neurons are active, but outputs are scaled by (1-p) to compensate (or equivalently, training outputs are scaled by 1/(1-p) — "inverted dropout").

Dropout can be viewed as training an exponential number of different "thinned" networks that share weights, with the full network at test time being an approximate ensemble average of all possible thinned networks.

Key Properties

  • Regularization: Prevents overfitting by reducing co-adaptation between neurons.
  • Implicit Ensemble: Approximates averaging over all 2^n possible sub-networks.
  • Simple to Implement: Just multiply activations by random 0/1 mask during training.
  • Scale at Test Time: Multiply by (1-p) or use inverted dropout during training.

Implementation

function Dropout(rate) {
    var p = rate; // probability of dropping

    this.forward = function(x, training) {
        if (!training) return x; // No dropout at test time

        // Generate random mask
        var mask = x.map(function(val) {
            return Math.random() > p ? 1 : 0;
        });
        // Apply mask and scale (inverted dropout)
        return x.map(function(val, i) {
            return mask[i] ? val / (1 - p) : 0;
        });
    };
}
