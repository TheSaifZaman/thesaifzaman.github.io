Convolutional Neural Network (CNN)

Hard Conv: O(n·k²·c) Params: O(k²·c·f)

How It Works

A Convolutional Neural Network (CNN) is designed to process grid-structured data like images. Instead of fully connecting every neuron, CNNs use small learnable filters (kernels) that slide across the input, computing dot products at each position to produce feature maps.

Key operations include: Convolution (filter sliding over input), Pooling (downsampling to reduce spatial dimensions), and Fully Connected layers at the end for classification. CNNs exploit three properties: local connectivity, parameter sharing, and translation equivariance.

A typical architecture stacks multiple conv + pool layers to learn hierarchical features: edges in early layers, textures in middle layers, and objects in later layers.

Key Properties

  • Parameter Sharing: Same filter applied everywhere — far fewer parameters than MLP.
  • Translation Equivariance: Detecting a feature is independent of its position.
  • Hierarchical Features: Deep layers learn increasingly abstract representations.
  • Pooling: Provides spatial invariance and reduces computational cost.

Implementation

function conv2d(input, kernel, stride) {
    var inH = input.length, inW = input[0].length;
    var kH = kernel.length, kW = kernel[0].length;
    var outH = Math.floor((inH - kH) / stride) + 1;
    var outW = Math.floor((inW - kW) / stride) + 1;
    var output = [];

    for (var i = 0; i < outH; i++) {
        output[i] = [];
        for (var j = 0; j < outW; j++) {
            var sum = 0;
            for (var ki = 0; ki < kH; ki++) {
                for (var kj = 0; kj < kW; kj++) {
                    sum += input[i * stride + ki][j * stride + kj] * kernel[ki][kj];
                }
            }
            output[i][j] = Math.max(0, sum); // ReLU
        }
    }
    return output;
}
