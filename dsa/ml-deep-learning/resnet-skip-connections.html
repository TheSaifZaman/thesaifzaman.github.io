ResNet & Skip Connections

Hard Depth: 50-152 layers Key: identity shortcuts

How It Works

ResNet (Residual Network) introduced skip connections (shortcut connections) that allow the input to a block of layers to bypass those layers and be added directly to the output. Instead of learning H(x) directly, the network learns the residual F(x) = H(x) - x, making the output H(x) = F(x) + x.

This seemingly simple change solved the degradation problem: very deep networks (50+ layers) were actually performing worse than shallower ones during training. Skip connections provide gradient highways that allow gradients to flow directly backward through the network, enabling training of networks with hundreds of layers.

If the optimal transformation is close to identity (which is common in deep networks), it is much easier for the network to learn F(x) = 0 (residual = zero) than to learn an identity mapping from scratch.

Key Properties

  • Residual Learning: Learn F(x) = H(x) - x instead of H(x) directly.
  • Gradient Highway: Skip connections allow unimpeded gradient flow during backprop.
  • Very Deep Networks: Enables training of 100+ layer networks without degradation.
  • Identity Shortcut: If the block learns nothing useful, it defaults to identity (no harm done).

Implementation

function ResidualBlock(channels) {
    var conv1 = Conv2d(channels, channels, 3);
    var bn1 = BatchNorm(channels);
    var conv2 = Conv2d(channels, channels, 3);
    var bn2 = BatchNorm(channels);

    this.forward = function(x) {
        var identity = x;                    // Save input
        var out = relu(bn1.forward(conv1.forward(x)));
        out = bn2.forward(conv2.forward(out));
        out = add(out, identity);            // Skip connection!
        return relu(out);                    // H(x) = F(x) + x
    };
}

function ResNet(numBlocks) {
    var blocks = [];
    for (var i = 0; i < numBlocks; i++) {
        blocks.push(new ResidualBlock(64));
    }
    this.forward = function(x) {
        for (var i = 0; i < blocks.length; i++) {
            x = blocks[i].forward(x);
        }
        return x;
    };
}
