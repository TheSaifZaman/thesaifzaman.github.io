Multi-Layer Perceptron (MLP)

Medium Forward: O(n·h) Backprop: O(n·h)

How It Works

A Multi-Layer Perceptron (MLP) is the foundational feedforward neural network. It consists of an input layer, one or more hidden layers, and an output layer. Each neuron computes a weighted sum of its inputs, adds a bias, and applies a non-linear activation function.

Information flows forward from input to output (forward pass). During training, the error is propagated backward through the network (backpropagation), and weights are updated using gradient descent to minimize the loss function.

MLPs are universal function approximators: given enough neurons in a single hidden layer, they can approximate any continuous function to arbitrary precision (Universal Approximation Theorem).

Key Properties

  • Feedforward: No cycles; signal flows strictly from input to output.
  • Fully Connected: Every neuron connects to all neurons in the next layer.
  • Activation Functions: ReLU, sigmoid, tanh introduce non-linearity.
  • Universal Approximator: Can learn any continuous function with sufficient capacity.

function MLP(layerSizes) {
    var weights = [], biases = [];
    for (var i = 0; i < layerSizes.length - 1; i++) {
        weights.push(randomMatrix(layerSizes[i], layerSizes[i+1]));
        biases.push(randomVector(layerSizes[i+1]));
    }

    function relu(x) { return Math.max(0, x); }

    this.forward = function(input) {
        var activation = input;
        for (var i = 0; i < weights.length; i++) {
            var z = matVecMul(weights[i], activation);
            for (var j = 0; j < z.length; j++) z[j] += biases[i][j];
            activation = (i < weights.length - 1)
                ? z.map(relu)           // Hidden: ReLU
                : z.map(sigmoid);       // Output: sigmoid
        }
        return activation;
    };
}
