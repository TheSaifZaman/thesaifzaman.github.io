Transformer

Hard Attention: O(n²·d) FFN: O(n·d²)

How It Works

The Transformer architecture (Vaswani et al., 2017) revolutionized deep learning by replacing recurrence with self-attention. It processes all positions in parallel, computing attention weights that determine how much each position should attend to every other position.

The core mechanism is Scaled Dot-Product Attention: Attention(Q,K,V) = softmax(QK^T / sqrt(d_k)) * V. Each token creates a Query (what am I looking for?), Key (what do I contain?), and Value (what information do I provide?). Multi-head attention runs this process multiple times in parallel with different projections.

The full Transformer has an encoder (self-attention + feedforward) and decoder (masked self-attention + cross-attention + feedforward), with residual connections and layer normalization throughout.

Key Properties

  • Self-Attention: Every token can directly attend to every other token — no sequential bottleneck.
  • Parallelizable: All positions computed simultaneously, unlike RNN.
  • Multi-Head: Multiple attention heads capture different relationship types.
  • Positional Encoding: Sine/cosine functions inject position information since there is no inherent order.

function scaledDotProductAttention(Q, K, V) {
    var dk = K[0].length;
    var scale = Math.sqrt(dk);
    // QK^T / sqrt(d_k)
    var scores = matMul(Q, transpose(K));
    for (var i = 0; i < scores.length; i++)
        for (var j = 0; j < scores[i].length; j++)
            scores[i][j] /= scale;
    // Softmax each row
    var attnWeights = scores.map(softmaxRow);
    // Weighted sum of values
    return matMul(attnWeights, V);
}

function multiHeadAttention(Q, K, V, numHeads) {
    var heads = [];
    for (var h = 0; h < numHeads; h++) {
        var Qh = project(Q, h), Kh = project(K, h), Vh = project(V, h);
        heads.push(scaledDotProductAttention(Qh, Kh, Vh));
    }
    return project(concat(heads), 'output');
}
