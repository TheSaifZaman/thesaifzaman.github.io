Batch Normalization

Medium Per layer: O(n·d) Params: 2d per layer

How It Works

Batch Normalization (BatchNorm) normalizes the activations of each layer across a mini-batch during training. For each feature, it subtracts the batch mean and divides by the batch standard deviation, then scales and shifts using learnable parameters gamma and beta.

The formula is: y = gamma * (x - mu_batch) / sqrt(sigma_batch^2 + epsilon) + beta. During inference, running averages of mean and variance (computed during training) are used instead of batch statistics.

BatchNorm addresses internal covariate shift: the distribution of layer inputs changes during training as previous layers update, making optimization harder. By normalizing, each layer receives inputs with stable statistics, enabling higher learning rates and faster convergence.

Key Properties

  • Stabilizes Training: Reduces sensitivity to weight initialization and learning rate.
  • Higher Learning Rates: Normalized inputs allow more aggressive optimization.
  • Slight Regularization: Batch statistics add noise that has a regularizing effect.
  • Learnable Parameters: Gamma (scale) and beta (shift) let the network undo normalization if needed.

Implementation

function BatchNorm(numFeatures) {
    var gamma = onesVector(numFeatures);
    var beta = zerosVector(numFeatures);
    var runningMean = zerosVector(numFeatures);
    var runningVar = onesVector(numFeatures);
    var eps = 1e-5, momentum = 0.1;

    this.forward = function(x, training) {
        if (training) {
            var mean = batchMean(x);
            var variance = batchVariance(x, mean);
            // Update running statistics
            for (var i = 0; i < numFeatures; i++) {
                runningMean[i] = (1-momentum)*runningMean[i] + momentum*mean[i];
                runningVar[i] = (1-momentum)*runningVar[i] + momentum*variance[i];
            }
        } else {
            var mean = runningMean;
            var variance = runningVar;
        }
        // Normalize, scale, shift
        return x.map(function(sample) {
            return sample.map(function(val, i) {
                return gamma[i] * (val - mean[i]) / Math.sqrt(variance[i] + eps) + beta[i];
            });
        });
    };
}
