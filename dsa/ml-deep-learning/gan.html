Generative Adversarial Network (GAN)

Hard Training: adversarial Loss: minimax

How It Works

A GAN (Generative Adversarial Network) consists of two neural networks competing against each other: a Generator that creates fake data and a Discriminator that tries to distinguish fake from real data.

The Generator takes random noise z as input and produces synthetic data G(z). The Discriminator takes both real data x and generated data G(z), outputting a probability that its input is real. Training alternates between updating the Discriminator (to better distinguish real vs fake) and the Generator (to better fool the Discriminator).

The minimax objective is: min_G max_D [E[log D(x)] + E[log(1 - D(G(z)))]]. At equilibrium, the Generator produces data indistinguishable from real, and the Discriminator outputs 0.5 for everything.

Key Properties

  • Adversarial Training: Two networks compete, driving each other to improve.
  • Implicit Density: Generator learns the data distribution without explicitly modeling it.
  • Mode Collapse: Generator may learn to produce only a few types of outputs.
  • Training Instability: Balancing Generator and Discriminator training is challenging.

Implementation

function trainGAN(generator, discriminator, realData, epochs) {
    for (var epoch = 0; epoch < epochs; epoch++) {
        // 1. Train Discriminator
        var realBatch = sampleBatch(realData);
        var noise = randomNoise(batchSize);
        var fakeBatch = generator.forward(noise);

        var dLossReal = -mean(log(discriminator.forward(realBatch)));
        var dLossFake = -mean(log(1 - discriminator.forward(fakeBatch)));
        discriminator.backward(dLossReal + dLossFake);
        discriminator.updateWeights();

        // 2. Train Generator
        noise = randomNoise(batchSize);
        fakeBatch = generator.forward(noise);
        var gLoss = -mean(log(discriminator.forward(fakeBatch)));
        generator.backward(gLoss);
        generator.updateWeights();
    }
}
