How It Works

Diffusion Models (Denoising Diffusion Probabilistic Models, DDPM) generate data by learning to reverse a gradual noising process. The forward process adds small amounts of Gaussian noise over T steps until the data becomes pure noise. The reverse process learns to denoise step by step, starting from random noise and producing a clean sample.

The forward process is defined as: q(x_t | x_{t-1}) = N(x_t; sqrt(1-beta_t) * x_{t-1}, beta_t * I). At each step, a small amount of noise beta_t is added. After T steps (typically T=1000), the data is approximately standard Gaussian noise.

A neural network (typically a U-Net) is trained to predict the noise that was added at each step. At generation time, we start with random noise and iteratively denoise using the learned model, producing realistic samples.

Key Properties

  • Forward Process: Gradually adds noise until data becomes Gaussian noise (fixed, no learning).
  • Reverse Process: Neural network learns to denoise step by step.
  • Stable Training: Simple MSE loss on noise prediction, no adversarial training.
  • High Quality: State-of-the-art image generation (Stable Diffusion, DALL-E, Midjourney).

function DiffusionModel(T, betaSchedule) {
    var betas = betaSchedule; // [beta_1, ..., beta_T]
    var alphas = betas.map(function(b) { return 1 - b; });
    var alphaBar = cumulativeProduct(alphas);

    // Training: predict noise
    this.trainStep = function(x0) {
        var t = Math.floor(Math.random() * T);
        var noise = randomNormal(x0.length);
        // x_t = sqrt(alpha_bar_t) * x_0 + sqrt(1-alpha_bar_t) * noise
        var sqrtAB = Math.sqrt(alphaBar[t]);
        var sqrt1mAB = Math.sqrt(1 - alphaBar[t]);
        var xt = x0.map(function(v, i) {
            return sqrtAB * v + sqrt1mAB * noise[i];
        });
        var predictedNoise = model.forward(xt, t);
        return mseLoss(predictedNoise, noise);
    };

    // Sampling: iterative denoising
    this.sample = function() {
        var x = randomNormal(dataDim); // Start from noise
        for (var t = T - 1; t >= 0; t--) {
            var predictedNoise = model.forward(x, t);
            x = denoise(x, predictedNoise, t, alphas, betas);
        }
        return x;
    };
}
