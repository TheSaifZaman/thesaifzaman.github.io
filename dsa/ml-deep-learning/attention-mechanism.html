Attention Mechanism

Hard Time: O(n·m) Self-Attn: O(n²)

How It Works

The Attention Mechanism allows a model to dynamically focus on different parts of the input when producing each part of the output. Originally introduced for seq2seq models, attention computes a weighted average of encoder hidden states, where weights reflect the relevance of each input position to the current output position.

There are several attention variants:

  • Additive (Bahdanau): score(s, h) = v^T * tanh(W_s * s + W_h * h)
  • Dot-Product: score(s, h) = s^T * h
  • Scaled Dot-Product: score(s, h) = s^T * h / sqrt(d_k)
  • Multi-Head: Multiple attention heads capture different aspects of relationships.

Self-attention (used in Transformers) computes attention of a sequence with itself, allowing every position to attend to every other position.

Key Properties

  • Dynamic Weighting: Attention weights change based on the current query.
  • Interpretable: Attention weights can reveal which inputs the model focuses on.
  • Soft Alignment: Unlike hard alignment, it considers all positions with different weights.
  • Foundation: The core mechanism behind Transformers, BERT, GPT, and modern AI.

Implementation

function attention(query, keys, values) {
    var dk = keys[0].length;
    var scores = keys.map(function(key) {
        return dotProduct(query, key) / Math.sqrt(dk);
    });

    // Softmax
    var maxScore = Math.max.apply(null, scores);
    var exps = scores.map(function(s) { return Math.exp(s - maxScore); });
    var sumExps = exps.reduce(function(a, b) { return a + b; }, 0);
    var weights = exps.map(function(e) { return e / sumExps; });

    // Weighted sum of values
    var output = new Array(values[0].length).fill(0);
    for (var i = 0; i < values.length; i++) {
        for (var j = 0; j < output.length; j++) {
            output[j] += weights[i] * values[i][j];
        }
    }
    return { output: output, weights: weights };
}
