Long Short-Term Memory (LSTM)

Hard Time: O(T·h²) Params: 4× vanilla RNN

How It Works

LSTM (Long Short-Term Memory) solves the vanishing gradient problem by introducing a cell state — a highway that carries information across time steps with minimal modification. Three gates control information flow:

  • Forget Gate (f): Decides what to discard from the cell state. f = sigmoid(W_f * [h_{t-1}, x_t] + b_f)
  • Input Gate (i): Decides what new information to store. i = sigmoid(W_i * [h_{t-1}, x_t] + b_i)
  • Output Gate (o): Decides what to output. o = sigmoid(W_o * [h_{t-1}, x_t] + b_o)

The cell state update is: C_t = f * C_{t-1} + i * tanh(W_c * [h_{t-1}, x_t] + b_c). The hidden state is: h_t = o * tanh(C_t). This gating mechanism allows gradients to flow unchanged through the cell state, enabling learning of long-range dependencies.

Key Properties

  • Cell State: Unmodified gradient highway solves vanishing gradients.
  • Three Gates: Forget, Input, Output gates control information flow.
  • Long-Range Memory: Can remember information for hundreds of time steps.
  • 4x Parameters: Has 4 weight matrices per layer vs 1 in vanilla RNN.

Implementation

function LSTMCell(inputSize, hiddenSize) {
    // Initialize 4 gate weight matrices
    var Wf = randomMatrix(hiddenSize, inputSize + hiddenSize); // forget
    var Wi = randomMatrix(hiddenSize, inputSize + hiddenSize); // input
    var Wc = randomMatrix(hiddenSize, inputSize + hiddenSize); // candidate
    var Wo = randomMatrix(hiddenSize, inputSize + hiddenSize); // output

    this.forward = function(x, hPrev, cPrev) {
        var concat = hPrev.concat(x);
        var f = sigmoid(matVec(Wf, concat));  // forget gate
        var i = sigmoid(matVec(Wi, concat));  // input gate
        var cHat = tanh(matVec(Wc, concat));  // candidate
        var o = sigmoid(matVec(Wo, concat));  // output gate

        // Cell state update
        var c = add(multiply(f, cPrev), multiply(i, cHat));
        // Hidden state
        var h = multiply(o, tanh(c));
        return { h: h, c: c };
    };
}
