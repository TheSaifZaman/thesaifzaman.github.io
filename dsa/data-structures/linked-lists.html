Linked Lists

Easy Time: O(n) access, O(1) insert/delete Space: O(n)

How It Works

A linked list is a linear data structure where each element (node) contains a value and a pointer (reference) to the next node. Unlike arrays, linked list nodes are not stored in contiguous memory — each node can exist anywhere in the heap. This means you cannot access the i-th element directly; you must traverse from the head, following pointers one by one.

The key advantage of linked lists is O(1) insertion and deletion at a known position. If you have a reference to a node, you can insert a new node after it or remove it by simply rewiring pointers — no shifting of elements is needed. This makes linked lists ideal for implementing queues, stacks, and LRU caches.

There are several variants: singly linked lists (each node points to next), doubly linked lists (each node has next and prev pointers), and circular linked lists (the tail points back to the head). Many classic interview problems involve pointer manipulation techniques like the fast/slow pointer approach for cycle detection and finding the middle node.

Pointer-Based

Nodes connected via references; no contiguous memory required.

Implementation

class ListNode {
  constructor(val) {
    this.val = val;
    this.next = null;
  }
}

class LinkedList {
  constructor() {
    this.head = null;
    this.size = 0;
  }

  // O(1) — prepend
  addFirst(val) {
    const node = new ListNode(val);
    node.next = this.head;
    this.head = node;
    this.size++;
  }

  // O(n) — append
  addLast(val) {
    const node = new ListNode(val);
    if (!this.head) { this.head = node; }
    else {
      let curr = this.head;
      while (curr.next) curr = curr.next;
      curr.next = node;
    }
    this.size++;
  }

  // O(1) — remove head
  removeFirst() {
    if (!this.head) return null;
    const val = this.head.val;
    this.head = this.head.next;
    this.size--;
    return val;
  }

  // O(n) — reverse in place
  reverse() {
    let prev = null, curr = this.head;
    while (curr) {
      const next = curr.next;
      curr.next = prev;
      prev = curr;
      curr = next;
    }
    this.head = prev;
  }

  // O(n) — detect cycle (Floyd's)
  hasCycle() {
    let slow = this.head, fast = this.head;
    while (fast && fast.next) {
      slow = slow.next;
      fast = fast.next.next;
      if (slow === fast) return true;
    }
    return false;
  }
}
