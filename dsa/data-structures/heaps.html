Heaps / Priority Queues

Medium Time: O(log n) insert/extract Space: O(n)

How It Works

A heap is a complete binary tree that satisfies the heap property: in a min-heap, every parent node is smaller than or equal to its children; in a max-heap, every parent is greater than or equal to its children. The root always contains the minimum (or maximum) element, enabling O(1) access to the extreme value.

Heaps are typically stored as arrays. For a node at index i, its left child is at 2i+1, its right child at 2i+2, and its parent at floor((i-1)/2). Insertion appends the element and "bubbles up" (sift up) to restore the heap property in O(log n). Extraction removes the root, replaces it with the last element, and "sinks down" (sift down) in O(log n).

The priority queue abstract data type is almost always implemented as a heap. It is essential for algorithms like Dijkstra's shortest path, Prim's MST, Huffman coding, and the "top K elements" pattern. Building a heap from n elements can be done in O(n) time using the bottom-up heapify approach, which is faster than inserting elements one by one.

Heap Property

Parent is always smaller (min-heap) or larger (max-heap) than children.

Array-Based Tree

Complete binary tree stored in an array with index arithmetic.

Bubble Up / Sink Down

Restore heap property after insert (up) or extract (down) in O(log n).

Top K Pattern

Maintain a heap of size K to find K largest/smallest in O(n log k).

Implementation

class MinHeap {
  constructor() {
    this.heap = [];
  }

  // O(log n)
  insert(val) {
    this.heap.push(val);
    this._bubbleUp(this.heap.length - 1);
  }

  // O(log n)
  extractMin() {
    if (!this.heap.length) return undefined;
    const min = this.heap[0];
    const last = this.heap.pop();
    if (this.heap.length) {
      this.heap[0] = last;
      this._sinkDown(0);
    }
    return min;
  }

  // O(1)
  peek() {
    return this.heap[0];
  }

  _bubbleUp(i) {
    while (i > 0) {
      const parent = Math.floor((i - 1) / 2);
      if (this.heap[i] >= this.heap[parent]) break;
      [this.heap[i], this.heap[parent]] = [this.heap[parent], this.heap[i]];
      i = parent;
    }
  }

  _sinkDown(i) {
    const n = this.heap.length;
    while (true) {
      let smallest = i;
      const left = 2 * i + 1;
      const right = 2 * i + 2;
      if (left < n && this.heap[left] < this.heap[smallest]) smallest = left;
      if (right < n && this.heap[right] < this.heap[smallest]) smallest = right;
      if (smallest === i) break;
      [this.heap[i], this.heap[smallest]] = [this.heap[smallest], this.heap[i]];
      i = smallest;
    }
  }

  get size() {
    return this.heap.length;
  }
}
