Stacks

Easy Time: O(1) push/pop/peek Space: O(n)

How It Works

A stack is a Last-In-First-Out (LIFO) data structure. Think of a stack of plates: you can only add a new plate on top (push) and remove the topmost plate (pop). The most recently added element is always the first one to be removed. All three core operations — push, pop, and peek — run in O(1) time.

Stacks are fundamental to how computers work. Function call management uses a call stack; expression evaluation uses operand and operator stacks; backtracking algorithms use stacks to remember previous states. In interview problems, stacks are commonly used for matching parentheses, evaluating postfix expressions, and maintaining a monotonic sequence.

A stack can be implemented using an array (where the end acts as the top) or a linked list (where the head is the top). The array-based implementation is more cache-friendly, while the linked-list implementation avoids resizing concerns and has guaranteed O(1) push/pop without amortization.

LIFO Order

Last element pushed is the first to be popped.

O(1) Operations

Push, pop, and peek all execute in constant time.

Expression Parsing

Stacks evaluate infix, prefix, and postfix expressions.

Undo/Backtrack

Ideal for undo operations, DFS, and recursive call simulation.

Implementation

class Stack {
  constructor() {
    this.items = [];
  }

  // O(1)
  push(val) {
    this.items.push(val);
  }

  // O(1)
  pop() {
    if (this.isEmpty()) return undefined;
    return this.items.pop();
  }

  // O(1)
  peek() {
    return this.items[this.items.length - 1];
  }

  isEmpty() {
    return this.items.length === 0;
  }

  get size() {
    return this.items.length;
  }
}

// Classic problem: Valid Parentheses
function isValid(s) {
  const stack = new Stack();
  const map = { ')': '(', ']': '[', '}': '{' };

  for (const ch of s) {
    if ('({['.includes(ch)) {
      stack.push(ch);
    } else {
      if (stack.isEmpty() || stack.pop() !== map[ch]) {
        return false;
      }
    }
  }
  return stack.isEmpty();
}
