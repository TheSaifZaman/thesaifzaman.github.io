Hash Tables

Medium Time: O(1) avg lookup/insert Space: O(n)

How It Works

A hash table (also called a hash map or dictionary) stores key-value pairs and provides average O(1) time for insertions, deletions, and lookups. It works by applying a hash function to each key, producing a numeric index into an underlying array of "buckets." The element is then stored in (or retrieved from) that bucket.

When two different keys produce the same hash index, a collision occurs. The two most common collision resolution strategies are chaining (each bucket holds a linked list of entries) and open addressing (probing for the next empty slot). A good hash function distributes keys uniformly across buckets, minimizing collisions and keeping operations close to O(1).

Hash tables are among the most frequently used data structures in real-world software and interviews. They underpin JavaScript objects, Python dictionaries, and Java HashMaps. Common interview patterns include frequency counting, two-sum lookups, grouping anagrams, and detecting duplicates — all leveraging the hash table's constant-time access.

Hash Function

Maps keys to array indices; good distribution minimizes collisions.

O(1) Average

Insert, delete, and lookup run in amortized constant time.

Collision Handling

Chaining (linked lists) or open addressing (linear/quadratic probing).

Load Factor

Resize when load factor exceeds threshold to maintain performance.

Implementation

class HashTable {
  constructor(size = 16) {
    this.buckets = new Array(size).fill(null).map(() => []);
    this.size = size;
    this.count = 0;
  }

  _hash(key) {
    let hash = 0;
    const str = String(key);
    for (let i = 0; i < str.length; i++) {
      hash = (hash * 31 + str.charCodeAt(i)) % this.size;
    }
    return hash;
  }

  // O(1) average
  set(key, value) {
    const idx = this._hash(key);
    const bucket = this.buckets[idx];
    for (let i = 0; i < bucket.length; i++) {
      if (bucket[i][0] === key) {
        bucket[i][1] = value;
        return;
      }
    }
    bucket.push([key, value]);
    this.count++;
    if (this.count / this.size > 0.75) this._resize();
  }

  // O(1) average
  get(key) {
    const idx = this._hash(key);
    const bucket = this.buckets[idx];
    for (const [k, v] of bucket) {
      if (k === key) return v;
    }
    return undefined;
  }

  // O(1) average
  delete(key) {
    const idx = this._hash(key);
    const bucket = this.buckets[idx];
    for (let i = 0; i < bucket.length; i++) {
      if (bucket[i][0] === key) {
        bucket.splice(i, 1);
        this.count--;
        return true;
      }
    }
    return false;
  }

  _resize() {
    const old = this.buckets;
    this.size *= 2;
    this.buckets = new Array(this.size).fill(null).map(() => []);
    this.count = 0;
    for (const bucket of old) {
      for (const [k, v] of bucket) {
        this.set(k, v);
      }
    }
  }
}
