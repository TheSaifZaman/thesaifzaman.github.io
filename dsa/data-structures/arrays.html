Arrays

Easy Time: O(1) access, O(n) search Space: O(n)

How It Works

An array is a contiguous block of memory that stores elements of the same type sequentially. Each element is identified by its numerical index, starting from zero. Because elements are stored in adjacent memory locations, accessing any element by index takes constant time O(1) — the memory address is calculated directly as base + index * elementSize.

Insertion and deletion at the end of a dynamic array are amortized O(1), but inserting or removing elements in the middle requires shifting subsequent elements, making those operations O(n). This trade-off between fast random access and costly middle insertions makes arrays ideal for scenarios where you need frequent lookups and append operations.

Arrays form the foundation for many other data structures and algorithms. Techniques like the two-pointer approach, sliding window, prefix sums, and binary search all exploit the array's O(1) index access. Understanding arrays deeply is essential for solving a wide range of interview problems efficiently.

O(1) Random Access

Direct index calculation enables instant element retrieval.

Contiguous Memory

Cache-friendly layout leads to fast sequential traversal.

Dynamic Resizing

Dynamic arrays double capacity when full, giving amortized O(1) appends.

In-place Operations

Many problems can be solved with O(1) extra space using swaps and pointers.

Implementation

// Dynamic Array (simplified)
class DynamicArray {
  constructor() {
    this.data = [];
    this.length = 0;
  }

  // O(1)
  get(index) {
    if (index < 0 || index >= this.length) return undefined;
    return this.data[index];
  }

  // Amortized O(1)
  push(value) {
    this.data[this.length] = value;
    this.length++;
  }

  // O(1)
  pop() {
    if (this.length === 0) return undefined;
    this.length--;
    const val = this.data[this.length];
    delete this.data[this.length];
    return val;
  }

  // O(n) — shift elements right
  insertAt(index, value) {
    for (let i = this.length; i > index; i--) {
      this.data[i] = this.data[i - 1];
    }
    this.data[index] = value;
    this.length++;
  }

  // O(n) — shift elements left
  deleteAt(index) {
    const val = this.data[index];
    for (let i = index; i < this.length - 1; i++) {
      this.data[i] = this.data[i + 1];
    }
    delete this.data[this.length - 1];
    this.length--;
    return val;
  }

  // O(n) — linear search
  indexOf(value) {
    for (let i = 0; i < this.length; i++) {
      if (this.data[i] === value) return i;
    }
    return -1;
  }
}
