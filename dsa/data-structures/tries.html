Tries

Medium Time: O(m) per operation (m = key length) Space: O(n * m)

How It Works

A trie (pronounced "try") is a tree-like data structure used for efficient storage and retrieval of strings. Each node represents a single character, and paths from the root to marked nodes form complete words. Unlike a hash table, a trie allows prefix-based operations: you can find all words starting with a given prefix in O(p + k) time, where p is the prefix length and k is the number of matching words.

Each node in a trie typically has an array or map of children (one per possible character) and a boolean flag indicating whether the path from root to this node forms a complete word. Insertion and lookup both take O(m) time, where m is the length of the string — independent of how many strings are stored. This makes tries excellent for autocomplete systems, spell checkers, and IP routing tables.

The main trade-off is space: a trie can use significant memory since each node may have up to 26 (or more) child pointers. Compressed tries (radix trees) mitigate this by merging single-child chains into one node. In interviews, tries appear in problems like implementing autocomplete, word search in grids, and finding the longest common prefix.

Prefix Search

Find all words with a given prefix efficiently — ideal for autocomplete.

O(m) Operations

Insert and search depend only on string length, not dataset size.

Shared Prefixes

Common prefixes are stored once, saving space for similar strings.

Applications

Autocomplete, spell check, IP routing, and word games.

Implementation

class TrieNode {
  constructor() {
    this.children = {};
    this.isEnd = false;
  }
}

class Trie {
  constructor() {
    this.root = new TrieNode();
  }

  // O(m)
  insert(word) {
    let node = this.root;
    for (const ch of word) {
      if (!node.children[ch]) {
        node.children[ch] = new TrieNode();
      }
      node = node.children[ch];
    }
    node.isEnd = true;
  }

  // O(m)
  search(word) {
    const node = this._traverse(word);
    return node !== null && node.isEnd;
  }

  // O(m)
  startsWith(prefix) {
    return this._traverse(prefix) !== null;
  }

  _traverse(str) {
    let node = this.root;
    for (const ch of str) {
      if (!node.children[ch]) return null;
      node = node.children[ch];
    }
    return node;
  }

  // Collect all words with given prefix
  autocomplete(prefix) {
    const node = this._traverse(prefix);
    if (!node) return [];
    const results = [];
    this._dfs(node, prefix, results);
    return results;
  }

  _dfs(node, path, results) {
    if (node.isEnd) results.push(path);
    for (const ch in node.children) {
      this._dfs(node.children[ch], path + ch, results);
    }
  }
}
