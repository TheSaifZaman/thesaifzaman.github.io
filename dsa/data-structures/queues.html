Queues

Easy Time: O(1) enqueue/dequeue Space: O(n)

How It Works

A queue is a First-In-First-Out (FIFO) data structure. Elements are added at the back (enqueue) and removed from the front (dequeue), just like a line of people waiting. The element that has been in the queue the longest is always the first to be served. Both enqueue and dequeue operations run in O(1) time when implemented with a linked list or circular buffer.

Queues are essential for Breadth-First Search (BFS), level-order tree traversal, task scheduling, and buffering. In BFS, each node's neighbors are enqueued and processed in the order they were discovered, guaranteeing shortest-path exploration in unweighted graphs.

Common variants include the double-ended queue (deque), which allows insertion and deletion at both ends, and the priority queue, which dequeues elements based on priority rather than arrival order. In JavaScript, queues are often implemented with arrays, but for performance-critical code, a linked-list-based queue avoids the O(n) shift operation of Array.shift().

Implementation

class QueueNode {
  constructor(val) {
    this.val = val;
    this.next = null;
  }
}

class Queue {
  constructor() {
    this.head = null;
    this.tail = null;
    this.length = 0;
  }

  // O(1) — add to back
  enqueue(val) {
    const node = new QueueNode(val);
    if (this.tail) {
      this.tail.next = node;
    } else {
      this.head = node;
    }
    this.tail = node;
    this.length++;
  }

  // O(1) — remove from front
  dequeue() {
    if (!this.head) return undefined;
    const val = this.head.val;
    this.head = this.head.next;
    if (!this.head) this.tail = null;
    this.length--;
    return val;
  }

  // O(1)
  peek() {
    return this.head ? this.head.val : undefined;
  }

  isEmpty() {
    return this.length === 0;
  }
}

// BFS using a queue
function bfs(graph, start) {
  const visited = new Set([start]);
  const queue = new Queue();
  queue.enqueue(start);

  while (!queue.isEmpty()) {
    const node = queue.dequeue();
    for (const neighbor of graph[node]) {
      if (!visited.has(neighbor)) {
        visited.add(neighbor);
        queue.enqueue(neighbor);
      }
    }
  }
  return visited;
}
