Matrix / Grid

Easy Time: O(m x n) traversal Space: O(m x n)

How It Works

A matrix (or grid) is a 2D array organized in rows and columns. Each cell is accessed by its row index and column index: matrix[row][col]. Matrices are used to represent game boards, images, graphs (adjacency matrices), and many mathematical concepts. Traversal runs in O(m x n) where m is the number of rows and n is the number of columns.

Common matrix patterns include row-by-row traversal, column-by-column traversal, diagonal traversal, and spiral order traversal. The spiral pattern moves right along the top row, down the right column, left along the bottom row, and up the left column, progressively shrinking the boundaries. This technique is useful for problems like spiral matrix printing and rotating images.

Matrix problems often combine with BFS/DFS for flood fill, island counting, and shortest path in grids. In-place modifications (like rotating a matrix 90 degrees) require careful understanding of index transformations. For sorted matrices, binary search can find elements in O(log(m*n)) or O(m + n) time depending on the search strategy.

2D Indexing

Access any cell in O(1) using row and column indices.

Traversal Patterns

Row-wise, column-wise, diagonal, spiral, and boundary traversals.

In-Place Rotation

Transpose + reverse to rotate 90 degrees without extra space.

BFS/DFS on Grids

Flood fill, island counting, and shortest path via 4-directional moves.

// Spiral Order Traversal
function spiralOrder(matrix) {
  const result = [];
  if (!matrix.length) return result;

  let top = 0, bottom = matrix.length - 1;
  let left = 0, right = matrix[0].length - 1;

  while (top <= bottom && left <= right) {
    // Traverse right
    for (let i = left; i <= right; i++)
      result.push(matrix[top][i]);
    top++;

    // Traverse down
    for (let i = top; i <= bottom; i++)
      result.push(matrix[i][right]);
    right--;

    // Traverse left
    if (top <= bottom) {
      for (let i = right; i >= left; i--)
        result.push(matrix[bottom][i]);
      bottom--;
    }

    // Traverse up
    if (left <= right) {
      for (let i = bottom; i >= top; i--)
        result.push(matrix[i][left]);
      left++;
    }
  }
  return result;
}

// Rotate 90 degrees clockwise (in-place)
function rotate(matrix) {
  const n = matrix.length;
  // Transpose
  for (let i = 0; i < n; i++) {
    for (let j = i + 1; j < n; j++) {
      [matrix[i][j], matrix[j][i]] = [matrix[j][i], matrix[i][j]];
    }
  }
  // Reverse each row
  for (let i = 0; i < n; i++) {
    matrix[i].reverse();
  }
}

// Search in row-wise & column-wise sorted matrix
function searchMatrix(matrix, target) {
  let row = 0, col = matrix[0].length - 1;
  while (row < matrix.length && col >= 0) {
    if (matrix[row][col] === target) return true;
    if (matrix[row][col] > target) col--;
    else row++;
  }
  return false;
}
