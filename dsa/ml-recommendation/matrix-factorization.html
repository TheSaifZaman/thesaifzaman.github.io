Matrix Factorization

Hard Time: O(n·m·k) Space: O((n+m)·k)

How It Works

Matrix Factorization decomposes the sparse user-item rating matrix R (n users × m items) into two lower-rank matrices: U (n × k) and V (m × k), where k is the number of latent factors. The predicted rating is r̂ui = Uu · ViT.

The latent factors capture hidden patterns (e.g., genre preference, movie quality). Training minimizes the reconstruction error on known ratings using SGD or ALS, with regularization to prevent overfitting. This was the winning approach in the Netflix Prize competition.

Key Properties

  • Latent factors: k dimensions capturing user preferences and item attributes.
  • Loss: Σ(rui − Uu·ViT)² + λ(||U||² + ||V||²).
  • SGD update: Uu += α(e·Vi − λ·Uu).
  • Handles sparsity: Only trains on observed ratings; generalizes to unseen pairs.

Implementation

function matrixFactorization(R, k, alpha, lambda, iters) {
    var n = R.length, m = R[0].length;
    var U = randomMatrix(n, k);
    var V = randomMatrix(m, k);
    for (var iter = 0; iter < iters; iter++) {
        for (var u = 0; u < n; u++) {
            for (var i = 0; i < m; i++) {
                if (R[u][i] === null) continue;
                var pred = dotProduct(U[u], V[i]);
                var err = R[u][i] - pred;
                for (var f = 0; f < k; f++) {
                    U[u][f] += alpha * (err * V[i][f] - lambda * U[u][f]);
                    V[i][f] += alpha * (err * U[u][f] - lambda * V[i][f]);
                }
            }
        }
    }
    return { U: U, V: V };
}
