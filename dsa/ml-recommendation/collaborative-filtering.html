Collaborative Filtering

Medium Time: O(n·m) Space: O(n·m)

How It Works

Collaborative filtering recommends items based on the collective preferences of similar users. User-based CF finds users with similar rating patterns and recommends items they liked. Item-based CF finds items similar to those the target user already rated highly.

The user-item matrix stores ratings, with missing values representing items not yet rated. The algorithm fills in these missing values by leveraging similarity (cosine, Pearson correlation, or Jaccard) between users or items.

Key Properties

  • User-based: "Users who are similar to you liked X."
  • Item-based: "Items similar to what you liked include X."
  • Cold start: Struggles with new users or items (no history).
  • Scalability: Item-based is more scalable (items change less often).

Implementation

function userBasedCF(matrix, targetUser, targetItem) {
    var similarities = [];
    for (var u = 0; u < matrix.length; u++) {
        if (u === targetUser) continue;
        if (matrix[u][targetItem] === null) continue;
        var sim = cosineSimilarity(matrix[targetUser], matrix[u]);
        similarities.push({user: u, sim: sim, rating: matrix[u][targetItem]});
    }
    similarities.sort(function(a, b) { return b.sim - a.sim; });
    var topK = similarities.slice(0, 3);
    var num = 0, den = 0;
    for (var i = 0; i < topK.length; i++) {
        num += topK[i].sim * topK[i].rating;
        den += Math.abs(topK[i].sim);
    }
    return den > 0 ? num / den : 0;
}
