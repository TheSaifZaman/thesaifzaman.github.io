Simulated Annealing

Medium Time: O(n·T) Space: O(d)

How It Works

Simulated Annealing (SA) is a probabilistic global optimization technique inspired by the metallurgical process of annealing — heating metal and slowly cooling it to remove defects. In optimization, it allows "uphill" moves (accepting worse solutions) with a probability that decreases over time (as temperature cools).

At high temperature, the algorithm freely explores the search space, accepting many worse solutions to escape local minima. As temperature drops, it becomes more selective, eventually converging to a (hopefully global) minimum.

The acceptance probability for a worse solution is: P = exp(-ΔE / T), where ΔE is the energy increase and T is the current temperature.

Temperature

Controls exploration: high T = random, low T = greedy.

Escapes Local Minima

Accepts worse solutions probabilistically at high temperature.

Cooling Schedule

Geometric or linear cooling determines convergence speed.

Global Optimum

Provably converges to global minimum with slow enough cooling.

Implementation

function simulatedAnnealing(costFn, initial, T0, coolingRate, iterations) {
    var current = initial.slice();
    var best = current.slice();
    var currentCost = costFn(current);
    var bestCost = currentCost;
    var T = T0;

    for (var i = 0; i < iterations; i++) {
        // Generate neighbor
        var neighbor = perturb(current);
        var neighborCost = costFn(neighbor);
        var delta = neighborCost - currentCost;

        // Accept if better, or with probability exp(-delta/T)
        if (delta < 0 || Math.random() < Math.exp(-delta / T)) {
            current = neighbor;
            currentCost = neighborCost;
        }

        // Track best solution found
        if (currentCost < bestCost) {
            best = current.slice();
            bestCost = currentCost;
        }

        // Cool down
        T *= coolingRate;
    }
    return { solution: best, cost: bestCost };
}
