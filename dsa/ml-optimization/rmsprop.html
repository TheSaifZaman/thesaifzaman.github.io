RMSProp

Medium Time: O(d) per update Space: O(d)

How It Works

RMSProp (Root Mean Square Propagation) adapts the learning rate for each parameter by dividing by the running average of recent gradient magnitudes. This solves the problem of Adagrad's ever-decreasing learning rate.

Update rule: v = β · v + (1-β) · g², then θ -= α · g / (√v + ε)

Parameters with consistently large gradients get smaller effective learning rates (preventing overshooting), while parameters with small gradients get larger effective rates (accelerating slow dimensions). This is particularly useful for loss surfaces with different curvatures in different directions.

Adaptive Per-Parameter

Each parameter gets a learning rate scaled by its gradient history.

Running Average

Exponential moving average prevents vanishing learning rates.

Handles Curvature

Works well with elongated loss surfaces (different scales per dim).

RNN Favorite

Particularly effective for training recurrent neural networks.

Implementation

function rmsprop(gradFn, theta, lr, beta, epsilon, iterations) {
    var v = new Array(theta.length).fill(0);
    beta = beta || 0.9;
    epsilon = epsilon || 1e-8;

    for (var t = 0; t < iterations; t++) {
        var grad = gradFn(theta);
        for (var i = 0; i < theta.length; i++) {
            // Running average of squared gradients
            v[i] = beta * v[i] + (1 - beta) * grad[i] * grad[i];
            // Adaptive update
            theta[i] -= lr * grad[i] / (Math.sqrt(v[i]) + epsilon);
        }
    }
    return theta;
}
