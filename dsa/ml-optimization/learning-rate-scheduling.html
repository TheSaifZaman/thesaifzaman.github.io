Learning Rate Scheduling

Easy Time: O(1) per step Space: O(1)

How It Works

Learning Rate Scheduling dynamically adjusts the learning rate during training. Starting with a larger learning rate helps the optimizer explore broadly, while reducing it over time enables fine convergence near the minimum.

Common schedules include: Step decay (reduce by factor every N epochs), Exponential decay (α = α_0 · e^{-kt}), Cosine annealing (smooth cosine curve from high to low), and Warmup (start small, ramp up, then decay — used in Transformers).

The visualization shows how different schedules change the learning rate over training epochs, and their effect on convergence behavior.

Step Decay

Drop LR by a factor (e.g., 0.1) every N epochs.

Cosine Annealing

Smooth cosine curve from initial LR to near-zero.

Warmup

Gradually increase LR at start to stabilize early training.

Cosine Restart

Periodically reset LR to escape local minima.

Implementation

// Step Decay
function stepDecay(initialLR, epoch, dropEvery, dropFactor) {
    return initialLR * Math.pow(dropFactor, Math.floor(epoch / dropEvery));
}

// Exponential Decay
function exponentialDecay(initialLR, epoch, decayRate) {
    return initialLR * Math.exp(-decayRate * epoch);
}

// Cosine Annealing
function cosineAnnealing(initialLR, epoch, totalEpochs) {
    return initialLR * 0.5 * (1 + Math.cos(Math.PI * epoch / totalEpochs));
}

// Warmup + Cosine
function warmupCosine(initialLR, epoch, warmupEpochs, totalEpochs) {
    if (epoch < warmupEpochs) {
        return initialLR * (epoch / warmupEpochs); // linear warmup
    }
    var progress = (epoch - warmupEpochs) / (totalEpochs - warmupEpochs);
    return initialLR * 0.5 * (1 + Math.cos(Math.PI * progress));
}
