Momentum

Easy Time: O(d) per update Space: O(d)

How It Works

Momentum adds a "velocity" term to gradient descent, accumulating past gradients to push through flat regions and dampen oscillations. Like a ball rolling downhill, it builds up speed in consistent gradient directions and reduces jitter in oscillating directions.

Update rule: v = β · v + α · ∇L(θ), then θ -= v

The momentum coefficient β (typically 0.9) controls how much past velocity is retained. With β=0, it reduces to standard gradient descent. Higher β means more inertia — faster traversal of plateaus but risk of overshooting.

Ball Rolling

Accumulates velocity like a ball rolling downhill.

Accelerated

Pushes through flat regions and saddle points faster.

Dampens Oscillation

Cancels out oscillations in high-curvature directions.

Nesterov Variant

Look-ahead gradient for even better convergence.

Implementation

function momentumGD(gradFn, theta, lr, beta, iterations) {
    var velocity = new Array(theta.length).fill(0);
    beta = beta || 0.9;

    for (var t = 0; t < iterations; t++) {
        var grad = gradFn(theta);
        for (var i = 0; i < theta.length; i++) {
            velocity[i] = beta * velocity[i] + lr * grad[i];
            theta[i] -= velocity[i];
        }
    }
    return theta;
}

// Nesterov Accelerated Gradient (NAG)
function nesterovMomentum(gradFn, theta, lr, beta, iterations) {
    var velocity = new Array(theta.length).fill(0);
    for (var t = 0; t < iterations; t++) {
        // Look ahead
        var lookAhead = theta.map(function(th, i) { return th - beta * velocity[i]; });
        var grad = gradFn(lookAhead);
        for (var i = 0; i < theta.length; i++) {
            velocity[i] = beta * velocity[i] + lr * grad[i];
            theta[i] -= velocity[i];
        }
    }
    return theta;
}
