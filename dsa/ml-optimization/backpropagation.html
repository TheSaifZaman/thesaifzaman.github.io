Backpropagation

Medium Time: O(n·d²) Space: O(n·d)

How It Works

Backpropagation is the algorithm that makes neural network training possible. It efficiently computes the gradient of the loss function with respect to every weight in the network by applying the chain rule of calculus backward from the output layer to the input layer.

The process has two phases: (1) Forward pass — compute output by propagating input through layers. (2) Backward pass — propagate error gradients backward, computing how much each weight contributed to the error.

At each layer, the gradient flows backward: δ_l = δ_{l+1} · W_{l+1}^T ⊙ f'(z_l), where f' is the activation function's derivative. This allows gradient descent to update every weight simultaneously.

Chain Rule

Decomposes complex gradients into products of local derivatives.

Backward Pass

Error flows from output to input, computing all gradients in one pass.

Efficient

Computes all gradients in O(n) — same cost as forward pass.

Vanishing Gradients

Deep networks can suffer from gradients shrinking to near-zero.

Implementation

function backpropagation(network, input, target, lr) {
    // FORWARD PASS
    var activations = [input];
    var zValues = [];
    for (var l = 0; l < network.layers.length; l++) {
        var z = matMul(activations[l], network.layers[l].weights);
        z = addBias(z, network.layers[l].bias);
        zValues.push(z);
        activations.push(activate(z, network.layers[l].activation));
    }

    // COMPUTE OUTPUT ERROR
    var output = activations[activations.length - 1];
    var loss = meanSquaredError(output, target);
    var delta = output.map(function(o, i) {
        return (o - target[i]) * activationDerivative(zValues[zValues.length - 1][i]);
    });

    // BACKWARD PASS — propagate gradients
    var deltas = [delta];
    for (var l = network.layers.length - 2; l >= 0; l--) {
        var d = matMul(deltas[0], transpose(network.layers[l + 1].weights));
        d = d.map(function(v, i) {
            return v * activationDerivative(zValues[l][i]);
        });
        deltas.unshift(d);
    }

    // UPDATE WEIGHTS
    for (var l = 0; l < network.layers.length; l++) {
        for (var i = 0; i < network.layers[l].weights.length; i++) {
            for (var j = 0; j < network.layers[l].weights[i].length; j++) {
                network.layers[l].weights[i][j] -= lr * activations[l][i] * deltas[l][j];
            }
        }
    }
    return loss;
}
