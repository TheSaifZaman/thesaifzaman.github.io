Genetic Algorithm

Medium Time: O(n·g) Space: O(n·d)

How It Works

Genetic Algorithms (GA) are optimization methods inspired by natural evolution. A population of candidate solutions evolves over generations through selection, crossover, and mutation. Fitter individuals are more likely to survive and reproduce, gradually improving the population.

Each generation: (1) Selection — pick parents based on fitness. (2) Crossover — combine two parents to create offspring. (3) Mutation — randomly modify some genes. (4) Replacement — new generation replaces the old.

GAs are useful for complex optimization problems where the search space is large, non-differentiable, or has many local optima. They require no gradient information.

Population

Maintains diverse set of candidate solutions simultaneously.

Crossover

Combines genes from two parents to create better offspring.

Mutation

Random changes maintain diversity and prevent premature convergence.

Selection

Fitter individuals have higher probability of reproduction.

Implementation

function geneticAlgorithm(fitnessFn, popSize, geneLength, generations, mutRate) {
    // Initialize random population
    var population = [];
    for (var i = 0; i < popSize; i++) {
        population.push(randomChromosome(geneLength));
    }

    for (var gen = 0; gen < generations; gen++) {
        // Evaluate fitness
        var fitness = population.map(fitnessFn);

        // Selection (tournament)
        var newPop = [];
        while (newPop.length < popSize) {
            var p1 = tournamentSelect(population, fitness);
            var p2 = tournamentSelect(population, fitness);

            // Crossover
            var child = crossover(p1, p2);

            // Mutation
            for (var g = 0; g < child.length; g++) {
                if (Math.random() < mutRate) {
                    child[g] = randomGene();
                }
            }
            newPop.push(child);
        }
        population = newPop;
    }

    // Return best individual
    var finalFitness = population.map(fitnessFn);
    var bestIdx = argMax(finalFitness);
    return population[bestIdx];
}
