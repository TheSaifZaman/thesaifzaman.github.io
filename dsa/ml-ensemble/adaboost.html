AdaBoost

Medium Training: O(n·T) Prediction: O(T)

How It Works

AdaBoost (Adaptive Boosting) was one of the first successful boosting algorithms. It combines multiple weak classifiers (typically decision stumps) into a single strong classifier by adaptively re-weighting training samples.

In each round, samples that were misclassified by the previous weak learner receive higher weights. The next weak learner then focuses more on these harder samples. Each weak learner also gets a weight (alpha) based on its accuracy — more accurate classifiers have greater influence on the final prediction.

The final prediction is a weighted majority vote of all weak classifiers. AdaBoost can achieve arbitrarily low training error given enough rounds, and has strong theoretical guarantees via the PAC learning framework.

Key Properties

  • Adaptive Weighting: Sample weights change based on classification errors.
  • Weak Learner Requirement: Each model need only be slightly better than random (>50%).
  • Exponential Loss: Minimizes exponential loss function implicitly.
  • Sensitive to Noise: Outliers get increasingly high weights, which can hurt performance.

function adaBoost(X, y, numRounds) {
    var n = X.length;
    var weights = new Array(n).fill(1 / n);
    var alphas = [], stumps = [];

    for (var t = 0; t < numRounds; t++) {
        var stump = trainWeightedStump(X, y, weights);
        var error = 0;
        for (var i = 0; i < n; i++) {
            if (stump.predict(X[i]) !== y[i]) error += weights[i];
        }
        error = Math.max(error, 1e-10);
        var alpha = 0.5 * Math.log((1 - error) / error);

        // Update weights
        for (var i = 0; i < n; i++) {
            var correct = stump.predict(X[i]) === y[i] ? 1 : -1;
            weights[i] *= Math.exp(-alpha * correct);
        }
        // Normalize
        var sum = weights.reduce(function(a, b) { return a + b; }, 0);
        weights = weights.map(function(w) { return w / sum; });

        alphas.push(alpha);
        stumps.push(stump);
    }
    return {
        predict: function(x) {
            var score = 0;
            for (var t = 0; t < stumps.length; t++) {
                score += alphas[t] * (stumps[t].predict(x) === 1 ? 1 : -1);
            }
            return score >= 0 ? 1 : 0;
        }
    };
}
