LightGBM

Hard Training: O(n·d) Space: O(n + bins)

How It Works

LightGBM (Light Gradient Boosting Machine) by Microsoft is a highly efficient gradient boosting framework that uses two key innovations: leaf-wise tree growth and histogram-based split finding.

Unlike traditional GBM frameworks that grow trees level-wise (all nodes at the same depth), LightGBM grows trees leaf-wise: it always splits the leaf with the highest loss reduction. This produces deeper, more asymmetric trees that achieve lower loss with fewer splits.

LightGBM also uses Gradient-based One-Side Sampling (GOSS) to keep samples with large gradients while randomly sampling from small-gradient samples, and Exclusive Feature Bundling (EFB) to reduce the number of features by bundling mutually exclusive features.

Key Properties

  • Leaf-Wise Growth: Splits the leaf with maximum gain, leading to deeper but more accurate trees.
  • Histogram Binning: Discretizes features into bins for O(n) split finding.
  • GOSS: Gradient-based sampling keeps informative samples, reduces data size.
  • EFB: Exclusive Feature Bundling reduces dimensionality without information loss.

// LightGBM leaf-wise growth concept
function lightGBMGrow(leaves, maxLeaves) {
    while (leaves.length < maxLeaves) {
        // Find the leaf with highest potential gain
        var bestLeaf = null, bestGain = -Infinity;
        leaves.forEach(function(leaf) {
            if (leaf.gain > bestGain) {
                bestGain = leaf.gain;
                bestLeaf = leaf;
            }
        });
        if (!bestLeaf || bestGain <= 0) break;

        // Split the best leaf
        var split = findBestHistogramSplit(bestLeaf);
        var leftChild = { data: split.left, gain: computeGain(split.left) };
        var rightChild = { data: split.right, gain: computeGain(split.right) };

        // Replace the leaf with its children
        var idx = leaves.indexOf(bestLeaf);
        leaves.splice(idx, 1, leftChild, rightChild);
    }
    return leaves;
}
