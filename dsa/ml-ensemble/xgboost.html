XGBoost

Hard Training: O(n·d·T·log n) Prediction: O(T·d)

How It Works

XGBoost (eXtreme Gradient Boosting) is an optimized gradient boosting framework that uses second-order Taylor expansion of the loss function to make more precise updates. It adds L1 and L2 regularization on leaf weights to prevent overfitting.

At each iteration, XGBoost builds a new tree that fits the negative gradient (residuals) of the loss function. The key innovation is using both the first derivative (gradient) and second derivative (hessian) of the loss to determine optimal leaf values and splits. This leads to more accurate split decisions and better regularization.

XGBoost also introduces several engineering optimizations: column block structure for parallel split finding, cache-aware access patterns, sparsity-aware split finding, and out-of-core computation for large datasets.

Key Properties

  • Second-Order Optimization: Uses both gradient and hessian for better convergence.
  • Regularization: L1 (alpha) and L2 (lambda) on leaf weights reduce overfitting.
  • Column Subsampling: Reduces correlation between trees similar to Random Forest.
  • Pruning: Uses max_depth and gamma for tree pruning instead of post-pruning.

// XGBoost conceptual pseudocode
function xgboost(X, y, numRounds, lr, lambda, gamma) {
    var predictions = new Array(X.length).fill(0);
    var trees = [];

    for (var t = 0; t < numRounds; t++) {
        // Compute gradients and hessians
        var gradients = [], hessians = [];
        for (var i = 0; i < X.length; i++) {
            gradients.push(predictions[i] - y[i]);      // g_i
            hessians.push(1.0);                          // h_i (for MSE)
        }
        // Build tree using gain formula:
        // Gain = 0.5 * [(GL^2/(HL+lambda)) + (GR^2/(HR+lambda))
        //              - (G^2/(H+lambda))] - gamma
        var tree = buildXGBTree(X, gradients, hessians, lambda, gamma);
        trees.push(tree);

        // Update predictions
        for (var i = 0; i < X.length; i++) {
            predictions[i] += lr * tree.predict(X[i]);
        }
    }
    return trees;
}
