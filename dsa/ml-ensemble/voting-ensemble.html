Voting Ensemble

Easy Training: O(k·C) Prediction: O(k)

How It Works

A Voting Ensemble combines predictions from multiple models using a simple voting scheme. It is the most straightforward ensemble method: train several diverse models independently, then aggregate their predictions.

There are two main voting strategies:

  • Hard Voting: Each model casts one vote for its predicted class. The class with the most votes wins (majority rule).
  • Soft Voting: Models output class probabilities. The class with the highest average probability across all models wins. Generally more accurate than hard voting.

Voting ensembles work best when the base models are diverse and make different types of errors. If all models make the same mistakes, voting cannot improve on any single model.

Key Properties

  • Simple to Implement: No meta-learning needed, just aggregate predictions.
  • Diversity Matters: Works best with models that make uncorrelated errors.
  • Weighted Voting: Can assign higher weights to more accurate models.
  • Condorcet Jury Theorem: If each model is >50% accurate and errors are independent, ensemble accuracy approaches 100%.

Implementation

function votingEnsemble(models, mode) {
    return {
        predict: function(x) {
            if (mode === 'soft') {
                // Soft voting: average probabilities
                var avgProbs = {};
                models.forEach(function(model) {
                    var probs = model.predictProba(x);
                    Object.keys(probs).forEach(function(cls) {
                        avgProbs[cls] = (avgProbs[cls] || 0) + probs[cls];
                    });
                });
                return Object.keys(avgProbs).reduce(function(a, b) {
                    return avgProbs[a] > avgProbs[b] ? a : b;
                });
            } else {
                // Hard voting: majority rule
                var votes = {};
                models.forEach(function(model) {
                    var pred = model.predict(x);
                    votes[pred] = (votes[pred] || 0) + 1;
                });
                return Object.keys(votes).reduce(function(a, b) {
                    return votes[a] > votes[b] ? a : b;
                });
            }
        }
    };
}
