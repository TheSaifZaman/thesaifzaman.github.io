Bagging (Bootstrap Aggregating)

Medium Training: O(n·T) Prediction: O(T)

How It Works

Bagging (Bootstrap Aggregating) is an ensemble technique that reduces variance by training multiple models on different random subsets of the training data. Each subset is created by sampling with replacement (bootstrap sampling) from the original dataset.

Each base model (often a decision tree) is trained independently on its bootstrap sample. For classification, the final prediction is made by majority vote; for regression, by averaging. Because each model sees a slightly different version of the data, the ensemble is more robust and less prone to overfitting than any single model.

Random Forest is the most well-known extension of bagging, adding random feature selection at each split to further decorrelate the trees.

Key Properties

  • Variance Reduction: Averaging multiple models reduces overfitting.
  • Parallelizable: Each base model can be trained independently.
  • Bootstrap Sampling: Each model sees ~63.2% unique samples on average.
  • Out-of-Bag Error: ~36.8% unseen samples per model provide a free validation estimate.

function bagging(X, y, numModels, sampleSize) {
    var models = [];
    var n = X.length;
    for (var t = 0; t < numModels; t++) {
        // Bootstrap sample
        var sampleX = [], sampleY = [];
        for (var i = 0; i < sampleSize; i++) {
            var idx = Math.floor(Math.random() * n);
            sampleX.push(X[idx]);
            sampleY.push(y[idx]);
        }
        models.push(trainDecisionTree(sampleX, sampleY));
    }
    return {
        predict: function(x) {
            var votes = {};
            models.forEach(function(m) {
                var pred = m.predict(x);
                votes[pred] = (votes[pred] || 0) + 1;
            });
            return Object.keys(votes).reduce(function(a, b) {
                return votes[a] > votes[b] ? a : b;
            });
        }
    };
}
