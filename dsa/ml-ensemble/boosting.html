Boosting

Medium Training: O(n·T·d) Prediction: O(T·d)

How It Works

Boosting is a sequential ensemble technique that builds models one at a time, where each new model focuses on correcting the errors made by the previous ones. Unlike bagging (which trains models in parallel), boosting iteratively adjusts sample weights or fits residuals.

At each round, misclassified samples receive higher weights so that the next weak learner pays more attention to them. The final prediction is a weighted combination of all weak learners. This sequential focus on hard examples makes boosting extremely powerful at reducing both bias and variance.

Popular boosting algorithms include AdaBoost, Gradient Boosting, XGBoost, LightGBM, and CatBoost.

Key Properties

  • Sequential Training: Each model depends on the errors of previous models.
  • Bias Reduction: Boosting primarily reduces bias by focusing on hard examples.
  • Weighted Combination: Better-performing models get higher weights in the final prediction.
  • Overfitting Risk: Can overfit noisy data if too many rounds are used without regularization.

function gradientBoosting(X, y, numRounds, learningRate) {
    var predictions = new Array(X.length).fill(0);
    var models = [];

    for (var t = 0; t < numRounds; t++) {
        // Compute residuals
        var residuals = y.map(function(yi, i) {
            return yi - predictions[i];
        });
        // Fit weak learner to residuals
        var model = trainDecisionStump(X, residuals);
        models.push({ model: model, lr: learningRate });
        // Update predictions
        for (var i = 0; i < X.length; i++) {
            predictions[i] += learningRate * model.predict(X[i]);
        }
    }
    return {
        predict: function(x) {
            var sum = 0;
            models.forEach(function(m) {
                sum += m.lr * m.model.predict(x);
            });
            return sum;
        }
    };
}
