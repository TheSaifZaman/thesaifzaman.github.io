Stacking (Stacked Generalization)

Hard Training: O(k·n·C) Prediction: O(k + M)

How It Works

Stacking (Stacked Generalization) trains a meta-learner to combine the predictions of multiple diverse base models. Instead of simple voting or averaging, the meta-learner learns the optimal way to combine base model outputs.

The process has two levels: Level 0 consists of k diverse base models (e.g., Random Forest, SVM, Neural Network, KNN). Each base model is trained using cross-validation, and their out-of-fold predictions form a new feature matrix. Level 1 is the meta-learner (often logistic regression or a simple model) trained on this new feature matrix to produce the final prediction.

Cross-validation at Level 0 is critical to prevent the meta-learner from overfitting to the training set. Each base model never sees its own training data predictions, only out-of-fold predictions.

Key Properties

  • Diversity: Works best when base models are diverse (different algorithms, features, or hyperparameters).
  • Meta-Learning: Learns which models are reliable for which types of inputs.
  • Cross-Validation: Base model predictions are generated via CV to prevent leakage.
  • Multi-Level: Can be extended to multiple stacking levels, though diminishing returns apply.

function stacking(X, y, baseModels, metaModel, kFolds) {
    var n = X.length;
    var metaFeatures = [];
    for (var i = 0; i < n; i++) metaFeatures.push(new Array(baseModels.length).fill(0));

    // Level 0: generate out-of-fold predictions
    var folds = createKFolds(n, kFolds);
    baseModels.forEach(function(model, mi) {
        folds.forEach(function(fold) {
            var trainX = fold.trainIdx.map(function(i) { return X[i]; });
            var trainY = fold.trainIdx.map(function(i) { return y[i]; });
            model.fit(trainX, trainY);
            fold.valIdx.forEach(function(i) {
                metaFeatures[i][mi] = model.predict(X[i]);
            });
        });
    });

    // Level 1: train meta-learner
    metaModel.fit(metaFeatures, y);

    // Retrain base models on full data for inference
    baseModels.forEach(function(model) { model.fit(X, y); });

    return {
        predict: function(x) {
            var features = baseModels.map(function(m) { return m.predict(x); });
            return metaModel.predict(features);
        }
    };
}
