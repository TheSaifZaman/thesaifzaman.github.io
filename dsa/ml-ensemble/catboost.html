CatBoost

Hard Training: O(n·d·T) Prediction: O(T·d)

How It Works

CatBoost (Categorical Boosting) by Yandex is a gradient boosting framework specifically designed to handle categorical features natively without requiring manual one-hot encoding or label encoding.

CatBoost introduces Ordered Target Encoding: for each sample, the categorical feature value is encoded using the target statistics from only the preceding samples in a random permutation. This prevents target leakage that occurs with naive target encoding.

CatBoost also uses Ordered Boosting, where each tree is built using a separate random permutation of the training data to avoid prediction shift (a subtle form of overfitting unique to gradient boosting). It builds oblivious trees (symmetric) where the same split condition is used across all nodes at the same depth.

Key Properties

  • Native Categorical Support: Handles categorical features directly with ordered target encoding.
  • Ordered Boosting: Eliminates prediction shift using random permutations.
  • Oblivious Trees: Same split at each depth level enables fast inference via bitwise operations.
  • Low Overfitting: Ordered encoding + ordered boosting provide strong regularization.

Implementation

// CatBoost ordered target encoding concept
function orderedTargetEncode(data, catCol, targetCol, permutation) {
    var encoded = new Array(data.length);
    var countMap = {};  // category -> { sum, count }
    var prior = mean(data.map(function(r) { return r[targetCol]; }));

    for (var i = 0; i < permutation.length; i++) {
        var idx = permutation[i];
        var cat = data[idx][catCol];
        if (!countMap[cat]) countMap[cat] = { sum: 0, count: 0 };

        var stats = countMap[cat];
        // Encode using only preceding samples
        encoded[idx] = (stats.sum + prior) / (stats.count + 1);

        // Then update stats with current sample
        stats.sum += data[idx][targetCol];
        stats.count += 1;
    }
    return encoded;
}
