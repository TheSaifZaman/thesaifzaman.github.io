How It Works

Logic building is the foundation of all programming and algorithm design. It involves breaking down a problem into smaller, manageable pieces and systematically reasoning through each piece to arrive at a solution. Before diving into complex data structures and algorithms, a strong grasp of logical thinking, pattern recognition, and conditional reasoning is essential.

At its core, logic building relies on three fundamental constructs: sequence (executing steps in order), selection (making decisions with conditionals), and iteration (repeating actions with loops). Every algorithm, no matter how complex, is composed of these building blocks. Mastering them means you can decompose any problem into code.

A classic exercise for logic building is the FizzBuzz problem: iterate through numbers, apply conditional rules (divisible by 3, by 5, or both), and output the correct label. This simple exercise combines loops, conditionals, and the modulo operator — training your brain to translate rules into code.

Decomposition
Break complex problems into smaller sub-problems that are easier to solve individually.
Pattern Recognition
Identify recurring structures or rules (e.g., every 3rd number, alternating sequences) to simplify logic.
Conditional Thinking
Use if/else branches to handle different cases — order of conditions matters (check the most specific condition first).
Loop Mechanics
Understand when to use for vs. while loops, how to control iteration with counters, and when to break or continue.

// FizzBuzz — the classic logic building exercise
function fizzBuzz(n) {
    var results = [];
    for (var i = 1; i <= n; i++) {
        if (i % 15 === 0) {
            results.push("FizzBuzz");
        } else if (i % 3 === 0) {
            results.push("Fizz");
        } else if (i % 5 === 0) {
            results.push("Buzz");
        } else {
            results.push(String(i));
        }
    }
    return results;
}

// Number pattern — triangle of stars
function printTriangle(rows) {
    for (var i = 1; i <= rows; i++) {
        var line = "";
        for (var j = 0; j < i; j++) {
            line += "* ";
        }
        console.log(line);
    }
}

// Digit manipulation — subtract product and sum
function subtractProductAndSum(n) {
    var product = 1;
    var sum = 0;
    while (n > 0) {
        var digit = n % 10;
        product *= digit;
        sum += digit;
        n = Math.floor(n / 10);
    }
    return product - sum;
}

// Power of two check using bit manipulation
function isPowerOfTwo(n) {
    if (n <= 0) return false;
    return (n & (n - 1)) === 0;
}
