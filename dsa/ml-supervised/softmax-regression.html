Softmax Regression

Medium Train: O(n·k·d) Predict: O(k·d) Space: O(k·d)

How It Works

Softmax Regression (Multinomial Logistic Regression) extends logistic regression to handle multiple classes. Instead of a sigmoid function, it uses the softmax function to output a probability distribution over all k classes.

Each class has its own weight vector. The softmax function ensures all output probabilities sum to 1. The model is trained using cross-entropy loss and gradient descent.

Math Formula

  • Softmax: P(y=k|x) = ewkTx / Σj ewjTx
  • Cross-entropy loss: J = −(1/n) Σi Σk yik log(pik)
  • Gradient: ∇Jk = (1/n) XT(pk − yk)
  • Output: k probability values summing to 1.

Key Properties

  • Multiclass: Naturally handles more than 2 classes.
  • Probabilistic: Outputs calibrated class probabilities.
  • Linear boundaries: Decision regions are convex polytopes.
  • Foundation: Output layer of most neural network classifiers.

Implementation

function softmax(z) {
    var max = Math.max.apply(null, z);
    var exps = z.map(function(v) { return Math.exp(v - max); });
    var sum = exps.reduce(function(a, b) { return a + b; }, 0);
    return exps.map(function(e) { return e / sum; });
}

function softmaxRegression(X, Y, k, lr, epochs) {
    var d = X[0].length, n = X.length;
    var W = zeros(k, d); // k weight vectors
    for (var e = 0; e < epochs; e++) {
        for (var i = 0; i < n; i++) {
            var z = [];
            for (var c = 0; c < k; c++) z.push(dot(W[c], X[i]));
            var probs = softmax(z);
            for (var c2 = 0; c2 < k; c2++) {
                var err = probs[c2] - (Y[i] === c2 ? 1 : 0);
                for (var j = 0; j < d; j++) W[c2][j] -= lr * err * X[i][j];
            }
        }
    }
    return W;
}
