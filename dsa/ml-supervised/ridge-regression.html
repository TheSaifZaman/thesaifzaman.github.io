Ridge Regression (L2)

Medium Train: O(nd²) Predict: O(d) Space: O(d)

How It Works

Ridge Regression adds an L2 penalty term to the ordinary least squares cost function. This shrinks the coefficients toward zero (but never exactly to zero), preventing overfitting by discouraging large weights.

The regularization parameter λ controls the strength of the penalty. As λ increases, the coefficients shrink more, and the model becomes simpler and more biased but less prone to variance.

Math Formula

  • Cost Function: J(w) = (1/n) Σi(yi − ŷi)² + λ Σjwj²
  • L2 Penalty: λ||w||2² = λ Σ wj²
  • Closed-form: w = (XTX + λI)−1XTy
  • Effect: Shrinks all coefficients toward zero proportionally.

Key Properties

  • Regularized: Prevents overfitting by penalizing large weights.
  • No feature selection: All features are kept but shrunk.
  • Bias-variance tradeoff: λ controls the balance.
  • Multicollinearity: Handles correlated features better than OLS.

Implementation

function ridgeRegression(X, y, lambda) {
    // w = (X^T X + lambda*I)^-1 X^T y
    var XtX = matMul(transpose(X), X);
    var n = XtX.length;
    for (var i = 0; i < n; i++) {
        XtX[i][i] += lambda; // Add L2 penalty
    }
    var XtY = matVecMul(transpose(X), y);
    return solve(XtX, XtY);
}
