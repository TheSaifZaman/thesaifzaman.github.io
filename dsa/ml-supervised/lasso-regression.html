Lasso Regression (L1)

Medium Train: O(nd) Predict: O(d) Space: O(d)

How It Works

Lasso (Least Absolute Shrinkage and Selection Operator) adds an L1 penalty to the cost function. Unlike Ridge, the L1 penalty can drive coefficients exactly to zero, effectively performing feature selection.

This makes Lasso particularly useful when you suspect many features are irrelevant. As λ increases, more coefficients become exactly zero, yielding a sparser, more interpretable model.

Math Formula

  • Cost Function: J(w) = (1/n) Σi(yi − ŷi)² + λ Σj|wj|
  • L1 Penalty: λ||w||1 = λ Σ |wj|
  • No closed-form: Solved via coordinate descent or subgradient methods.
  • Sparsity: L1 penalty produces exact zeros in the weight vector.

Key Properties

  • Feature selection: Automatically zeros out irrelevant features.
  • Sparse models: Produces interpretable, compact models.
  • Non-differentiable: L1 penalty is not smooth at zero.
  • Coordinate descent: Standard solver for Lasso optimization.

Implementation

function lassoCoordinateDescent(X, y, lambda, maxIter) {
    var d = X[0].length, n = X.length;
    var w = new Array(d).fill(0);
    for (var iter = 0; iter < maxIter; iter++) {
        for (var j = 0; j < d; j++) {
            var rho = 0;
            for (var i = 0; i < n; i++) {
                var pred = dot(X[i], w) - w[j] * X[i][j];
                rho += X[i][j] * (y[i] - pred);
            }
            // Soft thresholding
            if (rho < -lambda/2) w[j] = (rho + lambda/2);
            else if (rho > lambda/2) w[j] = (rho - lambda/2);
            else w[j] = 0;
        }
    }
    return w;
}
