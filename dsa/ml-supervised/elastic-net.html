Elastic Net

Medium Train: O(nd) Predict: O(d) Space: O(d)

How It Works

Elastic Net combines L1 (Lasso) and L2 (Ridge) regularization. It uses a mixing parameter α to balance between the two penalties. This gives both feature selection (from L1) and coefficient shrinkage (from L2).

When features are correlated, Lasso tends to pick one and ignore the others. Elastic Net handles this grouping effect better — correlated features tend to get similar coefficients.

Math Formula

  • Cost: J(w) = (1/n) Σ(yi − ŷi)² + λ[α||w||1 + (1−α)||w||2²]
  • α = 1: Pure Lasso (L1 only)
  • α = 0: Pure Ridge (L2 only)
  • 0 < α < 1: Mix of both penalties

Key Properties

  • Best of both: Feature selection + coefficient shrinkage.
  • Grouping effect: Correlated features get similar weights.
  • Two hyperparameters: λ (strength) and α (mix ratio).
  • Robust: Works well when features outnumber samples.

Implementation

function elasticNet(X, y, lambda, alpha, maxIter) {
    var d = X[0].length, n = X.length;
    var w = new Array(d).fill(0);
    for (var iter = 0; iter < maxIter; iter++) {
        for (var j = 0; j < d; j++) {
            var rho = 0;
            for (var i = 0; i < n; i++) {
                var pred = dot(X[i], w) - w[j] * X[i][j];
                rho += X[i][j] * (y[i] - pred);
            }
            // Soft thresholding with L2 denominator
            var l1 = lambda * alpha;
            var l2 = lambda * (1 - alpha);
            if (rho < -l1/2) w[j] = (rho + l1/2) / (1 + l2);
            else if (rho > l1/2) w[j] = (rho - l1/2) / (1 + l2);
            else w[j] = 0;
        }
    }
    return w;
}
