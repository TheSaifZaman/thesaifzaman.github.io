Bayesian Linear Regression

Hard Train: O(n³) Predict: O(d²) Space: O(d²)

How It Works

Bayesian Linear Regression treats model parameters as probability distributions rather than fixed values. It starts with a prior belief about the weights and updates it using observed data to produce a posterior distribution. This naturally provides uncertainty estimates for predictions.

Instead of a single best-fit line, Bayesian regression gives a distribution of possible lines. Regions with more data have narrower uncertainty bands, while predictions far from the data have wider uncertainty.

Math Formula

  • Prior: P(w) = N(w | 0, σw²I)
  • Likelihood: P(y | X, w) = N(y | Xw, σ2I)
  • Posterior: P(w | X, y) ∝ P(y | X, w) · P(w)
  • Posterior mean: w̄ = (σ−2XTX + Σ0−1)−1σ−2XTy
  • Predictive: P(y* | x*) = N(x*Tw̄, x*TΣwx* + σ²)

Key Properties

  • Uncertainty quantification: Provides confidence intervals for predictions.
  • Prior knowledge: Can incorporate domain expertise via priors.
  • No overfitting: Bayesian framework naturally regularizes.
  • Computationally expensive: Requires matrix inversions for exact inference.

Implementation

function bayesianRegression(X, y, priorVar, noiseVar) {
    // Prior: w ~ N(0, priorVar * I)
    // Posterior: w | X, y ~ N(mean_w, cov_w)
    var XtX = matMul(transpose(X), X);
    var priorPrecision = scaleMatrix(identity(X[0].length), 1/priorVar);
    var posteriorPrec = addMatrices(
        scaleMatrix(XtX, 1/noiseVar),
        priorPrecision
    );
    var posteriorCov = inverse(posteriorPrec);
    var posteriorMean = matVecMul(
        posteriorCov,
        scaleVec(matVecMul(transpose(X), y), 1/noiseVar)
    );
    return { mean: posteriorMean, cov: posteriorCov };
}
