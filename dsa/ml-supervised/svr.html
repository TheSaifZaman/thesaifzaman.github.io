Support Vector Regression

Medium Train: O(n²) Predict: O(sv) Space: O(sv)

How It Works

Support Vector Regression (SVR) fits a function within an ε-insensitive tube around the data. Points inside the tube incur no loss; only points outside the tube contribute to the error. The goal is to find the flattest function that has at most ε deviation from the actual values.

The ε parameter controls the width of the tube. Larger ε means more tolerance and a simpler model. Support vectors are the data points that lie on or outside the tube boundary.

Math Formula

  • Objective: min ½||w||² + C Σi(ξi + ξi*)
  • Constraints: |yi − (w·xi + b)| ≤ ε + ξi
  • ε-insensitive loss: Lε(y, f(x)) = max(0, |y − f(x)| − ε)
  • Support vectors: Points on or outside the ε-tube boundary.

Key Properties

  • Epsilon tube: Points inside the tube have zero loss.
  • Sparse solution: Only support vectors define the model.
  • Kernel trick: Can model nonlinear relationships via kernels.
  • Robust: Less sensitive to individual outliers than OLS.

Implementation

function svrPredict(x, supportVectors, alphas, b, kernel) {
    var sum = 0;
    for (var i = 0; i < supportVectors.length; i++) {
        sum += alphas[i] * kernel(supportVectors[i], x);
    }
    return sum + b;
}

function epsilonLoss(y, pred, epsilon) {
    var diff = Math.abs(y - pred);
    return diff > epsilon ? diff - epsilon : 0;
}
