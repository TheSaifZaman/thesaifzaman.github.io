Naive Bayes

Easy Train: O(n·d) Predict: O(d) Space: O(k·d)

How It Works

Naive Bayes applies Bayes' theorem with the "naive" assumption that all features are conditionally independent given the class. Despite this strong assumption rarely being true, it works surprisingly well in practice, especially for text classification.

During training, it computes the prior probability of each class and the likelihood of each feature given the class. At prediction time, it combines these using Bayes' theorem and picks the class with the highest posterior probability.

Math Formula

  • Bayes' Theorem: P(C|x) = P(x|C) · P(C) / P(x)
  • Naive assumption: P(x|C) = Πj P(xj|C)
  • Prediction: ŷ = argmaxC P(C) Πj P(xj|C)
  • Gaussian NB: P(xj|C) = N(xj; μjC, σjC²)

Key Properties

  • Very fast: Training and prediction are both linear time.
  • Text classification: Excellent for spam detection, sentiment analysis.
  • Small data: Works well even with limited training data.
  • Independence assumption: Correlated features can hurt performance.

Implementation

function naiveBayes(trainData, trainLabels) {
    var classes = unique(trainLabels);
    var stats = {}; // { class: { prior, means[], vars[] } }
    classes.forEach(function(c) {
        var subset = trainData.filter(function(_, i) { return trainLabels[i] === c; });
        stats[c] = {
            prior: subset.length / trainData.length,
            means: columnMeans(subset),
            vars: columnVariances(subset)
        };
    });
    return {
        predict: function(x) {
            var bestClass = null, bestProb = -Infinity;
            classes.forEach(function(c) {
                var logProb = Math.log(stats[c].prior);
                for (var j = 0; j < x.length; j++) {
                    logProb += gaussianLogPdf(x[j], stats[c].means[j], stats[c].vars[j]);
                }
                if (logProb > bestProb) { bestProb = logProb; bestClass = c; }
            });
            return bestClass;
        }
    };
}
