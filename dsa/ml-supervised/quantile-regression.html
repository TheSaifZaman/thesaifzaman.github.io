Quantile Regression

Medium Train: O(n) Predict: O(1) Space: O(d)

How It Works

Quantile Regression predicts specific quantiles (e.g., median, 25th percentile, 75th percentile) of the response variable instead of the mean. It uses an asymmetric loss function that weights overestimation and underestimation differently.

This is useful when you care about prediction intervals or when the data has heteroscedastic variance (different spread at different x values). By fitting multiple quantile lines, you get a complete picture of the conditional distribution.

Math Formula

  • Check loss: ρτ(u) = u(τ − I(u < 0))
  • Cost: J(w) = Σi ρτ(yi − xiTw)
  • τ = 0.5: Median regression (robust to outliers)
  • τ = 0.1, 0.9: Lower and upper prediction bounds

Key Properties

  • Robust: Median regression is less sensitive to outliers than OLS.
  • Prediction intervals: Multiple quantiles give prediction bands.
  • No normality assumption: Works for any error distribution.
  • Linear programming: Can be solved as a linear program.

Implementation

function quantileLoss(y, yHat, tau) {
    var residual = y - yHat;
    if (residual >= 0) return tau * residual;
    return (tau - 1) * residual;
}

function quantileRegression(points, tau, lr, epochs) {
    var w = 0, b = 0;
    for (var e = 0; e < epochs; e++) {
        for (var i = 0; i < points.length; i++) {
            var pred = w * points[i].x + b;
            var err = points[i].y - pred;
            var grad = err >= 0 ? -tau : (1 - tau);
            w -= lr * grad * (-points[i].x);
            b -= lr * grad * (-1);
        }
    }
    return { slope: w, intercept: b };
}
