K-Nearest Neighbors

Easy Train: O(1) Predict: O(n) Space: O(n)

How It Works

KNN is a lazy learning algorithm that stores all training data and classifies new points by finding the k closest training examples and taking a majority vote. There is no explicit training phase — all computation happens at prediction time.

The choice of k controls the model's complexity. Small k values create complex, jagged boundaries (low bias, high variance), while large k values create smoother boundaries (high bias, low variance).

Math Formula

  • Euclidean distance: d(x, x') = √(Σj(xj − x'j)²)
  • Prediction: ŷ = mode({yi : xi ∈ Nk(x)})
  • Weighted KNN: wi = 1/d(x, xi)
  • Optimal k: Typically chosen via cross-validation, often k = √n.

Key Properties

  • Lazy learner: No training phase; stores entire dataset.
  • Non-parametric: Makes no assumptions about the data distribution.
  • Curse of dimensionality: Performance degrades in high dimensions.
  • Distance metric matters: Results depend heavily on the distance function.

function knn(trainData, trainLabels, query, k) {
    var distances = [];
    for (var i = 0; i < trainData.length; i++) {
        var d = euclidean(trainData[i], query);
        distances.push({ dist: d, label: trainLabels[i] });
    }
    distances.sort(function(a, b) { return a.dist - b.dist; });
    var votes = {};
    for (var j = 0; j < k; j++) {
        var lbl = distances[j].label;
        votes[lbl] = (votes[lbl] || 0) + 1;
    }
    return majorityVote(votes);
}

function euclidean(a, b) {
    var sum = 0;
    for (var i = 0; i < a.length; i++) sum += (a[i]-b[i])*(a[i]-b[i]);
    return Math.sqrt(sum);
}
