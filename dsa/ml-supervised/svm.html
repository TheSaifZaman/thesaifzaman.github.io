Support Vector Machine

Medium Train: O(n²) Predict: O(sv) Space: O(sv)

How It Works

SVM finds the optimal hyperplane that maximizes the margin between two classes. The margin is the distance between the decision boundary and the nearest data points from each class (called support vectors).

For non-linearly separable data, the kernel trick maps data to a higher-dimensional space where a linear separator exists. Common kernels include polynomial and RBF (Gaussian).

Math Formula

  • Objective: min ½||w||² subject to yi(w·xi + b) ≥ 1
  • Margin: 2/||w|| (maximized by minimizing ||w||²)
  • Soft margin: min ½||w||² + C Σξi
  • Kernel trick: K(xi, xj) = φ(xi)·φ(xj)

Key Properties

  • Maximum margin: Finds the widest possible separation.
  • Support vectors: Only boundary points define the model.
  • Kernel trick: Handles nonlinear boundaries efficiently.
  • Effective in high dimensions: Works well when d > n.

Implementation

function svmPredict(x, supportVectors, alphas, labels, b, kernel) {
    var sum = 0;
    for (var i = 0; i < supportVectors.length; i++) {
        sum += alphas[i] * labels[i] * kernel(supportVectors[i], x);
    }
    return sum + b >= 0 ? 1 : -1;
}

function linearKernel(a, b) {
    return a[0]*b[0] + a[1]*b[1];
}
