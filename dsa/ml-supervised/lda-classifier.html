Linear Discriminant Analysis

Medium Train: O(n·d²) Predict: O(d²) Space: O(d²)

How It Works

LDA projects data onto a lower-dimensional space that maximizes class separability. It finds the projection direction that maximizes the ratio of between-class variance to within-class variance. This makes it both a dimensionality reduction technique and a classifier.

LDA assumes that each class has a Gaussian distribution with the same covariance matrix but different means. The decision boundary is linear because the shared covariance leads to linear discriminant functions.

Math Formula

  • Between-class scatter: SB = Σk nk(μk − μ)(μk − μ)T
  • Within-class scatter: SW = Σk Σi∈Ck (xi − μk)(xi − μk)T
  • Objective: max J(w) = wTSBw / wTSWw
  • Solution: Eigenvectors of SW−1SB

Key Properties

  • Dimensionality reduction: Projects to at most (k−1) dimensions for k classes.
  • Shared covariance: Assumes same spread for all classes.
  • Optimal for Gaussian: Best linear classifier when assumptions hold.
  • Supervised PCA: Like PCA but uses class labels to find best projection.

Implementation

function lda(data, labels) {
    var classes = unique(labels);
    var globalMean = mean2D(data);
    var Sw = zeros(2,2), Sb = zeros(2,2);

    classes.forEach(function(c) {
        var subset = data.filter(function(_,i){ return labels[i]===c; });
        var mu = mean2D(subset);
        var n = subset.length;
        // Within-class scatter
        subset.forEach(function(x) {
            var diff = [x[0]-mu[0], x[1]-mu[1]];
            Sw = addMat(Sw, outerProduct(diff, diff));
        });
        // Between-class scatter
        var diffMu = [mu[0]-globalMean[0], mu[1]-globalMean[1]];
        Sb = addMat(Sb, scaleMat(outerProduct(diffMu, diffMu), n));
    });

    // w = Sw^-1 * (mu1 - mu2)
    var w = matVecMul(inverse2x2(Sw), [/* mu1 - mu2 */]);
    return w;
}
