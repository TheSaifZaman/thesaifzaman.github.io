Polynomial Regression

Easy Train: O(nd²) Predict: O(d) Space: O(d)

How It Works

Polynomial Regression extends linear regression by fitting a polynomial curve instead of a straight line. It transforms the input features into polynomial features (x, x², x³, ...) and then applies linear regression on the expanded feature set.

This allows the model to capture nonlinear relationships while still using the same OLS framework. The degree of the polynomial controls the model's flexibility — higher degrees fit the training data better but risk overfitting.

Math Formula

  • Model: ŷ = w0 + w1x + w2x² + ... + wdxd
  • Cost Function: J(w) = (1/n) Σi=1n (yi − ŷi)²
  • Feature Transform: φ(x) = [1, x, x², ..., xd]
  • Solution: w = (ΦTΦ)−1ΦTy

Key Properties

  • Flexible: Can model nonlinear relationships with appropriate degree.
  • Overfitting risk: High-degree polynomials can overfit small datasets.
  • Feature engineering: Polynomial features are created before fitting.
  • Same solver: Uses OLS after feature transformation.

function polyRegression(points, degree) {
    // Build Vandermonde matrix
    var n = points.length;
    var X = [], Y = [];
    for (var i = 0; i < n; i++) {
        var row = [];
        for (var d = 0; d <= degree; d++) {
            row.push(Math.pow(points[i].x, d));
        }
        X.push(row);
        Y.push(points[i].y);
    }
    // Solve using normal equation (simplified)
    // w = (X^T X)^-1 X^T Y
    return solveNormalEquation(X, Y);
}
