Quadratic Discriminant Analysis

Medium Train: O(n·d²) Predict: O(d²) Space: O(k·d²)

How It Works

QDA is similar to LDA but allows each class to have its own covariance matrix. This results in quadratic (curved) decision boundaries rather than linear ones, making QDA more flexible at the cost of more parameters to estimate.

When class distributions truly differ in spread and orientation, QDA outperforms LDA. However, it requires more training data to reliably estimate per-class covariance matrices.

Math Formula

  • Discriminant: δk(x) = −½log|Σk| − ½(x−μk)TΣk−1(x−μk) + log P(Ck)
  • Per-class covariance: Σk (each class has its own)
  • Decision boundary: Quadratic curve where δ1(x) = δ2(x)
  • Parameters: k·(d² + d)/2 covariance parameters (more than LDA)

Key Properties

  • Curved boundaries: Can model elliptical and parabolic decision surfaces.
  • Per-class covariance: More flexible than LDA's shared covariance.
  • More parameters: Needs more data to avoid overfitting.
  • Generative model: Models each class as a separate Gaussian.

Implementation

function qda(data, labels) {
    var classes = unique(labels);
    var models = {};
    classes.forEach(function(c) {
        var subset = data.filter(function(_,i){ return labels[i]===c; });
        models[c] = {
            prior: subset.length / data.length,
            mean: mean2D(subset),
            cov: covariance2D(subset)
        };
    });
    return {
        predict: function(x) {
            var best = null, bestScore = -Infinity;
            classes.forEach(function(c) {
                var m = models[c];
                var score = -0.5 * logDet(m.cov)
                    - 0.5 * mahalanobis(x, m.mean, m.cov)
                    + Math.log(m.prior);
                if (score > bestScore) { bestScore = score; best = c; }
            });
            return best;
        }
    };
}
