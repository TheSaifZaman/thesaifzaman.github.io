Logistic Regression

Easy Train: O(nd) Predict: O(d) Space: O(d)

How It Works

Logistic Regression models the probability that an input belongs to a particular class using the sigmoid (logistic) function. Despite its name, it is a classification algorithm. It finds a linear decision boundary and maps predictions to probabilities between 0 and 1.

The model optimizes the binary cross-entropy loss using gradient descent. The decision boundary is the set of points where the predicted probability equals 0.5.

Math Formula

  • Sigmoid: σ(z) = 1 / (1 + e−z)
  • Model: P(y=1|x) = σ(wTx + b)
  • Loss (BCE): J = −(1/n) Σ[yi log(pi) + (1−yi) log(1−pi)]
  • Decision boundary: wTx + b = 0 (where P = 0.5)

Key Properties

  • Probabilistic output: Returns class probabilities, not just labels.
  • Linear boundary: Decision surface is a hyperplane.
  • Convex loss: Binary cross-entropy has a single global minimum.
  • Regularizable: Can add L1/L2 penalties to prevent overfitting.

Implementation

function sigmoid(z) { return 1 / (1 + Math.exp(-z)); }

function logisticRegression(X, y, lr, epochs) {
    var w = new Array(X[0].length).fill(0);
    var b = 0, n = X.length;
    for (var e = 0; e < epochs; e++) {
        var dw = new Array(w.length).fill(0), db = 0;
        for (var i = 0; i < n; i++) {
            var z = dot(w, X[i]) + b;
            var a = sigmoid(z);
            var err = a - y[i];
            for (var j = 0; j < w.length; j++) dw[j] += err * X[i][j];
            db += err;
        }
        for (var j2 = 0; j2 < w.length; j2++) w[j2] -= lr * dw[j2] / n;
        b -= lr * db / n;
    }
    return { weights: w, bias: b };
}
