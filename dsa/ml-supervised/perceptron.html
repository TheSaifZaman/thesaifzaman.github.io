Perceptron

Easy Train: O(n·d) Predict: O(d) Space: O(d)

How It Works

The Perceptron is the simplest neural network — a single-layer linear classifier. It computes a weighted sum of inputs, applies a step function, and outputs a binary class. It learns by updating weights whenever a misclassification occurs.

The perceptron convergence theorem guarantees that if the data is linearly separable, the algorithm will find a separating hyperplane in finite steps. For non-linearly separable data, it will not converge.

Math Formula

  • Output: ŷ = sign(w·x + b)
  • Update rule: w ← w + η(yi − ŷi)xi
  • Bias update: b ← b + η(yi − ŷi)
  • Convergence: Guaranteed for linearly separable data.

Key Properties

  • Simplest NN: Foundation of all neural network architectures.
  • Online learning: Updates weights one sample at a time.
  • Linear only: Cannot solve XOR or non-linearly separable problems.
  • Building block: Multi-layer perceptrons (MLP) overcome the linear limitation.

Implementation

function perceptron(X, y, lr, maxEpochs) {
    var w = new Array(X[0].length).fill(0);
    var b = 0;
    for (var epoch = 0; epoch < maxEpochs; epoch++) {
        var errors = 0;
        for (var i = 0; i < X.length; i++) {
            var pred = dot(w, X[i]) + b >= 0 ? 1 : -1;
            if (pred !== y[i]) {
                for (var j = 0; j < w.length; j++) {
                    w[j] += lr * y[i] * X[i][j];
                }
                b += lr * y[i];
                errors++;
            }
        }
        if (errors === 0) break; // Converged
    }
    return { weights: w, bias: b };
}
