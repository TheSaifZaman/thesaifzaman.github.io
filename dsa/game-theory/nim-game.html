Nim Game

Medium Single Pile: O(1) Multi-Pile (Sprague-Grundy): O(n) Space: O(1)

How It Works

Nim is a mathematical strategy game where two players take turns removing objects from piles. The player who takes the last object wins (in normal play) or loses (in misere play). Despite its simplicity, Nim contains deep mathematical structure that forms the foundation of combinatorial game theory.

The Sprague-Grundy theorem provides a complete solution: compute the XOR (nim-sum) of all pile sizes. If the XOR is non-zero, the first player has a winning strategy; if zero, the second player wins. The winning move is to reduce one pile so that the resulting XOR becomes zero. For a single pile where players can take 1 to 3 objects, the first player loses if and only if the pile size is a multiple of 4.

The Sprague-Grundy theorem generalizes Nim to all impartial games (where both players have the same moves available). Each game position has a Grundy number, and combined games are solved by XOR-ing their Grundy numbers — the same principle as Nim. This applies to many LeetCode game-theory problems.

Key Properties

  • Nim-Sum (XOR): XOR of all pile sizes determines the winner. Non-zero = first player wins.
  • Winning Move: Find a pile to reduce so that the new XOR equals zero.
  • Single Pile (1-3): First player wins iff n is NOT a multiple of 4.
  • Sprague-Grundy: Generalizes to any impartial game via Grundy numbers.

Implementation

// Single pile Nim (take 1 to 3)
function canWinNim(n) {
    return n % 4 !== 0;
}

// Multi-pile Nim
function multiPileNim(piles) {
    var nimSum = 0;
    for (var i = 0; i < piles.length; i++) {
        nimSum ^= piles[i];
    }
    return nimSum !== 0; // true = first player wins
}

// Find winning move in multi-pile Nim
function findWinningMove(piles) {
    var nimSum = 0;
    for (var i = 0; i < piles.length; i++) nimSum ^= piles[i];
    if (nimSum === 0) return null; // no winning move

    for (var i = 0; i < piles.length; i++) {
        var target = piles[i] ^ nimSum;
        if (target < piles[i]) {
            return { pile: i, take: piles[i] - target };
        }
    }
    return null;
}

// Sprague-Grundy for general impartial games
function grundy(n, memo) {
    if (n === 0) return 0;
    if (memo[n] !== undefined) return memo[n];
    var reachable = new Set();
    // For each possible move (game-specific)
    for (var take = 1; take <= 3 && take <= n; take++) {
        reachable.add(grundy(n - take, memo));
    }
    // MEX: minimum excludant
    var mex = 0;
    while (reachable.has(mex)) mex++;
    memo[n] = mex;
    return mex;
}
