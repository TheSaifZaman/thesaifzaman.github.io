Alpha-Beta Pruning

Hard Best: O(b^(d/2)) Worst: O(b^d) Space: O(d)

How It Works

Alpha-Beta Pruning is an optimization of the Minimax algorithm that skips (prunes) branches of the game tree that cannot possibly influence the final decision. It maintains two values: alpha (the best score the maximizer can guarantee) and beta (the best score the minimizer can guarantee). When alpha ≥ beta, the current branch is pruned.

At a maximizer node, if we find a child with value ≥ beta, we can stop (beta cutoff) — the minimizer parent would never choose this path since it already has a better option. Similarly, at a minimizer node, if we find a child with value ≤ alpha, we can stop (alpha cutoff) — the maximizer parent would never choose this path.

With optimal move ordering, alpha-beta pruning reduces the effective branching factor from b to √b, doubling the searchable depth for the same computation time. In chess, this means searching 12 moves deep instead of 6. Move ordering heuristics (killer moves, transposition tables, iterative deepening) are critical for achieving near-optimal pruning.

Key Properties

  • Same Result: Always produces the same result as full Minimax — just faster.
  • Alpha: Best already-guaranteed value for maximizer (lower bound).
  • Beta: Best already-guaranteed value for minimizer (upper bound).
  • Move Ordering: Examining best moves first maximizes pruning (O(b^(d/2)) best case).

Implementation

function alphaBeta(node, depth, alpha, beta, isMaximizing) {
    if (depth === 0 || isTerminal(node)) {
        return evaluate(node);
    }

    if (isMaximizing) {
        var maxEval = -Infinity;
        var children = getChildren(node);
        for (var i = 0; i < children.length; i++) {
            var val = alphaBeta(children[i], depth - 1, alpha, beta, false);
            maxEval = Math.max(maxEval, val);
            alpha = Math.max(alpha, val);
            if (beta <= alpha) break; // Beta cutoff (prune!)
        }
        return maxEval;
    } else {
        var minEval = Infinity;
        var children = getChildren(node);
        for (var i = 0; i < children.length; i++) {
            var val = alphaBeta(children[i], depth - 1, alpha, beta, true);
            minEval = Math.min(minEval, val);
            beta = Math.min(beta, val);
            if (beta <= alpha) break; // Alpha cutoff (prune!)
        }
        return minEval;
    }
}
