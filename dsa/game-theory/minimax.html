Minimax Algorithm

Medium Time: O(b^d) Space: O(d)

How It Works

Minimax is a decision-making algorithm for two-player zero-sum games (where one player's gain is the other's loss). It assumes both players play optimally: the maximizing player tries to maximize the score, while the minimizing player tries to minimize it. The algorithm explores the entire game tree to find the optimal move.

At each node, if it is the maximizer's turn, the algorithm picks the child with the highest value. If it is the minimizer's turn, it picks the child with the lowest value. Leaf nodes are evaluated using a scoring function (e.g., +10 for a win, -10 for a loss, 0 for a draw). Values propagate upward through the tree via this alternating max/min selection.

Minimax is used in chess engines, tic-tac-toe, checkers, Go, and any adversarial search scenario. The time complexity is O(b^d) where b is the branching factor and d is the depth, which is often too expensive for complex games — hence the need for pruning (alpha-beta) and evaluation function heuristics.

Key Properties

  • Optimal Play: Assumes both players play their best possible move.
  • Zero-Sum: One player's gain equals the other's loss.
  • Complete Search: Explores all possible game states (exponential cost).
  • Depth-Limited: In practice, search is limited with a heuristic evaluation function.

Implementation

function minimax(node, depth, isMaximizing) {
    // Base case: leaf node or depth limit
    if (depth === 0 || isTerminal(node)) {
        return evaluate(node);
    }

    if (isMaximizing) {
        var maxEval = -Infinity;
        var children = getChildren(node);
        for (var i = 0; i < children.length; i++) {
            var eval = minimax(children[i], depth - 1, false);
            maxEval = Math.max(maxEval, eval);
        }
        return maxEval;
    } else {
        var minEval = Infinity;
        var children = getChildren(node);
        for (var i = 0; i < children.length; i++) {
            var eval = minimax(children[i], depth - 1, true);
            minEval = Math.min(minEval, eval);
        }
        return minEval;
    }
}

// Find best move for maximizer
function bestMove(board) {
    var best = -Infinity;
    var bestAction = null;
    var moves = getAvailableMoves(board);
    for (var i = 0; i < moves.length; i++) {
        var newBoard = applyMove(board, moves[i]);
        var score = minimax(newBoard, maxDepth, false);
        if (score > best) {
            best = score;
            bestAction = moves[i];
        }
    }
    return bestAction;
}
