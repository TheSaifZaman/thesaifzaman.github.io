Modular Arithmetic

Medium Time: O(log m) for inverse Space: O(1)

How It Works

Modular arithmetic is the arithmetic of remainders. When we say a ≡ b (mod m), we mean that a and b have the same remainder when divided by m. This is fundamental in competitive programming because results often need to be computed modulo a large prime (typically 10^9 + 7) to prevent integer overflow.

The key properties are: (a + b) mod m = ((a mod m) + (b mod m)) mod m, and similarly for multiplication. However, division is different — we cannot simply divide remainders. Instead, we multiply by the modular inverse: a/b mod m = a * b^(-1) mod m, where b^(-1) is computed using Fermat's little theorem (when m is prime): b^(-1) = b^(m-2) mod m.

Common applications include: counting problems with large answers, hashing, cryptography, and combinatorics (computing nCr mod p). Understanding modular arithmetic is essential for handling any problem where the answer could exceed standard integer limits.

Addition/Subtraction
(a ± b) mod m = ((a mod m) ± (b mod m) + m) mod m. Add m before mod to handle negatives.
Multiplication
(a * b) mod m = ((a mod m) * (b mod m)) mod m. Safe to mod at every step.
Modular Inverse
b^(-1) mod p = b^(p-2) mod p (Fermat's theorem, p must be prime).
10^9 + 7
The most common modulus in competitive programming. It is prime and fits in 32-bit integers.

Implementation

var MOD = 1000000007;

function modAdd(a, b) {
    return ((a % MOD) + (b % MOD)) % MOD;
}

function modMul(a, b) {
    return ((a % MOD) * (b % MOD)) % MOD;
}

// Modular exponentiation
function modPow(base, exp, mod) {
    var result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 === 1) result = (result * base) % mod;
        exp = Math.floor(exp / 2);
        base = (base * base) % mod;
    }
    return result;
}

// Modular inverse (mod must be prime)
function modInverse(a, mod) {
    return modPow(a, mod - 2, mod);
}

// Modular division: a / b mod p
function modDiv(a, b, mod) {
    return (a % mod) * modInverse(b, mod) % mod;
}
