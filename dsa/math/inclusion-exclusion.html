Inclusion-Exclusion

Medium Time: O(2^n) subsets Space: O(n)

How It Works

The Inclusion-Exclusion Principle is a counting technique that computes the size of the union of multiple sets by alternately adding and subtracting the sizes of their intersections. For two sets: |A ∪ B| = |A| + |B| - |A ∩ B|. For three sets: |A ∪ B ∪ C| = |A| + |B| + |C| - |A ∩ B| - |A ∩ C| - |B ∩ C| + |A ∩ B ∩ C|.

The general formula alternates signs: add sizes of individual sets, subtract sizes of pairwise intersections, add sizes of triple intersections, and so on. For n sets, this involves summing over all 2^n - 1 non-empty subsets, with the sign determined by whether the subset size is odd (add) or even (subtract).

In competitive programming, inclusion-exclusion is used for counting numbers divisible by at least one of several primes, computing Euler's totient function, solving derangement problems, and counting surjective functions. It often transforms a hard counting problem into a manageable sum.

Two Sets
|A ∪ B| = |A| + |B| - |A ∩ B|. Subtract the overlap to avoid double-counting.
Alternating Signs
Add odd-sized intersections, subtract even-sized ones. Sign = (-1)^(k+1) for k-set intersection.
Divisibility Counting
Count integers ≤ n divisible by any of {p1, p2, ...}: apply IE with |divisible by pi| = floor(n/pi).
Derangements
D(n) = n! * sum((-1)^k / k!, k=0..n). A classic application of inclusion-exclusion.

// Count integers in [1, n] divisible by at least one prime in primes[]
function countDivisible(n, primes) {
    var k = primes.length;
    var total = 0;

    // Iterate over all non-empty subsets (bitmask)
    for (var mask = 1; mask < (1 << k); mask++) {
        var product = 1;
        var bits = 0;
        for (var i = 0; i < k; i++) {
            if (mask & (1 << i)) {
                product *= primes[i];
                bits++;
            }
        }
        // Odd number of primes → add, even → subtract
        if (bits % 2 === 1) {
            total += Math.floor(n / product);
        } else {
            total -= Math.floor(n / product);
        }
    }
    return total;
}

// Derangements: permutations where no element is in its original position
function derangements(n) {
    if (n === 0) return 1;
    if (n === 1) return 0;
    var result = 0;
    var factorial = 1;
    for (var i = 2; i <= n; i++) factorial *= i;
    var sign = 1;
    var kFact = 1;
    for (var k = 0; k <= n; k++) {
        if (k > 0) kFact *= k;
        result += sign * factorial / kFact;
        sign *= -1;
    }
    return result;
}

// Euler's totient via inclusion-exclusion
function eulerTotient(n) {
    var result = n;
    for (var p = 2; p * p <= n; p++) {
        if (n % p === 0) {
            while (n % p === 0) n /= p;
            result -= result / p;
        }
    }
    if (n > 1) result -= result / n;
    return result;
}
