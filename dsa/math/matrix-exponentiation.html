Matrix Exponentiation

Hard Time: O(k³ log n) Space: O(k²)

How It Works

Matrix exponentiation combines binary exponentiation with matrix multiplication to compute the n-th term of a linear recurrence in O(k³ log n) time, where k is the matrix size. The classic example is computing the n-th Fibonacci number: F(n) can be expressed as [[1,1],[1,0]]^n, allowing us to find F(10^18) efficiently.

Any linear recurrence of the form f(n) = c1*f(n-1) + c2*f(n-2) + ... + ck*f(n-k) can be converted into matrix form. The transformation matrix captures the recurrence coefficients, and raising it to the n-th power gives us the n-th term directly. This is far more efficient than the naive O(n) iterative approach for very large n.

The technique works because matrix multiplication is associative, allowing us to use the same binary exponentiation (square-and-multiply) approach. At each step we square a k*k matrix (O(k^3)) and we need O(log n) such steps, giving the overall O(k^3 log n) complexity.

Fibonacci Matrix
[[1,1],[1,0]]^n gives [[F(n+1),F(n)],[F(n),F(n-1)]]. Extract F(n) from position [0][1].
Linear Recurrences
Any recurrence f(n) = sum(ci * f(n-i)) maps to a companion matrix of size k*k.
Binary Exponentiation
Same square-and-multiply technique, but operations are matrix multiplications instead of scalar.
Modular Computation
Take mod at each matrix multiplication step to keep values manageable.

Implementation

var MOD = 1000000007;

// Multiply two k*k matrices mod MOD
function matMul(A, B) {
    var k = A.length;
    var C = [];
    for (var i = 0; i < k; i++) {
        C[i] = [];
        for (var j = 0; j < k; j++) {
            var sum = 0;
            for (var p = 0; p < k; p++) {
                sum = (sum + A[i][p] * B[p][j]) % MOD;
            }
            C[i][j] = sum;
        }
    }
    return C;
}

// Matrix exponentiation: compute M^n mod MOD
function matPow(M, n) {
    var k = M.length;
    // Identity matrix
    var result = [];
    for (var i = 0; i < k; i++) {
        result[i] = [];
        for (var j = 0; j < k; j++) {
            result[i][j] = (i === j) ? 1 : 0;
        }
    }
    while (n > 0) {
        if (n % 2 === 1) result = matMul(result, M);
        M = matMul(M, M);
        n = Math.floor(n / 2);
    }
    return result;
}

// Fibonacci(n) via matrix exponentiation
function fibonacci(n) {
    if (n <= 1) return n;
    var M = [[1, 1], [1, 0]];
    var res = matPow(M, n - 1);
    return res[0][0];
}
