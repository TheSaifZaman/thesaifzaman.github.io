Sieve of Eratosthenes

Easy Time: O(n log log n) Space: O(n)

How It Works

The Sieve of Eratosthenes is an ancient and efficient algorithm for finding all prime numbers up to a given limit n. It works by iteratively marking the multiples of each prime number as composite (not prime), starting from 2. After processing all numbers up to sqrt(n), the unmarked numbers that remain are the primes.

The algorithm begins by assuming all numbers from 2 to n are prime. It then takes the smallest unmarked number (2), and marks all its multiples (4, 6, 8, ...) as composite. Next it moves to the next unmarked number (3) and marks its multiples (9, 12, 15, ...). This continues until we have processed all numbers up to sqrt(n).

The time complexity is O(n log log n), which is nearly linear and far more efficient than testing each number individually. The sieve is one of the most practical algorithms in number theory and serves as a building block for many advanced techniques like segmented sieves and prime factorization.

Start from 2
2 is the smallest and only even prime. All even numbers greater than 2 are composite.
Mark Multiples
For each prime p, start marking from p*p (smaller multiples are already marked by smaller primes).
Stop at sqrt(n)
If a number has no factor up to its square root, it is prime. So we only sieve up to sqrt(n).
Nearly Linear
O(n log log n) is effectively linear for practical inputs, making it extremely fast.

Implementation

function sieveOfEratosthenes(n) {
    var isPrime = new Array(n + 1).fill(true);
    isPrime[0] = isPrime[1] = false;

    for (var p = 2; p * p <= n; p++) {
        if (isPrime[p]) {
            // Mark multiples of p starting from p*p
            for (var m = p * p; m <= n; m += p) {
                isPrime[m] = false;
            }
        }
    }

    var primes = [];
    for (var i = 2; i <= n; i++) {
        if (isPrime[i]) primes.push(i);
    }
    return primes;
}
