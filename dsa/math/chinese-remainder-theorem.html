Chinese Remainder Theorem

Hard Time: O(n log M) Space: O(1)

How It Works

The Chinese Remainder Theorem (CRT) states that if we have a system of congruences x ≡ r1 (mod m1), x ≡ r2 (mod m2), ..., x ≡ rk (mod mk), where the moduli m1, m2, ..., mk are pairwise coprime, then there exists a unique solution modulo M = m1 * m2 * ... * mk.

The constructive proof gives us an algorithm: for each i, compute Mi = M/mi, then find the modular inverse yi of Mi mod mi using the extended Euclidean algorithm. The solution is x = sum(ri * Mi * yi) mod M. This reconstruction process is at the heart of many number-theoretic algorithms.

CRT is used in competitive programming for combining results computed under different moduli, in RSA decryption optimization, and in polynomial interpolation. The generalized CRT extends to non-coprime moduli by checking compatibility conditions.

Pairwise Coprime
The moduli must be pairwise coprime (GCD(mi, mj) = 1 for all i != j) for CRT to guarantee a unique solution.
Unique Solution
The solution x is unique modulo M = m1 * m2 * ... * mk.
Construction
x = sum(ri * Mi * yi) mod M, where Mi = M/mi and yi = Mi^(-1) mod mi.

// Extended GCD: returns [gcd, x, y] where a*x + b*y = gcd
function extGcd(a, b) {
    if (b === 0) return [a, 1, 0];
    var res = extGcd(b, a % b);
    return [res[0], res[2], res[1] - Math.floor(a / b) * res[2]];
}

// Chinese Remainder Theorem
// remainders[i], moduli[i] — x ≡ r_i (mod m_i)
function crt(remainders, moduli) {
    var n = remainders.length;
    var M = 1;
    for (var i = 0; i < n; i++) M *= moduli[i];

    var x = 0;
    for (var i = 0; i < n; i++) {
        var Mi = M / moduli[i];
        var res = extGcd(Mi, moduli[i]);
        var yi = ((res[1] % moduli[i]) + moduli[i]) % moduli[i];
        x += remainders[i] * Mi * yi;
    }
    return ((x % M) + M) % M;
}

// Example: x ≡ 2 (mod 3), x ≡ 3 (mod 5), x ≡ 2 (mod 7)
// crt([2, 3, 5], [3, 5, 7]) → 23
