Number Theory

Medium Topic: Primes, Divisibility, GCD

How It Works

Number theory is the branch of mathematics that studies integers and their properties. In competitive programming, number theory provides essential tools for solving problems involving divisibility, primes, modular arithmetic, and combinatorics. Many seemingly complex problems reduce to elegant number-theoretic observations.

The fundamental theorem of arithmetic states that every integer greater than 1 can be uniquely represented as a product of prime factors. This factorization underpins algorithms for GCD computation (Euclidean algorithm), primality testing (trial division, Miller-Rabin), and the Sieve of Eratosthenes for generating all primes up to a given limit.

Key number theory concepts include: divisibility rules, prime numbers, greatest common divisor (GCD) and least common multiple (LCM), modular arithmetic, Euler's totient function, Fermat's little theorem, and the Chinese Remainder Theorem. Mastering these topics unlocks efficient solutions to a wide array of algorithmic challenges.

Primes
Building blocks of integers. Sieve of Eratosthenes finds all primes up to n in O(n log log n).
GCD & LCM
Euclidean algorithm computes GCD in O(log min(a,b)). LCM(a,b) = a*b / GCD(a,b).
Modular Arithmetic
Essential for handling large numbers. (a+b) mod m = ((a mod m) + (b mod m)) mod m.
Euler's Totient
phi(n) counts integers from 1 to n that are coprime to n. Key to Euler's theorem: a^phi(n) = 1 (mod n).

// Check if a number is prime — O(sqrt(n))
function isPrime(n) {
    if (n < 2) return false;
    if (n < 4) return true;
    if (n % 2 === 0 || n % 3 === 0) return false;
    for (var i = 5; i * i <= n; i += 6) {
        if (n % i === 0 || n % (i + 2) === 0) return false;
    }
    return true;
}

// GCD using Euclidean algorithm — O(log min(a,b))
function gcd(a, b) {
    while (b !== 0) {
        var t = b;
        b = a % b;
        a = t;
    }
    return a;
}

// LCM via GCD
function lcm(a, b) {
    return (a / gcd(a, b)) * b;
}

// Euler's totient function — O(sqrt(n))
function eulerTotient(n) {
    var result = n;
    for (var p = 2; p * p <= n; p++) {
        if (n % p === 0) {
            while (n % p === 0) n /= p;
            result -= result / p;
        }
    }
    if (n > 1) result -= result / n;
    return result;
}
