Combinatorics

Medium Time: O(n) for nCr Space: O(n²) for Pascal's

How It Works

Combinatorics is the mathematics of counting. The two fundamental operations are permutations (ordered arrangements) and combinations (unordered selections). The binomial coefficient C(n, r) = n! / (r! * (n-r)!) counts the number of ways to choose r items from n distinct items, and is central to many algorithmic problems.

Pascal's Triangle provides an elegant way to compute binomial coefficients: each entry is the sum of the two entries above it, C(n, r) = C(n-1, r-1) + C(n-1, r). This DP approach avoids computing large factorials and naturally handles the modular arithmetic needed for competitive programming.

Key formulas include: P(n, r) = n!/(n-r)! for permutations, C(n, r) = n!/(r!(n-r)!) for combinations, and the multinomial coefficient for distributing items into groups. Stars and bars, the pigeonhole principle, and generating functions are advanced tools that extend basic counting.

Pascal's Rule
C(n, r) = C(n-1, r-1) + C(n-1, r). Build the triangle row by row in O(n^2).
Symmetry
C(n, r) = C(n, n-r). Always compute the smaller side for efficiency.
Stars and Bars
Ways to put n identical items into k bins = C(n+k-1, k-1). Powerful counting tool.
Modular nCr
Precompute factorials and inverse factorials for O(1) queries of C(n,r) mod p.

// Pascal's Triangle — O(n^2)
function pascalsTriangle(n) {
    var tri = [[1]];
    for (var i = 1; i <= n; i++) {
        var row = [1];
        for (var j = 1; j < i; j++) {
            row.push(tri[i-1][j-1] + tri[i-1][j]);
        }
        row.push(1);
        tri.push(row);
    }
    return tri;
}

// nCr using Pascal's Triangle
function nCr(n, r) {
    return pascalsTriangle(n)[n][r];
}

// Modular nCr with precomputed factorials
var MOD = 1000000007;
function precompute(n) {
    var fact = [1], inv = [1];
    for (var i = 1; i <= n; i++) {
        fact[i] = (fact[i-1] * i) % MOD;
    }
    inv[n] = modPow(fact[n], MOD - 2, MOD);
    for (var j = n - 1; j >= 0; j--) {
        inv[j] = (inv[j+1] * (j+1)) % MOD;
    }
    return { fact: fact, inv: inv };
}

function nCrMod(n, r, tables) {
    if (r < 0 || r > n) return 0;
    return tables.fact[n] * tables.inv[r] % MOD
           * tables.inv[n-r] % MOD;
}
