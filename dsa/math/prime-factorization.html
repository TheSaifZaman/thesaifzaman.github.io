Prime Factorization

Easy Time: O(sqrt(n)) Space: O(log n)

How It Works

Prime factorization decomposes a number into its prime factors. By the fundamental theorem of arithmetic, every integer greater than 1 has a unique prime factorization. For example, 360 = 2^3 * 3^2 * 5. Finding this decomposition is a core operation in number theory.

The simplest approach is trial division: try dividing by 2, then by odd numbers from 3 up to sqrt(n). Each time a divisor d divides n, extract all copies of d (divide n by d repeatedly) and record d as a factor. If n > 1 after processing all divisors up to sqrt(n), then n itself is a prime factor.

For factorizing many numbers, precompute a smallest prime factor (SPF) array using a modified Sieve of Eratosthenes. Then any number can be factorized in O(log n) by repeatedly dividing by its SPF. This is essential for problems requiring bulk factorization.

Trial Division
Test divisors from 2 to sqrt(n). If d divides n, extract all copies of d.
Uniqueness
Every integer > 1 has exactly one prime factorization (fundamental theorem of arithmetic).
SPF Sieve
Precompute smallest prime factor for each number. Factorize any n in O(log n) lookups.
Applications
Divisor counting, Euler's totient, GCD/LCM, and multiplicative functions all rely on factorization.

Implementation

// Trial division — O(sqrt(n))
function primeFactors(n) {
    var factors = [];
    // Extract all 2s
    while (n % 2 === 0) {
        factors.push(2);
        n = Math.floor(n / 2);
    }
    // Try odd divisors from 3 to sqrt(n)
    for (var d = 3; d * d <= n; d += 2) {
        while (n % d === 0) {
            factors.push(d);
            n = Math.floor(n / d);
        }
    }
    // If n is still > 1, it's a prime factor
    if (n > 1) factors.push(n);
    return factors;
}

// SPF Sieve for bulk factorization
function buildSPF(limit) {
    var spf = [];
    for (var i = 0; i <= limit; i++) spf[i] = i;
    for (var p = 2; p * p <= limit; p++) {
        if (spf[p] === p) { // p is prime
            for (var m = p * p; m <= limit; m += p) {
                if (spf[m] === m) spf[m] = p;
            }
        }
    }
    return spf;
}

function factorizeWithSPF(n, spf) {
    var factors = [];
    while (n > 1) {
        factors.push(spf[n]);
        n = Math.floor(n / spf[n]);
    }
    return factors;
}
