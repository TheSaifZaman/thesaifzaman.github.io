Catalan Numbers

Medium Time: O(n) with formula Space: O(n) for DP

How It Works

The Catalan numbers form a sequence of natural numbers that appear in many counting problems: C(0)=1, C(1)=1, C(2)=2, C(3)=5, C(4)=14, C(5)=42, ... The n-th Catalan number can be computed using the formula C(n) = (2n)! / ((n+1)! * n!), or equivalently C(n) = C(2n, n) / (n+1) where C(2n, n) is the binomial coefficient.

Catalan numbers count an astonishing variety of combinatorial structures: the number of valid parenthesizations with n pairs, the number of distinct binary search trees with n nodes, the number of paths in a grid that don't cross the diagonal, the number of triangulations of a polygon, and many more.

The recurrence relation is C(n) = sum of C(i)*C(n-1-i) for i=0 to n-1, which can be computed in O(n^2) with DP. The closed-form formula gives O(n) computation. For modular arithmetic, precompute factorials and use modular inverses.

Recurrence
C(n) = C(0)*C(n-1) + C(1)*C(n-2) + ... + C(n-1)*C(0). Convolution-like structure.
Closed Form
C(n) = (2n choose n) / (n+1). Efficient O(n) computation.
Valid Parentheses
C(n) = number of ways to arrange n pairs of parentheses validly.
BST Count
C(n) = number of structurally unique BSTs with n nodes (LeetCode #96).

Implementation

// DP approach — O(n^2)
function catalanDP(n) {
    var dp = new Array(n + 1).fill(0);
    dp[0] = dp[1] = 1;
    for (var i = 2; i <= n; i++) {
        for (var j = 0; j < i; j++) {
            dp[i] += dp[j] * dp[i - 1 - j];
        }
    }
    return dp[n];
}

// Formula approach — O(n)
function catalanFormula(n) {
    var result = 1;
    for (var i = 0; i < n; i++) {
        result = result * (2 * n - i) / (i + 1);
    }
    return Math.round(result / (n + 1));
}

// Modular Catalan (for competitive programming)
function catalanMod(n, MOD) {
    // Precompute factorials and inverse factorials
    var fact = [1];
    for (var i = 1; i <= 2 * n; i++) {
        fact[i] = (fact[i-1] * i) % MOD;
    }
    function modPow(b, e, m) {
        var r = 1; b %= m;
        while (e > 0) {
            if (e % 2 === 1) r = (r * b) % m;
            e = Math.floor(e / 2);
            b = (b * b) % m;
        }
        return r;
    }
    var inv = modPow(fact[n + 1], MOD - 2, MOD);
    var invN = modPow(fact[n], MOD - 2, MOD);
    return (fact[2 * n] * inv % MOD) * invN % MOD;
}
