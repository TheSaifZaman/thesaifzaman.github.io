Q-Learning

Medium Time: O(S·A·episodes) Space: O(S·A)

How It Works

Q-Learning is a model-free reinforcement learning algorithm that learns the value of taking an action in a particular state. It maintains a Q-table mapping every (state, action) pair to an expected cumulative reward. The agent explores the environment, and after each action it updates the Q-value using the Bellman equation:

Q(s, a) ← Q(s, a) + α [r + γ max Q(s', a') - Q(s, a)]

Because Q-Learning uses the maximum Q-value of the next state (regardless of the action actually taken), it is an off-policy algorithm. Over many episodes the Q-values converge to the optimal action-value function, allowing the agent to derive an optimal policy by always choosing the action with the highest Q-value in each state.

Q-Table

Stores expected rewards for every state-action pair.

ε-Greedy

Balances exploration (random) and exploitation (best Q).

Off-Policy

Updates use max future Q regardless of action taken.

Convergence

Guaranteed to converge with sufficient exploration and decaying ε.

Implementation

function qLearning(env, episodes, alpha, gamma, epsilon) {
    var Q = {};  // Q[state][action] = value
    for (var ep = 0; ep < episodes; ep++) {
        var state = env.reset();
        while (!env.done(state)) {
            var action;
            if (Math.random() < epsilon) {
                action = env.randomAction(state);
            } else {
                action = argMax(Q[state]);
            }
            var result = env.step(state, action);
            var reward = result.reward;
            var next = result.nextState;

            if (!Q[state]) Q[state] = {};
            if (!Q[state][action]) Q[state][action] = 0;
            if (!Q[next]) Q[next] = {};

            var maxNext = Math.max.apply(null,
                env.actions(next).map(function(a) {
                    return Q[next][a] || 0;
                }));
            Q[state][action] += alpha * (reward + gamma * maxNext - Q[state][action]);
            state = next;
        }
    }
    return Q;
}
