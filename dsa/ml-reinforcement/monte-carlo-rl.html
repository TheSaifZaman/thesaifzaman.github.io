Monte Carlo Methods (RL)

Medium Time: O(T · episodes) Space: O(S·A)

How It Works

Monte Carlo (MC) methods in reinforcement learning estimate value functions by averaging actual returns observed over complete episodes. Unlike Temporal Difference methods (Q-Learning, SARSA) which bootstrap from estimates, MC waits until the episode ends and uses the true cumulative reward.

In first-visit MC, the return for a state is recorded only the first time it appears in an episode. In every-visit MC, every occurrence counts. After many episodes, the average converges to the true expected value by the law of large numbers.

MC methods are model-free, require no bootstrapping, and work well with episodic tasks. However, they can only learn from complete episodes and have higher variance than TD methods.

Episode-Based

Learns only after complete episodes, using actual returns.

Sample Averaging

Values converge to true expectations as episodes increase.

No Bootstrapping

Does not use estimates of other states — only real returns.

High Variance

Full returns are noisy; needs many episodes to converge.

Implementation

function monteCarloFirstVisit(env, episodes, gamma) {
    var V = {};      // state values
    var returns = {}; // list of returns per state

    for (var ep = 0; ep < episodes; ep++) {
        var episode = generateEpisode(env);

        // Compute returns backward
        var G = 0;
        var visited = {};
        for (var t = episode.length - 1; t >= 0; t--) {
            G = episode[t].reward + gamma * G;
            var state = episode[t].state;

            // First-visit: only count first occurrence
            if (!visited[state]) {
                visited[state] = true;
                if (!returns[state]) returns[state] = [];
                returns[state].push(G);
                // Average all returns for this state
                var sum = 0;
                for (var i = 0; i < returns[state].length; i++) {
                    sum += returns[state][i];
                }
                V[state] = sum / returns[state].length;
            }
        }
    }
    return V;
}
