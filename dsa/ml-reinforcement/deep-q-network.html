Deep Q-Network (DQN)

Hard Time: O(n·d²) Space: O(d²)

How It Works

Deep Q-Network (DQN) extends Q-Learning by replacing the Q-table with a neural network that approximates Q-values. This allows the algorithm to handle high-dimensional state spaces (like raw pixel inputs) where a table would be infeasibly large.

Two key innovations stabilize training: (1) Experience Replay stores past transitions in a buffer and samples random mini-batches to break correlations, and (2) a Target Network is a frozen copy of the Q-network used to compute target values, updated periodically to prevent oscillation.

DQN was famously used by DeepMind to play Atari games at superhuman levels directly from screen pixels, marking a breakthrough in deep reinforcement learning.

Neural Q-Function

Network takes state as input, outputs Q-values for all actions.

Experience Replay

Stores and samples transitions to break temporal correlations.

Target Network

Frozen copy provides stable training targets.

Atari Breakthrough

First to learn control policies from raw pixels end-to-end.

// DQN pseudocode
function trainDQN(env, episodes) {
    var qNet = new NeuralNetwork([stateSize, 128, 64, actionSize]);
    var targetNet = qNet.clone();
    var replayBuffer = [];
    var batchSize = 32;

    for (var ep = 0; ep < episodes; ep++) {
        var state = env.reset();
        while (!env.done()) {
            // Epsilon-greedy action selection
            var action = (Math.random() < epsilon)
                ? env.randomAction()
                : argMax(qNet.predict(state));

            var result = env.step(action);
            replayBuffer.push([state, action, result.reward, result.next, result.done]);

            if (replayBuffer.length >= batchSize) {
                var batch = sample(replayBuffer, batchSize);
                // Compute targets using frozen target network
                var targets = batch.map(function(t) {
                    if (t[4]) return t[2]; // terminal
                    return t[2] + gamma * max(targetNet.predict(t[3]));
                });
                qNet.trainBatch(batch, targets);
            }
            state = result.next;
        }
        // Periodically sync target network
        if (ep % 10 === 0) targetNet = qNet.clone();
    }
}
