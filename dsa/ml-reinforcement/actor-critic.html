Actor-Critic

Hard Time: O(T·d²) Space: O(d²)

How It Works

Actor-Critic methods combine two neural networks: the Actor learns a policy π(a|s) that decides which action to take, and the Critic learns a value function V(s) that evaluates how good the current state is. The critic provides a lower-variance baseline for the actor's gradient updates.

The advantage function A(s,a) = r + γV(s') - V(s) tells the actor whether the action was better or worse than expected. This dramatically reduces the variance problem of pure policy gradient methods while maintaining the ability to handle continuous action spaces.

Modern variants like A2C (synchronous) and A3C (asynchronous) use parallel workers for faster and more stable training.

Actor (Policy)

Learns which actions to take in each state.

Critic (Value)

Evaluates states to provide a baseline for the actor.

Implementation

function actorCritic(env, episodes, alphaActor, alphaCritic, gamma) {
    var actorParams = randomParams();  // policy network
    var criticParams = randomParams(); // value network

    for (var ep = 0; ep < episodes; ep++) {
        var state = env.reset();

        while (!env.done()) {
            var probs = actorForward(state, actorParams);
            var action = sampleFromDistribution(probs);
            var result = env.step(action);
            var nextState = result.nextState;

            // Critic evaluates current and next state
            var V = criticForward(state, criticParams);
            var Vnext = env.done() ? 0 : criticForward(nextState, criticParams);

            // Temporal Difference error (advantage)
            var advantage = result.reward + gamma * Vnext - V;

            // Update critic: minimize TD error
            var criticGrad = criticBackward(state, criticParams);
            criticParams = addVectors(criticParams,
                scaleVector(criticGrad, alphaCritic * advantage));

            // Update actor: reinforce with advantage
            var actorGrad = logPolicyGradient(state, action, actorParams);
            actorParams = addVectors(actorParams,
                scaleVector(actorGrad, alphaActor * advantage));

            state = nextState;
        }
    }
}
