Policy Gradient (REINFORCE)

Hard Time: O(T·d) Space: O(d)

How It Works

Policy Gradient methods directly optimize a parameterized policy π(a|s;θ) by computing the gradient of expected reward with respect to the policy parameters θ. Instead of learning value functions and deriving a policy, we directly learn which actions to take.

The REINFORCE algorithm collects complete episodes, then updates parameters using: θ ← θ + α ∇ log π(a|s;θ) · G_t, where G_t is the discounted return from time t. Actions that led to high returns get reinforced (higher probability), while poor actions get suppressed.

The policy surface visualization shows how the policy parameters move across the reward landscape, seeking the peak of expected return.

Direct Optimization

Optimizes policy parameters directly via gradient ascent on expected reward.

Stochastic Policy

Outputs action probabilities, enabling natural exploration.

High Variance

Monte Carlo returns are noisy; baselines help reduce variance.

Continuous Actions

Naturally handles continuous action spaces unlike Q-Learning.

Implementation

function reinforce(env, episodes, alpha, gamma) {
    var theta = randomParams();

    for (var ep = 0; ep < episodes; ep++) {
        var trajectory = [];
        var state = env.reset();

        // Collect full episode
        while (!env.done()) {
            var probs = policy(state, theta); // softmax over actions
            var action = sampleFromDistribution(probs);
            var result = env.step(action);
            trajectory.push({ state: state, action: action, reward: result.reward });
            state = result.nextState;
        }

        // Compute discounted returns
        var G = 0;
        for (var t = trajectory.length - 1; t >= 0; t--) {
            G = trajectory[t].reward + gamma * G;
            trajectory[t].G = G;
        }

        // Update policy parameters
        for (var t = 0; t < trajectory.length; t++) {
            var grad = logPolicyGradient(trajectory[t].state, trajectory[t].action, theta);
            theta = addVectors(theta, scaleVector(grad, alpha * trajectory[t].G));
        }
    }
    return theta;
}
